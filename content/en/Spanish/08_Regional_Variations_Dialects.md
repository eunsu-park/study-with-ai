## 2. Spanish Spanish

### 2.1 vosotros Conjugation

```
┌─────────────────────────────────────────────────────────────────┐
│                 vosotros Conjugation (Spain only)               │
│                                                                 │
│   Personal pronouns:                                            │
│   - vosotros/vosotras (you all) - informal plural              │
│   - ustedes (Uds.) - formal plural                              │
│                                                                 │
│   Verb conjugation example (hablar):                            │
│   - vosotros habláis (you all speak)                           │
│   - ustedes hablan (you all speak - formal)                    │
│                                                                 │
│   Object pronouns:                                              │
│   - os (you all - direct/indirect object)                      │
│   Example: Os quiero. (I love you all.)                        │
│   Example: Os lo doy. (I give it to you all.)                  │
│                                                                 │
│   Possessive adjectives:                                        │
│   - vuestro/vuestra/vuestros/vuestras                          │
│   Example: Vuestra casa es bonita. (Your house is pretty.)     │
└─────────────────────────────────────────────────────────────────┘

vosotros Conjugation Chart:
┌────────┬──────────┬──────────┬──────────┐
│ Tense  │ -AR      │ -ER      │ -IR      │
│        │ (hablar) │ (comer)  │ (vivir)  │
├────────┼──────────┼──────────┼──────────┤
│ Present│ habláis  │ coméis   │ vivís    │
│ Preterite│hablasteis│comisteis│vivisteis│
│ Imperfect│hablabais│comíais  │ vivíais  │
│ Future │ hablaréis│ comeréis │ viviréis │
│ Conditional│hablaríais│comeríais│viviríais│
│ Present Subj│habléis│comáis   │ viváis   │
│ Imperative│hablad  │ comed    │ vivid    │
└────────┴──────────┴──────────┴──────────┘
```

### 2.2 Spanish Pronunciation Features

```
1. Distinción (c/z vs s distinction):
   cena [θéna] vs sena [séna]
   zapato [θapáto]
   cerveza [θerβéθa]

2. Final -d pronunciation:
   Madrid [maðríð] or [maðrí]
   ciudad [θjuðáð]

3. j pronunciation (strong [x]):
   jota [xóta] - harsh sound from back of throat

4. ll pronunciation:
   Most regions [ʎ] (lateral palatal)
   Some regions [j] (yeísmo)
```

## 3. Mexican Spanish

### 3.1 Characteristics

```
┌─────────────────────────────────────────────────────────────────┐
│                    Mexican Spanish Features                     │
│                                                                 │
│   Pronunciation:                                                │
│   - Seseo: c/z pronounced as [s]                               │
│   - Final -s retained (unlike Caribbean)                        │
│   - Consonant clusters preserved: extra [éks.tra]              │
│                                                                 │
│   Grammar:                                                      │
│   - ustedes used (vosotros not used)                           │
│   - Preterite preferred: Hoy comí. (I ate today)               │
│   - Diminutives (-ito/-ita) frequently used                    │
│                                                                 │
│   Influence:                                                    │
│   - Náhuatl (Aztec language) vocabulary borrowing             │
│   - US English influence (northern regions)                     │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 Mexican-Specific Expressions

```
Common Expressions:
┌────────────────────┬──────────────────────────────────────────┐
│ Expression         │ Meaning                                  │
├────────────────────┼──────────────────────────────────────────┤
│ ¿Qué onda?        │ How's it going? / What's up?             │
│ ¡Órale!           │ Cool! / Really? / Okay!                  │
│ ¡Híjole!          │ Wow! (surprise)                          │
│ No manches        │ No way! / You're kidding?                │
│ ¡Ándale!          │ Come on! / Hurry up!                     │
│ Padre/Padrísimo   │ Awesome, cool                            │
│ Chido/Chida       │ Cool, nice                               │
│ Güey              │ Dude, friend (slang)                     │
│ Neta              │ Really, for real                         │
│ Fresa             │ Snobby person                            │
└────────────────────┴──────────────────────────────────────────┘

Náhuatl-Derived Vocabulary:
- chocolate (chocolatl) - chocolate
- tomate (tomatl) - tomato
- aguacate (ahuacatl) - avocado
- chile (chilli) - chili pepper
- guajolote - turkey (Mexican term for pavo)
```

### 3.3 Diminutive Usage

```
Especially frequent diminutives in Mexico (-ito/-ita):

ahorita (ahora + ita) - right now, in a moment
tantito (tanto + ito) - just a little bit
momentito - just a moment
lueguito - right away

Example sentences:
- Ahorita vengo. (I'll be right back.)
- Dame tantito. (Give me just a little.)
- Un momentito, por favor. (Just a moment, please.)

※ Diminutives add friendliness, modesty, or softening
```

## 4. Argentine/Uruguayan Spanish

### 4.1 Voseo

```
┌─────────────────────────────────────────────────────────────────┐
│                    Voseo (vos usage)                            │
│                                                                 │
│   vos used instead of tú (Argentina, Uruguay, parts of C.A.)   │
│                                                                 │
│   Verb conjugation comparison:                                  │
│   ┌──────────┬──────────────┬──────────────┐                   │
│   │ Verb     │ tú           │ vos          │                   │
│   ├──────────┼──────────────┼──────────────┤                   │
│   │ hablar   │ tú hablas    │ vos hablás   │                   │
│   │ comer    │ tú comes     │ vos comés    │                   │
│   │ vivir    │ tú vives     │ vos vivís    │                   │
│   │ ser      │ tú eres      │ vos sos      │                   │
│   │ tener    │ tú tienes    │ vos tenés    │                   │
│   │ poder    │ tú puedes    │ vos podés    │                   │
│   │ querer   │ tú quieres   │ vos querés   │                   │
│   │ pensar   │ tú piensas   │ vos pensás   │ (no stem change) │
│   │ venir    │ tú vienes    │ vos venís    │                   │
│   └──────────┴──────────────┴──────────────┘                   │
│                                                                 │
│   Rule: Stress on final syllable, no stem changes              │
│                                                                 │
│   Imperative:                                                   │
│   - tú: habla, come, vive                                      │
│   - vos: hablá, comé, viví                                     │
└─────────────────────────────────────────────────────────────────┘

Examples:
- ¿Vos tenés tiempo? (Do you have time?)
- ¿De dónde sos? (Where are you from?)
- Vení acá. (Come here.) - imperative
- Decime la verdad. (Tell me the truth.) - imperative
```

### 4.2 Pronunciation Features

```
1. Yeísmo/Sheísmo (ll, y pronunciation):
   - Most Spanish: ll, y → [j] (English y)
   - Argentina: ll, y → [ʃ] or [ʒ] (sh or zh)

   pollo: most [pójo] → Arg [póʃo]
   yo: most [jo] → Arg [ʃo]
   llamar: most [jamár] → Arg [ʃamár]

2. Intonation (Italian influence):
   - Melodic intonation similar to Italian
   - Tendency for rising sentence-final intonation

3. Final -s:
   - Retained (unlike Caribbean)
```

### 4.3 Argentine-Specific Expressions

```
┌────────────────────┬──────────────────────────────────────────┐
│ Expression         │ Meaning                                  │
├────────────────────┼──────────────────────────────────────────┤
│ Che               │ Hey, buddy (address)                     │
│ Boludo/Boluda     │ Friend/idiot (context-dependent)         │
│ ¡Qué quilombo!    │ What a mess!                             │
│ Bárbaro           │ Great, awesome                           │
│ Copado/Copada     │ Cool, nice                               │
│ Morfar            │ To eat (lunfardo)                        │
│ Laburo            │ Work, job (trabajo)                      │
│ Afanar            │ To steal                                 │
│ Mango             │ Money, peso                              │
│ Bondi             │ Bus                                      │
│ Pibe/Piba         │ Young person, friend                     │
│ Re-               │ Very (re lindo = very pretty)            │
└────────────────────┴──────────────────────────────────────────┘

Lunfardo (Buenos Aires slang):
- Influenced by Italian immigrant language
- Often appears in tango lyrics
```

## 5. Other Latin American Variations

### 5.1 Caribbean Region (Cuba, Dominican Republic, Puerto Rico)

```
Pronunciation Features:
┌─────────────────────────────────────────────────────────────────┐
│   1. Final/syllable-final -s → [h] or dropped                  │
│      estos → [éhtoh] or [éto]                                  │
│      más → [máh]                                               │
│                                                                 │
│   2. Final -r → -l or dropped                                  │
│      comer → [komél]                                           │
│      puerta → [pwélta]                                         │
│                                                                 │
│   3. Rapid speech, syllable shortening                         │
│                                                                 │
│   4. African language influence on vocabulary                  │
└─────────────────────────────────────────────────────────────────┘

Distinctive Expressions:
- ¿Qué lo qué? (Cuba/Dominican) - What's up? / How are you?
- ¡Wepa! (Puerto Rico) - Wow! / Cool!
- Chévere - Cool, nice (general Caribbean)
```

### 5.2 Colombia

```
┌─────────────────────────────────────────────────────────────────┐
│                    Colombian Spanish                            │
│                                                                 │
│   Features:                                                     │
│   - Considered "clearest" Spanish                              │
│   - Bogotá dialect: formal, clear pronunciation                │
│   - usted used even in informal situations (Bogotá)            │
│                                                                 │
│   Distinctive Expressions:                                      │
│   - ¿Quiubo? (qué hubo) - Hi?                                 │
│   - ¡Qué pena! - I'm sorry/That's a shame                     │
│   - A la orden - You're welcome/At your service                │
│   - Bacano - Cool                                               │
│   - Parce/Parcero - Friend                                     │
│                                                                 │
│   Regional Differences:                                         │
│   - Caribbean coast: Caribbean pronunciation influence         │
│   - Andean highlands: Clear pronunciation                      │
│   - Paisa region (Medellín): vos usage                        │
└─────────────────────────────────────────────────────────────────┘
```

### 5.3 Chile

```
Pronunciation Features:
- Final -s weakening: [h] or dropped
- -ado → -ao: cansado → cansao
- ch pronunciation unique: [tʃ] → closer to [ʃ]

Distinctive Expressions:
┌────────────────────┬──────────────────────────────────────────┐
│ Expression         │ Meaning                                  │
├────────────────────┼──────────────────────────────────────────┤
│ ¿Cachai?          │ You know? (Understand?)                  │
│ Po (pues)         │ Sentence-final emphasis (Sí po, No po)   │
│ Fome              │ Boring                                   │
│ Pololo/Polola     │ Boyfriend/girlfriend                     │
│ Guagua            │ Baby                                     │
│ Luca              │ 1,000 pesos                              │
│ Bacán             │ Cool                                     │
└────────────────────┴──────────────────────────────────────────┘
```

## 6. Practice Exercises

### Exercise 1: Regional Identification
Identify which region's Spanish the following sentences represent.

```
1. "¿Vos tenés tiempo para tomar un mate?"
   → Argentina/Uruguay (vos usage, mate culture)

2. "¿Qué onda, güey? ¿Todo bien?"
   → Mexico (güey, qué onda)

3. "Vosotros sois muy majos."
   → Spain (vosotros, majos)

4. "¿Cachai lo que te digo?"
   → Chile (cachai)

5. "A la orden, con mucho gusto."
   → Colombia (a la orden)
```

### Exercise 2: Vocabulary Conversion
Convert Spanish Spanish to Mexican Spanish.

```
1. ordenador → computadora
2. móvil → celular
3. coche → carro
4. piso → departamento
5. patatas → papas
6. zumo → jugo
```

### Exercise 3: vosotros vs ustedes
Choose the appropriate form.

```
Spain:
1. ¿(Tenéis / Tienen) hambre, chicos?
   → Tenéis (informal plural)

Latin America:
2. ¿(Tenéis / Tienen) hambre, chicos?
   → Tienen (ustedes form)

Argentina:
3. ¿(Tienes / Tenés) tiempo?
   → Tenés (vos form)
```

### Exercise 4: Pronunciation Notation
Distinguish pronunciation between Spain and Latin America for these words.

```
1. zapato
   - Spain: [θapáto]
   - Latin America: [sapáto]

2. cena
   - Spain: [θéna]
   - Latin America: [séna]

3. pollo (Argentina)
   - General: [pójo]
```
