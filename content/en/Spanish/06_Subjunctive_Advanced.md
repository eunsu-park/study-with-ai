## 1. Subjunctive Overview

### 1.1 What is the Subjunctive?

```
┌─────────────────────────────────────────────────────────────────┐
│                    Mood Comparison                               │
│                                                                 │
│   Indicative              Subjunctive                           │
│   ├── Facts, certainty    ├── Wishes, doubt, emotions          │
│   ├── Objective statement ├── Subjective attitude              │
│   └── Reality             └── Hypothetical, uncertain          │
│                                                                 │
│   Examples:                                                      │
│   Sé que viene. (I know he's coming) - Indicative              │
│   Espero que venga. (I hope he comes) - Subjunctive            │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 Subjunctive Tense System

```
Subjunctive Tenses:
┌────────────────────────────────────────────────────────────────┐
│ Present Subjunctive (Presente de Subjuntivo)                   │
│   → Present/future uncertain situations                         │
│   Espero que llegue pronto. (I hope he arrives soon)          │
│                                                                 │
│ Past Subjunctive (Imperfecto de Subjuntivo)                    │
│   → Past/hypothetical situations, polite requests              │
│   Si tuviera tiempo, viajaría. (If I had time, I would travel)│
│                                                                 │
│ Present Perfect Subjunctive (Pretérito Perfecto de Subjuntivo) │
│   → Completed uncertain past situations                         │
│   Espero que haya llegado. (I hope he has arrived)            │
│                                                                 │
│ Pluperfect Subjunctive (Pluscuamperfecto de Subjuntivo)       │
│   → Past hypothetical completed situations                      │
│   Si hubiera sabido... (If I had known...)                    │
└────────────────────────────────────────────────────────────────┘
```

## 2. Present Subjunctive

### 2.1 Regular Conjugation

```
┌─────────────────────────────────────────────────────────────────┐
│                 Present Subjunctive Regular                      │
│                                                                 │
│   -AR verbs       -ER verbs        -IR verbs                   │
│   (hablar)       (comer)          (vivir)                      │
├─────────────────────────────────────────────────────────────────┤
│   hable          coma             viva                         │
│   hables         comas            vivas                        │
│   hable          coma             viva                         │
│   hablemos       comamos          vivamos                      │
│   habléis        comáis           viváis                       │
│   hablen         coman            vivan                        │
│                                                                 │
│   Rule: -AR verbs use -e, -ER/-IR verbs use -a vowels         │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 Irregular Conjugations

```
Major Irregular Verbs:
┌──────────┬───────────────────────────────────────────────────────┐
│ Verb     │ Present Subjunctive                                   │
├──────────┼───────────────────────────────────────────────────────┤
│ ser      │ sea, seas, sea, seamos, seáis, sean                  │
│ estar    │ esté, estés, esté, estemos, estéis, estén            │
│ ir       │ vaya, vayas, vaya, vayamos, vayáis, vayan            │
│ haber    │ haya, hayas, haya, hayamos, hayáis, hayan            │
│ saber    │ sepa, sepas, sepa, sepamos, sepáis, sepan            │
│ dar      │ dé, des, dé, demos, deis, den                        │
│ ver      │ vea, veas, vea, veamos, veáis, vean                  │
├──────────┼───────────────────────────────────────────────────────┤
│ Stem-changing verbs (e→ie, o→ue, e→i patterns maintained)      │
│ pensar   │ piense, pienses, piense, pensemos, penséis, piensen  │
│ poder    │ pueda, puedas, pueda, podamos, podáis, puedan        │
│ pedir    │ pida, pidas, pida, pidamos, pidáis, pidan            │
└──────────┴───────────────────────────────────────────────────────┘
```

### 2.3 Usage Examples

```
Wishes/Hopes:
- Espero que tengas un buen día.
  (I hope you have a good day.)
- Ojalá llegue a tiempo.
  (I hope he arrives on time.)

Emotions:
- Me alegra que estés aquí.
  (I'm glad you're here.)
- Siento que no puedas venir.
  (I'm sorry you can't come.)

Doubt/Uncertainty:
- No creo que sea verdad.
  (I don't think it's true.)
- Dudo que lo sepa.
  (I doubt he knows it.)

Commands/Requests:
- Quiero que vengas conmigo.
  (I want you to come with me.)
- Te pido que me ayudes.
  (I ask you to help me.)
```

## 3. Past Subjunctive

### 3.1 Conjugation Forms

```
┌─────────────────────────────────────────────────────────────────┐
│            Past Subjunctive (Imperfecto de Subjuntivo)          │
│                                                                 │
│   Two forms (-ra / -se)                                         │
│   (Same meaning, -ra form more common)                          │
│                                                                 │
│   hablar                                                        │
│   -ra form           -se form                                   │
│   hablara            hablase                                    │
│   hablaras           hablases                                   │
│   hablara            hablase                                    │
│   habláramos         hablásemos                                 │
│   hablarais          hablaseis                                  │
│   hablaran           hablasen                                   │
│                                                                 │
│   comer: comiera/comiese, comieras/comieses...                 │
│   vivir: viviera/viviese, vivieras/vivieses...                 │
│                                                                 │
│   ※ Stem derived from 3rd person plural preterite             │
│   hablaron → habla- → hablara                                  │
│   comieron → comie- → comiera                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 Irregular Verbs

```
Past Subjunctive Irregulars:
┌──────────┬───────────────────────────────────────────────────────┐
│ Verb     │ Past Subjunctive (-ra form)                           │
├──────────┼───────────────────────────────────────────────────────┤
│ ser/ir   │ fuera, fueras, fuera, fuéramos, fuerais, fueran     │
│ estar    │ estuviera, estuvieras, estuviera...                  │
│ tener    │ tuviera, tuvieras, tuviera...                        │
│ poder    │ pudiera, pudieras, pudiera...                        │
│ poner    │ pusiera, pusieras, pusiera...                        │
│ saber    │ supiera, supieras, supiera...                        │
│ hacer    │ hiciera, hicieras, hiciera...                        │
│ decir    │ dijera, dijeras, dijera...                           │
│ querer   │ quisiera, quisieras, quisiera...                     │
│ venir    │ viniera, vinieras, viniera...                        │
└──────────┴───────────────────────────────────────────────────────┘
```

### 3.3 Usage Situations

```
1. Past uncertain situations:
   - Esperaba que viniera. (I was hoping he would come.)
   - No creía que fuera posible. (I didn't think it was possible.)

2. Polite requests (quisiera):
   - Quisiera un café, por favor. (I would like a coffee, please.)
   - Quisiera hablar con el gerente. (I would like to speak with the manager.)

3. Hypothetical/conditionals:
   - Si pudiera, te ayudaría. (If I could, I would help you.)
   - Si fuera rico, viajaría mucho. (If I were rich, I would travel a lot.)

4. "como si" (as if):
   - Habla como si supiera todo. (He talks as if he knew everything.)
   - Actúa como si no pasara nada. (He acts as if nothing happened.)
```

## 4. Subjunctive Triggers

### 4.1 WEIRDO Rule

```
┌─────────────────────────────────────────────────────────────────┐
│                    WEIRDO Mnemonic                               │
│                                                                 │
│   W - Wishes (hopes/desires)                                    │
│       querer que, desear que, esperar que, ojalá (que)         │
│                                                                 │
│   E - Emotions                                                  │
│       alegrarse de que, sentir que, temer que, molestar que    │
│                                                                 │
│   I - Impersonal expressions                                    │
│       es importante que, es necesario que, es posible que      │
│                                                                 │
│   R - Recommendations (advice/commands)                         │
│       recomendar que, sugerir que, aconsejar que, pedir que    │
│                                                                 │
│   D - Doubt/Denial                                              │
│       dudar que, no creer que, no pensar que, negar que        │
│                                                                 │
│   O - Ojalá (hope) + conjunctions                              │
│       para que, antes de que, sin que, a menos que             │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 Detailed Examples

```
Wishes:
- Quiero que estudies más. (I want you to study more.)
- Deseo que seas feliz. (I wish you to be happy.)
- Espero que llueva mañana. (I hope it rains tomorrow.)
- Ojalá venga pronto. (I hope he comes soon.)

Emotions:
- Me alegro de que estés bien. (I'm glad you're well.)
- Siento que no puedas venir. (I'm sorry you can't come.)
- Temo que sea demasiado tarde. (I fear it's too late.)
- Me molesta que hables así. (It bothers me that you talk like that.)

Impersonal:
- Es importante que llegues a tiempo.
  (It's important that you arrive on time.)
- Es necesario que estudies. (It's necessary that you study.)
- Es posible que llueva. (It's possible it will rain.)
- Es mejor que te vayas. (It's better that you leave.)

Recommendations:
- Te recomiendo que descanses. (I recommend you rest.)
- Sugiero que lo pienses bien. (I suggest you think it over.)
- Te pido que me ayudes. (I ask you to help me.)
- Ordeno que salgas. (I order you to leave.)

Doubt/Denial:
- Dudo que lo sepa. (I doubt he knows it.)
- No creo que sea verdad. (I don't think it's true.)
- Niego que haya dicho eso. (I deny having said that.)
```

### 4.3 Conjunctions + Subjunctive

```
Conjunctions that always require subjunctive:
┌─────────────────┬───────────────────────────────────────────────┐
│ Conjunction     │ Example                                        │
├─────────────────┼───────────────────────────────────────────────┤
│ para que        │ Te lo digo para que lo sepas.                 │
│ (so that)       │ (I tell you so you know.)                      │
├─────────────────┼───────────────────────────────────────────────┤
│ antes de que    │ Llámame antes de que salgas.                  │
│ (before)        │ (Call me before you leave.)                    │
├─────────────────┼───────────────────────────────────────────────┤
│ sin que         │ Salió sin que nadie lo viera.                 │
│ (without)       │ (He left without anyone seeing him.)           │
├─────────────────┼───────────────────────────────────────────────┤
│ a menos que     │ Iré a menos que llueva.                       │
│ (unless)        │ (I'll go unless it rains.)                     │
├─────────────────┼───────────────────────────────────────────────┤
│ con tal de que  │ Lo haré con tal de que me pagues.             │
│ (provided that) │ (I'll do it provided you pay me.)              │
├─────────────────┼───────────────────────────────────────────────┤
│ en caso de que  │ En caso de que no venga, llámame.             │
│ (in case)       │ (In case he doesn't come, call me.)            │
└─────────────────┴───────────────────────────────────────────────┘
```

## 5. Conditionals and Subjunctive

### 5.1 Conditional Types

```
┌─────────────────────────────────────────────────────────────────┐
│                    Conditional Types                             │
│                                                                 │
│   Type 1: Realizable condition (indicative)                     │
│   Si + present indicative, future/present indicative            │
│   Si llueve, no salgo. (If it rains, I won't go out.)         │
│   Si estudias, aprobarás. (If you study, you'll pass.)         │
│                                                                 │
│   Type 2: Hypothetical/unreal condition (subjunctive)           │
│   Si + past subjunctive, simple conditional                     │
│   Si tuviera dinero, compraría un coche.                       │
│   (If I had money, I would buy a car.)                         │
│                                                                 │
│   Type 3: Past hypothetical condition (subjunctive)             │
│   Si + pluperfect subjunctive, conditional perfect              │
│   Si hubiera estudiado, habría aprobado.                       │
│   (If I had studied, I would have passed.)                     │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 Type 2 Conditional Examples

```
Present/future hypothetical situations:

Si fuera rico, viajaría por todo el mundo.
(If I were rich, I would travel around the world.)

Si pudiera volar, iría a verte ahora mismo.
(If I could fly, I would go see you right now.)

Si tuviera más tiempo, aprendería japonés.
(If I had more time, I would learn Japanese.)

Si no lloviera, iríamos a la playa.
(If it weren't raining, we would go to the beach.)
```

### 5.3 Type 3 Conditional Examples

```
Past hypothetical situations:

Si hubiera sabido, te habría ayudado.
(If I had known, I would have helped you.)

Si hubieras venido, habrías visto a María.
(If you had come, you would have seen María.)

Si no hubiera llovido, habríamos ido al parque.
(If it hadn't rained, we would have gone to the park.)

Mixed conditional (past condition → present result):
Si hubiera estudiado medicina, ahora sería médico.
(If I had studied medicine, I would be a doctor now.)
```

## 6. Practice Exercises

### Exercise 1: Indicative vs Subjunctive Choice
Choose the correct form.

```
1. Creo que (viene / venga) mañana.
   → viene (creer + affirmative = indicative)

2. No creo que (viene / venga) mañana.
   → venga (no creer = subjunctive)

3. Sé que (está / esté) aquí.
   → está (saber = certainty = indicative)

4. Espero que (tienes / tengas) suerte.
   → tengas (esperar = hope = subjunctive)

5. Es verdad que (llueve / llueva).
   → llueve (es verdad = fact = indicative)

6. Es posible que (llueve / llueva).
   → llueva (es posible = uncertainty = subjunctive)
```

### Exercise 2: Complete Subjunctive Conjugations
Fill in the blanks.

```
1. Quiero que tú _______ (venir) a la fiesta.
   → vengas

2. Esperamos que ellos _______ (llegar) a tiempo.
   → lleguen

3. Si yo _______ (tener) dinero, viajaría.
   → tuviera

4. Me alegro de que tú _______ (estar) bien.
   → estés

5. Es importante que nosotros _______ (estudiar).
   → estudiemos

6. Si ella _______ (saber), te lo habría dicho.
   → hubiera sabido
```

### Exercise 3: Sentence Composition
Create sentences using subjunctive for the given situations.

```
1. Tell a friend you hope they're well
   → Espero que estés bien. / Deseo que te mejores.

2. Hope it doesn't rain tomorrow
   → Ojalá no llueva mañana.

3. Say you'd buy a big house if you were rich
   → Si fuera rico, compraría una casa grande.

4. Ask someone to arrive early
   → Te pido que llegues temprano.

5. Doubt whether she will come
   → Dudo que ella venga.
```
