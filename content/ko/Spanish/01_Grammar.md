## 1. 동사 기초

### A. SER vs ESTAR (두 가지 '~이다' 동사)

#### SER - 영구적/본질적 특성
**활용 (Conjugation):**
- yo soy / I am / 나는 ~이다
- tú eres / you are / 너는 ~이다
- él/ella/usted es / he/she/you(formal) is / 그/그녀/당신은 ~이다
- nosotros/as somos / we are / 우리는 ~이다
- vosotros/as sois / you all are / 너희는 ~이다
- ellos/ellas/ustedes son / they/you all are / 그들/당신들은 ~이다

**사용 (Uses):**
- 정체성: Soy estudiante / I am a student / 나는 학생이다
- 국적: Ella es coreana / She is Korean / 그녀는 한국인이다
- 성격: Él es amable / He is kind / 그는 친절하다
- 직업: Somos ingenieros / We are engineers / 우리는 엔지니어다
- 시간: Son las tres / It's three o'clock / 3시다
- 소유: Es mi libro / It's my book / 내 책이다
- 재료: La mesa es de madera / The table is (made of) wood / 탁자는 나무로 되어있다

#### ESTAR - 일시적 상태/위치
**활용 (Conjugation):**
- yo estoy / I am / 나는 ~이다
- tú estás / you are / 너는 ~이다
- él/ella/usted está / he/she/you is / 그/그녀/당신은 ~이다
- nosotros/as estamos / we are / 우리는 ~이다
- vosotros/as estáis / you all are / 너희는 ~이다
- ellos/ellas/ustedes están / they/you all are / 그들/당신들은 ~이다

**사용 (Uses):**
- 위치: Estoy en casa / I am at home / 나는 집에 있다
- 감정: Está feliz / He/She is happy / 그/그녀는 행복하다
- 상태: El café está frío / The coffee is cold / 커피가 차갑다
- 진행형: Estoy estudiando / I am studying / 나는 공부하고 있다

**비교 예제:**
- Soy feliz / I am (a) happy (person) / 나는 행복한 사람이다
- Estoy feliz / I am happy (right now) / 나는 (지금) 행복하다

### B. TENER (가지다) 동사

**활용 (Conjugation):**
- yo tengo / I have / 나는 가진다
- tú tienes / you have / 너는 가진다
- él/ella/usted tiene / he/she/you has / 그/그녀/당신은 가진다
- nosotros/as tenemos / we have / 우리는 가진다
- vosotros/as tenéis / you all have / 너희는 가진다
- ellos/ellas/ustedes tienen / they/you all have / 그들/당신들은 가진다

**관용 표현 (Idiomatic Expressions):**
- tener hambre / to be hungry / 배고프다
- tener sed / to be thirsty / 목마르다
- tener frío / to be cold / 춥다
- tener calor / to be hot / 덥다
- tener sueño / to be sleepy / 졸리다
- tener miedo / to be afraid / 두렵다
- tener razón / to be right / 옳다
- tener ... años / to be ... years old / ...살이다

## 2. 관사

### A. 정관사 (Definite Articles - "the")
|       | 남성형 (Masculine) | 여성형 (Feminine) |
|-------|-------------------|------------------|
| 단수 (Singular) | el | la |
| 복수 (Plural) | los | las |

**예제:**
- el libro / the book / 그 책
- la mesa / the table / 그 탁자
- los libros / the books / 그 책들
- las mesas / the tables / 그 탁자들

**특별 규칙:** 여성 단수 명사가 강세가 있는 a/ha로 시작하면 el을 사용
- el agua / the water / 그 물 (하지만 las aguas)
- el alma / the soul / 그 영혼 (하지만 las almas)

### B. 부정관사 (Indefinite Articles - "a/an/some")
|       | 남성형 (Masculine) | 여성형 (Feminine) |
|-------|-------------------|------------------|
| 단수 (Singular) | un | una |
| 복수 (Plural) | unos | unas |

**예제:**
- un libro / a book / 한 권의 책
- una mesa / a table / 한 개의 탁자
- unos libros / some books / 몇 권의 책들
- unas mesas / some tables / 몇 개의 탁자들

## 3. 전치사

### EN (in, on, at)
- Estoy en casa / I am at home / 나는 집에 있다
- en la mesa / on the table / 탁자 위에
- en España / in Spain / 스페인에
- en enero / in January / 1월에
- en la mañana / in the morning / 아침에

### A (to, at)
- Voy a la escuela / I go to school / 나는 학교에 간다
- a las tres / at three o'clock / 3시에
- a pie / on foot / 걸어서

### DE (of, from)
- el libro de Juan / Juan's book / 후안의 책
- Soy de Corea / I am from Korea / 나는 한국 출신이다
- una taza de café / a cup of coffee / 커피 한 잔

### CON (with)
- Voy con mi amigo / I go with my friend / 나는 친구와 간다
- café con leche / coffee with milk / 카페라떼

### PARA (for, in order to)
- Es para ti / It's for you / 너를 위한 거야
- Estudio para aprender / I study to learn / 나는 배우기 위해 공부한다

### POR (for, by, through, because of)
- por la mañana / in the morning / 아침에
- por favor / please / 부탁합니다
- Gracias por todo / Thanks for everything / 모든 것에 감사합니다

## 4. 명사의 성과 수

### A. 성 (Gender)
**남성형 규칙:**
- -o로 끝남: el libro, el gato
- 예외: la mano (손), la foto (사진)

**여성형 규칙:**
- -a로 끝남: la casa, la mesa
- -ción, -sión, -dad, -tad로 끝남: la nación, la universidad
- 예외: el día (날), el mapa (지도)

### B. 수 (Number - Singular/Plural)
**복수형 만들기:**
- 모음으로 끝나면 **-s** 추가: libro → libros, casa → casas
- 자음으로 끝나면 **-es** 추가: papel → papeles, ciudad → ciudades
- -z로 끝나면 -z를 **-ces**로 변경: lápiz → lápices, vez → veces

## 5. 형용사의 일치

형용사는 수식하는 명사의 **성(gender)**과 **수(number)**에 일치해야 합니다.

### A. 성 일치 (Gender Agreement)
- el gato negro / the black cat (masculine) / 검은 고양이
- la gata negra / the black cat (feminine) / 검은 고양이

### B. 수 일치 (Number Agreement)
- el libro nuevo / the new book / 새 책
- los libros nuevos / the new books / 새 책들
- la casa nueva / the new house / 새 집
- las casas nuevas / the new houses / 새 집들

### C. 형용사 어미 변화 패턴

| 어미 | 남성 단수 | 여성 단수 | 남성 복수 | 여성 복수 |
|------|----------|----------|----------|----------|
| -o/a | bonito | bonita | bonitos | bonitas |
| -e | grande | grande | grandes | grandes |
| 자음 | azul | azul | azules | azules |

**예외:**
- 국적 형용사: español/española, alemán/alemana
- 비교급: mejor/mejor, peor/peor (불변)

## 6. 동사 활용

### A. 규칙 동사 현재형 (Regular Verbs - Present Tense)

#### -AR 동사 (예: hablar / to speak)
- yo hablo / I speak / 나는 말한다
- tú hablas / you speak / 너는 말한다
- él/ella habla / he/she speaks / 그/그녀는 말한다
- nosotros hablamos / we speak / 우리는 말한다
- vosotros habláis / you all speak / 너희는 말한다
- ellos hablan / they speak / 그들은 말한다

#### -ER 동사 (예: comer / to eat)
- yo como / I eat / 나는 먹는다
- tú comes / you eat / 너는 먹는다
- él/ella come / he/she eats / 그/그녀는 먹는다
- nosotros comemos / we eat / 우리는 먹는다
- vosotros coméis / you all eat / 너희는 먹는다
- ellos comen / they eat / 그들은 먹는다

#### -IR 동사 (예: vivir / to live)
- yo vivo / I live / 나는 산다
- tú vives / you live / 너는 산다
- él/ella vive / he/she lives / 그/그녀는 산다
- nosotros vivimos / we live / 우리는 산다
- vosotros vivís / you all live / 너희는 산다
- ellos viven / they live / 그들은 산다

## 8. 부정

부정문을 만들려면 동사 앞에 **no**를 붙입니다.

- No hablo español / I don't speak Spanish / 나는 스페인어를 말하지 않는다
- No tengo tiempo / I don't have time / 나는 시간이 없다
- No es mi libro / It's not my book / 내 책이 아니다

**이중 부정 (허용됨!):**
- No tengo nada / I don't have anything / 나는 아무것도 없다
- No veo a nadie / I don't see anyone / 나는 아무도 보지 못한다
- Nunca como carne / I never eat meat / 나는 절대 고기를 먹지 않는다

## 9. 소유형용사와 소유대명사

### A. 소유형용사 (Possessive Adjectives)

| 소유자 | 단수 명사 | 복수 명사 | 의미 |
|--------|----------|----------|------|
| yo | mi | mis | my / 나의 |
| tú | tu | tus | your / 너의 |
| él/ella/usted | su | sus | his/her/your / 그의/그녀의/당신의 |
| nosotros/as | nuestro/a | nuestros/as | our / 우리의 |
| vosotros/as | vuestro/a | vuestros/as | your / 너희의 |
| ellos/ellas/ustedes | su | sus | their/your / 그들의/당신들의 |

**예문:**
- mi libro / my book / 나의 책
- tu casa / your house / 너의 집
- nuestro padre / our father / 우리 아버지
- nuestra madre / our mother / 우리 어머니

### B. 소유대명사 (Possessive Pronouns)

| 소유자 | 남성 단수 | 여성 단수 | 남성 복수 | 여성 복수 |
|--------|----------|----------|----------|----------|
| yo | el mío | la mía | los míos | las mías |
| tú | el tuyo | la tuya | los tuyos | las tuyas |
| él/ella/usted | el suyo | la suya | los suyos | las suyas |
| nosotros/as | el nuestro | la nuestra | los nuestros | las nuestras |

**예문:**
- Este libro es el mío / This book is mine / 이 책은 내 것이다
- Esa casa es la tuya / That house is yours / 저 집은 네 것이다

## 10. 지시형용사와 지시대명사

### A. 지시형용사 (Demonstrative Adjectives)

| 거리 | 남성 단수 | 여성 단수 | 남성 복수 | 여성 복수 |
|------|----------|----------|----------|----------|
| 가까이 | este | esta | estos | estas |
| 중간 | ese | esa | esos | esas |
| 멀리 | aquel | aquella | aquellos | aquellas |

**예문:**
- este libro / this book / 이 책
- esa casa / that house / 저 집
- aquella montaña / that mountain (over there) / 저기 저 산

### B. 중성 지시대명사
- esto / this (thing) / 이것
- eso / that (thing) / 그것
- aquello / that (thing) over there / 저것

**예문:**
- ¿Qué es esto? / What is this? / 이것은 무엇입니까?
- Eso es verdad / That's true / 그것은 사실이다

## 11. 대명사

### A. 재귀 대명사 (Reflexive Pronouns)

| 주어 | 재귀 대명사 | 의미 |
|------|------------|------|
| yo | me | myself / 나 자신을 |
| tú | te | yourself / 너 자신을 |
| él/ella/usted | se | himself/herself/yourself / 그/그녀/당신 자신을 |
| nosotros/as | nos | ourselves / 우리 자신을 |
| vosotros/as | os | yourselves / 너희 자신을 |
| ellos/ellas/ustedes | se | themselves/yourselves / 그들/당신들 자신을 |

**예제:**
- Me levanto / I get up / 나는 일어난다
- Te lavas / You wash yourself / 너는 씻는다
- Se viste / He/She gets dressed / 그/그녀는 옷을 입는다

### B. 직접 목적어 대명사 (Direct Object Pronouns)

| 목적어 | 대명사 | 의미 |
|--------|--------|------|
| a mí | me | me / 나를 |
| a ti | te | you / 너를 |
| a él | lo | him, it / 그를, 그것을 |
| a ella | la | her, it / 그녀를, 그것을 |
| a nosotros/as | nos | us / 우리를 |
| a ellos | los | them / 그들을 |
| a ellas | las | them / 그녀들을 |

**예제:**
- Lo veo / I see it/him / 나는 그것/그를 본다
- Te quiero / I love you / 나는 너를 사랑한다
- La conozco / I know her / 나는 그녀를 안다

### C. 간접 목적어 대명사 (Indirect Object Pronouns)

| 목적어 | 대명사 | 의미 |
|--------|--------|------|
| a mí | me | to me / 나에게 |
| a ti | te | to you / 너에게 |
| a él/ella/usted | le | to him/her/you / 그/그녀/당신에게 |
| a nosotros/as | nos | to us / 우리에게 |
| a ellos/ellas/ustedes | les | to them/you all / 그들/당신들에게 |

**예제:**
- Me da el libro / He gives the book to me / 그는 나에게 책을 준다
- Te escribo / I write to you / 나는 너에게 쓴다
- Le digo / I tell him/her / 나는 그/그녀에게 말한다

**Le/Les → Se 변환:**
직접 목적어와 함께 쓸 때 le/les는 se로 변경
- Se lo doy / I give it to him/her / 나는 그/그녀에게 그것을 준다

### D. GUSTAR 구문

간접 목적어 + gustar + 주어

- Me gusta el café / I like coffee / 나는 커피를 좋아한다
- Te gustan los libros / You like books / 너는 책을 좋아한다
- Le gusta bailar / He/She likes to dance / 그/그녀는 춤추는 것을 좋아한다
- Nos gusta la música / We like music / 우리는 음악을 좋아한다

## 12. 접속사

### A. 등위 접속사
- **y** (e) / and / 그리고
- **o** (u) / or / 또는
- **pero** / but / 하지만
- **sino** / but rather / 그게 아니라
- **ni** / neither, nor / ~도 아니고

### B. 원인/이유
- **porque** / because / 왜냐하면
- **como** / since, as / ~이므로
- **pues** / well, since / ~하니까
- **ya que** / since / ~이므로

### C. 결과
- **así que** / so, therefore / 그래서
- **por lo tanto** / therefore / 따라서
- **por eso** / that's why / 그래서
- **entonces** / then, so / 그러면

### D. 목적
- **para que** / so that / ~하도록 (+ 접속법)

### E. 조건
- **si** / if / 만약
- **a menos que** / unless / ~하지 않는 한 (+ 접속법)

### F. 양보
- **aunque** / although / 비록 ~이지만
- **a pesar de que** / despite / ~에도 불구하고
- **sin embargo** / however / 그러나

### G. 시간
- **cuando** / when / ~할 때
- **mientras** / while / ~하는 동안
- **antes de que** / before / ~하기 전에 (+ 접속법)
- **después de que** / after / ~한 후에
- **hasta que** / until / ~할 때까지

## 13. 비교급과 최상급

### A. 동등 비교
**tan... como** / as... as / ~만큼 ~하다
- Es tan alto como su hermano / He's as tall as his brother / 그는 형만큼 크다

**tanto/a/os/as... como** / as much/many... as
- Tengo tantos libros como tú / I have as many books as you / 나는 너만큼 많은 책을 가지고 있다

### B. 비교급
**más... que** / more... than / ~보다 더
- Es más alto que yo / He's taller than me / 그는 나보다 크다

**menos... que** / less... than / ~보다 덜
- Es menos caro que ese / It's less expensive than that one / 그것은 저것보다 덜 비싸다

### C. 최상급
**el/la/los/las más** / the most / 가장
- Es el más alto de la clase / He's the tallest in the class / 그는 반에서 가장 크다

### D. 불규칙 비교급/최상급

| 형용사 | 비교급 | 최상급 |
|--------|--------|--------|
| bueno/a | mejor | el/la mejor |
| malo/a | peor | el/la peor |
| grande | mayor | el/la mayor |
| pequeño/a | menor | el/la menor |

## 14. 시제

### A. 미래 시제 (Future Tense)

#### 규칙 미래형
부정사 + 어미 (-é, -ás, -á, -emos, -éis, -án)

- hablaré / I will speak / 나는 말할 것이다
- comerás / you will eat / 너는 먹을 것이다
- vivirá / he/she will live / 그/그녀는 살 것이다

#### 불규칙 동사
- decir → diré
- hacer → haré
- poder → podré
- poner → pondré
- querer → querré
- saber → sabré
- salir → saldré
- tener → tendré
- venir → vendré

#### IR A + 부정사 (근접 미래)
- Voy a estudiar / I'm going to study / 나는 공부할 거야
- Vas a comer / You're going to eat / 너는 먹을 거야

### B. 과거 시제

#### 단순 과거 (Preterite)
완료된 행동

**-AR 동사 (hablar):**
- hablé, hablaste, habló, hablamos, hablasteis, hablaron

**-ER/-IR 동사 (comer):**
- comí, comiste, comió, comimos, comisteis, comieron

**주요 불규칙:**
- ser/ir: fui, fuiste, fue, fuimos, fuisteis, fueron
- estar: estuve, estuviste, estuvo, estuvimos, estuvisteis, estuvieron
- tener: tuve, tuviste, tuvo, tuvimos, tuvisteis, tuvieron
- hacer: hice, hiciste, hizo, hicimos, hicisteis, hicieron

#### 불완료 과거 (Imperfect)
습관적/진행 중인 행동

**-AR 동사 (hablar):**
- hablaba, hablabas, hablaba, hablábamos, hablabais, hablaban

**-ER/-IR 동사 (comer):**
- comía, comías, comía, comíamos, comíais, comían

**불규칙 (3개만!):**
- ser: era, eras, era, éramos, erais, eran
- ir: iba, ibas, iba, íbamos, ibais, iban
- ver: veía, veías, veía, veíamos, veíais, veían

#### Preterite vs Imperfect
- **Preterite**: 특정 시점의 완료된 행동
  - Ayer comí pizza / Yesterday I ate pizza / 어제 피자를 먹었다
- **Imperfect**: 습관적 행동, 배경 설명
  - Cuando era niño, comía pizza todos los días / When I was a child, I ate pizza every day / 어렸을 때 매일 피자를 먹었다

### C. 현재완료 (Present Perfect)
HABER (현재형) + 과거분사

**HABER:** he, has, ha, hemos, habéis, han

**과거분사:**
- -AR: -ado (hablado)
- -ER/-IR: -ido (comido, vivido)

**불규칙 과거분사:**
- hacer → hecho
- decir → dicho
- escribir → escrito
- ver → visto
- abrir → abierto
- poner → puesto
- volver → vuelto
- morir → muerto
- romper → roto

**예문:**
- He comido / I have eaten / 나는 먹었다
- Has viajado / You have traveled / 너는 여행했다
- Hemos estudiado / We have studied / 우리는 공부했다

### D. 조건법 (Conditional)
부정사 + 어미 (-ía, -ías, -ía, -íamos, -íais, -ían)

- hablaría / I would speak / 나는 말할 것이다
- comería / I would eat / 나는 먹을 것이다

**불규칙:** 미래형과 동일한 어간
- decir → diría
- hacer → haría
- poder → podría

**예문:**
- Me gustaría viajar / I would like to travel / 여행하고 싶어요

## 15. 명령법

### A. 긍정 명령

**Tú (비격식):**
대부분 3인칭 단수 현재형과 동일
- ¡Habla! / Speak! / 말해!
- ¡Come! / Eat! / 먹어!

**불규칙 tú 명령:**
- decir → di
- hacer → haz
- ir → ve
- poner → pon
- salir → sal
- ser → sé
- tener → ten
- venir → ven

**Usted (격식):**
접속법 3인칭 단수 사용
- ¡Hable! / Speak! / 말씀하세요!

**Vosotros:**
부정사의 -r을 -d로 변경
- ¡Hablad! / Speak! / 말해!

### B. 부정 명령
모든 인칭에서 접속법 사용
- ¡No hables! / Don't speak! / 말하지 마!
- ¡No comas! / Don't eat! / 먹지 마!

### C. 대명사 위치
**긍정 명령:** 동사 뒤에 붙임
- ¡Lávate! / Wash yourself! / 씻어!
- ¡Dámelo! / Give it to me! / 나에게 그것을 줘!

**부정 명령:** 동사 앞에 위치
- ¡No te levantes! / Don't get up! / 일어나지 마!
- ¡No me lo des! / Don't give it to me! / 나에게 그것을 주지 마!

## 16. 접속법 (기초)

### A. 현재 접속법 형태

**-AR 동사 (hablar):**
- hable, hables, hable, hablemos, habléis, hablen

**-ER 동사 (comer):**
- coma, comas, coma, comamos, comáis, coman

**-IR 동사 (vivir):**
- viva, vivas, viva, vivamos, viváis, vivan

### B. 주요 불규칙 동사
- ser: sea, seas, sea, seamos, seáis, sean
- estar: esté, estés, esté, estemos, estéis, estén
- haber: haya, hayas, haya, hayamos, hayáis, hayan
- ir: vaya, vayas, vaya, vayamos, vayáis, vayan
- saber: sepa, sepas, sepa, sepamos, sepáis, sepan

### C. 접속법 사용 상황 (WEIRDO)

**W - Wishes (희망/바람):**
- Espero que vengas / I hope you come / 네가 오길 바란다
- Quiero que estudies / I want you to study / 네가 공부하길 원한다

**E - Emotions (감정):**
- Me alegra que estés bien / I'm glad you're well / 네가 잘 지내서 기쁘다
- Siento que no puedas venir / I'm sorry you can't come / 네가 못 와서 유감이다

**I - Impersonal Expressions (비인칭 표현):**
- Es importante que estudies / It's important that you study / 네가 공부하는 것이 중요하다
- Es necesario que vengas / It's necessary that you come / 네가 오는 것이 필요하다

**R - Recommendations (권고/명령):**
- Te recomiendo que descanses / I recommend that you rest / 네가 쉬길 권한다

**D - Doubt/Denial (의심/부정):**
- Dudo que sea verdad / I doubt it's true / 사실인지 의심스럽다
- No creo que llueva / I don't think it will rain / 비가 올 것 같지 않다

**O - Ojalá (바람):**
- Ojalá que vengas / I hope you come / 네가 오길 바란다

**접속사 후:**
- para que, antes de que, aunque (미래/불확실)
- Te llamo para que lo sepas / I call you so you know / 네가 알도록 전화한다

## 동명사와 부정사

### A. 현재분사/동명사 (Gerund)
**형태:**
- -AR: -ando (hablando)
- -ER/-IR: -iendo (comiendo, viviendo)

**불규칙:**
- leer → leyendo
- ir → yendo
- decir → diciendo

**사용:**
- 진행형: Estoy estudiando / I am studying / 나는 공부하고 있다

### B. 부정사 (Infinitive)
**사용:**
- 동사 후: Quiero comer / I want to eat / 나는 먹고 싶다
- 전치사 후: antes de comer / before eating / 먹기 전에
- 주어로: Estudiar es importante / Studying is important / 공부하는 것은 중요하다

## 학습 팁 (Study Tips)

1. **체계적 학습**: 현재 → 과거 → 미래 → 조건법 → 접속법 순서로
2. **시제 구별**: Preterite vs Imperfect 차이 명확히
3. **매일 연습**: 동사 활용을 매일 한 동사씩 연습
4. **실생활 적용**: 일상을 스페인어로 표현하는 연습
5. **WEIRDO**: 접속법 사용 상황 암기
6. **대명사 위치**: 규칙 숙지하고 반복 연습
7. **예문 암기**: 문법 규칙보다 예문 통째로 외우기
