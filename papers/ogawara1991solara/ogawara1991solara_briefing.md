## 1. 핵심 기여 / Core Contribution

본 논문은 1991년 8월 ISAS(일본우주과학연구소)가 발사한 SOLAR-A(발사 후 YOHKOH로 개명) 위성의 전체적 개요를 제공한다. SOLAR-A는 HINOTORI(1981)의 후속 임무로, 태양 활동 극대기(1989–1992) 동안 X-선과 감마선 영역에서 태양 플레어의 고에너지 현상을 정밀히 관측하기 위한 협업 우주 천문대이다. 본 논문은 4개의 핵심 탑재 기기(HXT, SXT, WBS, BCS), 위성체 설계, 운용 모드, 과학적 목표를 종합적으로 소개한다.

This paper provides a comprehensive overview of SOLAR-A (renamed YOHKOH after launch), a Japanese spacecraft launched by ISAS in August 1991 from the Kagoshima Space Center. As the successor to HINOTORI, the mission was dedicated to studying solar flares — especially high-energy phenomena in X- and gamma-ray ranges — during solar cycle 22's maximum (1989–1992). The paper introduces the four-instrument payload (HXT, SXT, WBS, BCS), spacecraft design, automated flare-mode operations, and scientific objectives, and serves as the introductory paper for the YOHKOH special issue of Solar Physics.

## 2. 역사적 맥락 / Historical Context

### 시대 배경 / The Setting

1980년대 SMM(미국, 1980)과 HINOTORI(일본, 1981)는 처음으로 hard X-ray 영상을 획득하여 태양 플레어의 새로운 측면(다중 footpoint, type B/C 플레어, 초고온 플라즈마 T > 3×10⁷ K)을 드러냈다. 그러나 두 위성 모두 (a) hard X-ray 공간 분해능 부족, (b) 30 keV 이상의 에너지 범위 부재, (c) soft X-ray 반사망원경 부재(SKYLAB 1973–74 이후 처음), (d) Bragg 분광계의 낮은 감도 등의 한계가 있었다. SOLAR-A는 이러한 결점을 동시에 해결하기 위해 설계되었다.

In the 1980s, SMM (USA, 1980) and HINOTORI (Japan, 1981) produced the first hard X-ray images of flares, revealing multi-source brightenings (type B), coronal sources (type C), and superhot plasmas (T > 3×10⁷ K). However, both missions suffered from poor hard X-ray spatial resolution, no imaging above ~30 keV, lack of grazing-incidence soft X-ray telescopes since SKYLAB (1973–74), and low-sensitivity Bragg spectrometers. SOLAR-A was designed to overcome all of these limitations simultaneously, riding the next solar maximum.

### 타임라인 / Timeline


## 3. 필요한 배경 지식 / Prerequisites

- **Solar flare physics / 태양 플레어 물리**: impulsive vs gradual phases, particle acceleration, chromospheric evaporation, footpoint sources
- **X-ray emission mechanisms / X-선 방출 메커니즘**: thermal bremsstrahlung, non-thermal bremsstrahlung, line emission from highly-ionized Fe/Ca/S
- **Imaging techniques / 영상 기법**: Fourier-synthesis (modulation collimators, u,v-plane sampling) for hard X-rays; grazing-incidence (Wolter type I) optics for soft X-rays
- **Bragg spectroscopy / Bragg 분광법**: $n\lambda = 2d\sin\theta$ for high-resolution X-ray line spectra
- **Spacecraft engineering / 위성 공학**: three-axis stabilization, momentum wheels, sun-synchronous-like LEO orbit, telemetry budgets
- **Prior missions / 선행 임무**: SKYLAB ATM, SMM, HINOTORI instruments and findings

## 4. 핵심 용어 / Key Vocabulary

| 용어 / Term | 설명 / Explanation |
|---|---|
| HXT (Hard X-ray Telescope) | 64-element bigrid Fourier-synthesis collimator, 15–100 keV in 4 bands, ~5″ resolution / 64개 격자쌍 푸리에 합성 망원경, 4개 에너지 대역 |
| SXT (Soft X-ray Telescope) | Modified Wolter-I grazing-incidence + 1024×1024 CCD, 3–60 Å, ~2.5″ / 변형 Wolter-I 입사각 망원경 + CCD |
| BCS (Bragg Crystal Spectrometer) | Four bent crystals for S XV, Ca XIX, Fe XXV, Fe XXVI lines / 4개 굽은 결정 분광계 |
| WBS (Wide Band Spectrometer) | Four detectors: SXS (gas), HXS (NaI), GRS (BGO), RBM / 4종 검출기 모음 |
| Modulation collimator / 변조 콜리메이터 | Pair of grids creating spatial fringes — measures one Fourier component / 격자쌍이 만드는 공간 무늬, 푸리에 성분 측정 |
| Wolter type I optics | Two-reflection (paraboloid + hyperboloid) grazing-incidence X-ray mirrors / 2회 반사 입사각 광학계 |
| Type B / C flares | Tanaka's classification — B: impulsive double-source, C: gradual coronal source / 다나카 분류 |
| Flare mode / Quiet mode | Onboard automated observing modes triggered by counting-rate thresholds / 계수율 임계값으로 자동 전환 |
| BCS-out mode | Mode for sweeping out queue memory after flare ends / 플레어 종료 후 큐 메모리 비우기 모드 |
| Bubble Data Recorder (BDR) | 10 Mbyte magnetic-bubble onboard recorder / 자기 버블 메모리 기록기 |
| ISAS | Institute of Space and Astronautical Science (Japan) / 일본 우주과학연구소 |
| u,v-plane | Spatial-frequency plane sampled by interferometric/synthesis imaging / 합성 영상의 공간 주파수 평면 |

## 5. 수식 미리보기 / Equations Preview

**(a) Bragg's law for BCS / BCS의 Bragg 법칙**:
$$n\lambda = 2d \sin\theta$$
파장 $\lambda$의 X-선이 격자 간격 $d$인 결정에서 각도 $\theta$로 회절. BCS는 이를 이용해 ~mÅ 분해능으로 Fe XXV(1.85 Å), Fe XXVI(1.78 Å), Ca XIX(3.18 Å), S XV(5.04 Å) 라인을 측정.

X-rays of wavelength $\lambda$ diffract at angle $\theta$ from a crystal of lattice spacing $d$. BCS uses bent crystals to achieve ~mÅ resolution on key flare lines.

**(b) Fourier-synthesis imaging / 푸리에 합성 영상**:
$$I(x,y) = \sum_{i=1}^{32} V_i \exp\!\left[2\pi j\,(u_i x + v_i y)\right]$$
HXT의 32개 복소 푸리에 성분 $V_i$로부터 영상을 재구성 (visibility-based imaging). 각 격자쌍이 하나의 $(u_i, v_i)$ 점을 표본화.

The image is reconstructed from 32 complex Fourier components $V_i$ measured by the bigrid collimator pairs at specific spatial-frequency points.

**(c) Thermal bremsstrahlung X-ray flux / 열적 제동복사 플럭스**:
$$F_\nu \propto \int n_e^2 T^{-1/2} \exp(-h\nu/kT)\,dV$$
SXT(soft, T~10⁶–10⁷ K)와 BCS(superhot, T~3×10⁷ K) 진단의 기본식.

Underlying diagnostic for SXT (T~10⁶–10⁷ K) and BCS (superhot, T>3×10⁷ K).

**(d) Orbital period (Kepler) / 궤도 주기**:
$$T = 2\pi\sqrt{\frac{(R_\oplus + h)^3}{GM_\oplus}} \approx 97~\text{min for } h=600~\text{km}$$
SOLAR-A 궤도: 600 km 고도, 31° 경사, 97분 주기, 약 2/3 일조 + 1/3 식.

SOLAR-A's orbit: 600 km altitude, 31° inclination, 97-minute period.

**(e) Diffraction-limited angular resolution / 회절 한계 각분해능**:
$$\theta_{\min} \approx 1.22\,\frac{\lambda}{D}$$
HXT 5″는 회절 한계가 아니라 가장 좁은 격자 간격이 정함; SXT 2.5″는 CCD 픽셀(2.4″)이 결정.

HXT's 5″ is set by the finest grid pitch, not diffraction; SXT's 2.5″ is set by the CCD pixel size.

## 7. 현대적 의의 / Modern Significance

YOHKOH(1991–2001)는 10년간의 운용으로 태양 플레어 연구의 패러다임을 전환시켰다. SXT의 부드러운 X-선 영상은 cusp형 자기 재결합 구조(Tsuneta 1996), 시그모이드(sigmoid) 형성, X-ray jets, 코로나 홀 진화를 시각적으로 입증했다. HXT는 footpoint hard X-ray 발생과 looptop source(Masuda 1994)를 동시에 영상화하여 표준 플레어 모델의 결정적 증거를 제공했다. SOLAR-A의 4기기 통합 설계 철학은 후속 임무 Hinode(SOLAR-B, 2006), RHESSI(2002), Solar Orbiter(2020)에 직접 계승되었다. 또한 자동 flare-mode 로직과 우선순위 기반 데이터 보호는 현대 모든 태양 미션의 표준이다.

YOHKOH (1991–2001) revolutionized flare physics over its decade of operation. SXT visually confirmed cusp-shaped reconnection structures (Tsuneta 1996), sigmoids, X-ray jets, and coronal hole evolution. HXT simultaneously imaged footpoint and looptop hard X-ray sources (Masuda 1994), providing decisive evidence for the standard flare model. The four-instrument coordinated payload philosophy directly inspired Hinode (SOLAR-B, 2006), RHESSI (2002), and Solar Orbiter (2020). The automated flare-mode logic with priority-based data protection became the standard for all subsequent solar missions.
