# Auto-Encoding Variational Bayes: Reading Notes / 읽기 노트

---

## 1. 핵심 기여 / Core Contribution

이 논문은 연속적인 latent variable을 가진 directed probabilistic model에서 효율적인 추론과 학습을 위한 두 가지 핵심 기법을 제시한다: (1) **Stochastic Gradient Variational Bayes (SGVB) estimator** — variational lower bound (ELBO)의 reparameterization을 통해 표준 확률적 경사하강법으로 최적화할 수 있는 미분 가능한 추정량, (2) **Auto-Encoding Variational Bayes (AEVB) algorithm** — SGVB 추정량을 사용하여 recognition model (encoder)과 generative model (decoder)을 동시에 학습하는 알고리즘. Recognition model로 neural network을 사용하면 **Variational Auto-Encoder (VAE)**가 된다. 이 방법은 posterior가 intractable한 모델에서도 작동하며, 대규모 데이터셋에 확장 가능하고, 데이터포인트당 비싼 반복적 추론(MCMC 등)이 필요 없다.

This paper presents two key techniques for efficient inference and learning in directed probabilistic models with continuous latent variables: (1) the **Stochastic Gradient Variational Bayes (SGVB) estimator** — a differentiable estimator of the variational lower bound (ELBO) that can be optimized via standard stochastic gradient descent through reparameterization, and (2) the **Auto-Encoding Variational Bayes (AEVB) algorithm** — which jointly trains a recognition model (encoder) and generative model (decoder) using the SGVB estimator. When a neural network serves as the recognition model, the result is the **Variational Auto-Encoder (VAE)**. The method works even when the posterior is intractable, scales to large datasets, and eliminates the need for expensive per-datapoint iterative inference (e.g., MCMC).

## 2. 읽기 노트 / Reading Notes

### §1 Introduction (pp. 1)

논문은 핵심 질문으로 시작한다: "연속적인 latent variable과 intractable posterior를 가진 directed probabilistic model에서 어떻게 효율적인 근사 추론과 학습을 수행할 수 있는가?"

The paper opens with the central question: "How can we perform efficient approximate inference and learning with directed probabilistic models whose continuous latent variables and/or parameters have intractable posterior distributions?"

기존 방법의 한계:
- **Variational Bayesian (VB) approach**: ELBO를 최적화하지만, mean-field approximation에서 approximate posterior에 대한 기댓값이 해석적으로 계산 불가능한 경우가 많음
- **Monte Carlo EM**: 데이터포인트당 비싼 sampling loop 필요 → 대규모 데이터에 부적합

Limitations of existing methods:
- **Variational Bayesian (VB) approach**: optimizes the ELBO, but expectations w.r.t. the approximate posterior are often analytically intractable in the mean-field case
- **Monte Carlo EM**: requires an expensive sampling loop per datapoint → unsuitable for large datasets

저자들의 해결책은 두 가지 혁신에 기반한다:
1. **Reparameterization trick**: variational lower bound의 미분 가능한 비편향 추정량을 얻는 방법
2. **Amortized inference**: 개별 데이터포인트마다 최적화하는 대신, 하나의 recognition model (neural network)이 모든 데이터포인트에 대해 approximate posterior를 생성

The authors' solution rests on two innovations:
1. **Reparameterization trick**: a method to obtain a differentiable unbiased estimator of the variational lower bound
2. **Amortized inference**: instead of optimizing per datapoint, a single recognition model (neural network) produces the approximate posterior for all datapoints

### §2.1 Problem Scenario (pp. 2)

데이터셋 $\mathbf{X} = \{\mathbf{x}^{(i)}\}_{i=1}^{N}$이 관측되지 않은 연속 확률변수 $\mathbf{z}$를 포함하는 random process에서 생성된다고 가정한다. 생성 과정은 두 단계이다:
1. $\mathbf{z}^{(i)} \sim p_{\theta^*}(\mathbf{z})$ — prior에서 latent variable 샘플링
2. $\mathbf{x}^{(i)} \sim p_{\theta^*}(\mathbf{x}|\mathbf{z})$ — conditional distribution에서 데이터 생성

We assume dataset $\mathbf{X} = \{\mathbf{x}^{(i)}\}_{i=1}^{N}$ is generated by a random process involving an unobserved continuous random variable $\mathbf{z}$. The generation process has two steps:
1. $\mathbf{z}^{(i)} \sim p_{\theta^*}(\mathbf{z})$ — sample latent variable from prior
2. $\mathbf{x}^{(i)} \sim p_{\theta^*}(\mathbf{x}|\mathbf{z})$ — generate data from conditional distribution

논문은 기존 방법들이 실패하는 두 가지 시나리오를 명시한다:

The paper identifies two scenarios where existing methods fail:

1. **Intractability**: marginal likelihood $p_\theta(\mathbf{x}) = \int p_\theta(\mathbf{z})p_\theta(\mathbf{x}|\mathbf{z})\,d\mathbf{z}$가 intractable (적분 불가). True posterior $p_\theta(\mathbf{z}|\mathbf{x})$도 intractable. EM algorithm이나 mean-field VB도 사용 불가. 이는 likelihood function $p_\theta(\mathbf{x}|\mathbf{z})$가 nonlinear hidden layer를 가진 neural network일 때 흔히 발생.

   Intractability: marginal likelihood $p_\theta(\mathbf{x}) = \int p_\theta(\mathbf{z})p_\theta(\mathbf{x}|\mathbf{z})\,d\mathbf{z}$ is intractable. True posterior $p_\theta(\mathbf{z}|\mathbf{x})$ is also intractable. Neither EM nor mean-field VB can be used. This commonly occurs when $p_\theta(\mathbf{x}|\mathbf{z})$ is a neural network with nonlinear hidden layers.

2. **Large dataset**: batch 최적화가 너무 비쌈. Minibatch 또는 단일 데이터포인트 단위 업데이트 필요. Monte Carlo EM은 데이터포인트당 sampling loop가 필요하므로 느림.

   Large dataset: batch optimization is too costly. Updates via small minibatches or single datapoints are needed. Monte Carlo EM is too slow due to its per-datapoint sampling loop.

해결하고자 하는 세 가지 관련 문제:
1. 파라미터 $\theta$의 효율적 ML/MAP 추정
2. 관측된 $\mathbf{x}$에 대한 latent variable $\mathbf{z}$의 효율적 posterior 추론
3. 변수 $\mathbf{x}$의 효율적 marginal 추론 (denoising, inpainting, super-resolution 등에 활용)

Three related problems to solve:
1. Efficient ML/MAP estimation of parameters $\theta$
2. Efficient posterior inference of $\mathbf{z}$ given observed $\mathbf{x}$
3. Efficient marginal inference of $\mathbf{x}$ (useful for denoising, inpainting, super-resolution)

### §2.2 The Variational Bound (pp. 3)

Recognition model $q_\phi(\mathbf{z}|\mathbf{x})$를 도입하여 intractable true posterior $p_\theta(\mathbf{z}|\mathbf{x})$를 근사한다. 코딩 이론 관점에서 $q_\phi(\mathbf{z}|\mathbf{x})$는 **encoder** (데이터 → latent code), $p_\theta(\mathbf{x}|\mathbf{z})$는 **decoder** (latent code → 데이터)로 해석된다.

We introduce a recognition model $q_\phi(\mathbf{z}|\mathbf{x})$ to approximate the intractable true posterior $p_\theta(\mathbf{z}|\mathbf{x})$. From a coding theory perspective, $q_\phi(\mathbf{z}|\mathbf{x})$ is the **encoder** (data → latent code) and $p_\theta(\mathbf{x}|\mathbf{z})$ is the **decoder** (latent code → data).

Marginal likelihood의 분해 (수식 1):

Decomposition of the marginal likelihood (equation 1):

$$\log p_\theta(\mathbf{x}^{(i)}) = D_{KL}(q_\phi(\mathbf{z}|\mathbf{x}^{(i)}) \| p_\theta(\mathbf{z}|\mathbf{x}^{(i)})) + \mathcal{L}(\theta, \phi; \mathbf{x}^{(i)})$$

- 첫째 항: approximate posterior와 true posterior 사이의 KL divergence (항상 ≥ 0)
- 둘째 항: **variational lower bound (ELBO)** — $\log p_\theta(\mathbf{x}^{(i)})$의 하한

- First term: KL divergence between approximate and true posterior (always ≥ 0)
- Second term: **variational lower bound (ELBO)** — lower bound on $\log p_\theta(\mathbf{x}^{(i)})$

ELBO를 두 가지 형태로 쓸 수 있다:

The ELBO can be written in two forms:

**형태 1** (수식 2): $\mathcal{L}(\theta, \phi; \mathbf{x}^{(i)}) = \mathbb{E}_{q_\phi(\mathbf{z}|\mathbf{x})}[-\log q_\phi(\mathbf{z}|\mathbf{x}) + \log p_\theta(\mathbf{x}, \mathbf{z})]$

**형태 2** (수식 3): $\mathcal{L}(\theta, \phi; \mathbf{x}^{(i)}) = -D_{KL}(q_\phi(\mathbf{z}|\mathbf{x}^{(i)}) \| p_\theta(\mathbf{z})) + \mathbb{E}_{q_\phi(\mathbf{z}|\mathbf{x}^{(i)})}[\log p_\theta(\mathbf{x}^{(i)}|\mathbf{z})]$

형태 2가 더 직관적이다: KL regularization + reconstruction error.

Form 2 is more intuitive: KL regularization + reconstruction error.

**naive Monte Carlo gradient의 문제**: ELBO의 gradient를 naive하게 추정하면 ($\nabla_\phi \mathbb{E}_{q_\phi}[f(\mathbf{z})] = \mathbb{E}_{q_\phi}[f(\mathbf{z})\nabla_{q_\phi}\log q_\phi(\mathbf{z})]$) 매우 높은 분산을 가져 실용적이지 않다 (REINFORCE/score function estimator).

The naive Monte Carlo gradient estimator of the ELBO exhibits very high variance (REINFORCE/score function estimator), making it impractical.

### §2.3 The SGVB Estimator and AEVB Algorithm (pp. 3–4)

**Reparameterization trick** (§2.4에서 상세히 설명)을 사용하여 $\mathbf{z} \sim q_\phi(\mathbf{z}|\mathbf{x})$를 $\tilde{\mathbf{z}} = g_\phi(\boldsymbol{\epsilon}, \mathbf{x})$, $\boldsymbol{\epsilon} \sim p(\boldsymbol{\epsilon})$으로 재매개변수화한다. 이를 통해 low-variance gradient estimator를 얻는다.

Using the **reparameterization trick** (detailed in §2.4), we reparameterize $\mathbf{z} \sim q_\phi(\mathbf{z}|\mathbf{x})$ as $\tilde{\mathbf{z}} = g_\phi(\boldsymbol{\epsilon}, \mathbf{x})$ with $\boldsymbol{\epsilon} \sim p(\boldsymbol{\epsilon})$, yielding a low-variance gradient estimator.

**SGVB Estimator — 두 가지 버전:**

**Version A** (generic, 수식 6): ELBO 형태 1 (수식 2)에 reparameterization trick 적용:

$$\tilde{\mathcal{L}}^A(\theta, \phi; \mathbf{x}^{(i)}) = \frac{1}{L}\sum_{l=1}^{L}\log p_\theta(\mathbf{x}^{(i)}, \mathbf{z}^{(i,l)}) - \log q_\phi(\mathbf{z}^{(i,l)}|\mathbf{x}^{(i)})$$

where $\mathbf{z}^{(i,l)} = g_\phi(\boldsymbol{\epsilon}^{(l)}, \mathbf{x}^{(i)})$ and $\boldsymbol{\epsilon}^{(l)} \sim p(\boldsymbol{\epsilon})$.

**Version B** (lower variance, 수식 7): ELBO 형태 2 (수식 3)에서 KL term을 해석적으로 계산 (가능할 때):

$$\tilde{\mathcal{L}}^B(\theta, \phi; \mathbf{x}^{(i)}) = -D_{KL}(q_\phi(\mathbf{z}|\mathbf{x}^{(i)}) \| p_\theta(\mathbf{z})) + \frac{1}{L}\sum_{l=1}^{L}\log p_\theta(\mathbf{x}^{(i)}|\mathbf{z}^{(i,l)})$$

Version B가 분산이 더 낮아 실제로 선호된다. Gaussian prior와 posterior일 때 KL term은 Appendix B의 해석적 공식으로 계산된다.

Version B has lower variance and is preferred in practice. When prior and posterior are Gaussian, the KL term is computed via the analytical formula in Appendix B.

**Minibatch 추정** (수식 8):

$$\mathcal{L}(\theta, \phi; \mathbf{X}) \simeq \tilde{\mathcal{L}}^M(\theta, \phi; \mathbf{X}^M) = \frac{N}{M}\sum_{i=1}^{M}\tilde{\mathcal{L}}(\theta, \phi; \mathbf{x}^{(i)})$$

실험에서 minibatch size $M = 100$, samples per datapoint $L = 1$로 충분했다.

In experiments, minibatch size $M = 100$ and $L = 1$ sample per datapoint sufficed.

**Algorithm 1 — AEVB 알고리즘:**
1. $\theta, \phi$ 초기화
2. 반복: minibatch $\mathbf{X}^M$ 추출 → noise $\boldsymbol{\epsilon}$ 샘플링 → gradient $\mathbf{g} = \nabla_{\theta,\phi}\tilde{\mathcal{L}}^M$ 계산 → SGD/Adagrad로 파라미터 업데이트
3. 수렴할 때까지 반복

1. Initialize $\theta, \phi$
2. Repeat: draw minibatch $\mathbf{X}^M$ → sample noise $\boldsymbol{\epsilon}$ → compute gradient $\mathbf{g} = \nabla_{\theta,\phi}\tilde{\mathcal{L}}^M$ → update parameters via SGD/Adagrad
3. Until convergence

**Auto-encoder와의 연결**: 수식 7의 관점에서 보면:
- 첫째 항 (KL divergence): prior로부터의 이탈에 대한 **정규화** 역할
- 둘째 항: $q_\phi$가 $\mathbf{x}^{(i)}$를 $\boldsymbol{\epsilon}^{(l)}$과 함께 latent code $\mathbf{z}^{(i,l)}$로 매핑하고, decoder $p_\theta$가 이를 다시 $\mathbf{x}^{(i)}$로 복원 → **negative reconstruction error**

Connection to auto-encoders: from the perspective of equation 7:
- First term (KL divergence): **regularizer** penalizing deviation from the prior
- Second term: $q_\phi$ maps $\mathbf{x}^{(i)}$ with noise $\boldsymbol{\epsilon}^{(l)}$ to latent code $\mathbf{z}^{(i,l)}$, and decoder $p_\theta$ reconstructs back to $\mathbf{x}^{(i)}$ → **negative reconstruction error**

### §2.4 The Reparameterization Trick (pp. 4–5)

이 논문의 핵심 기술적 혁신이다. 문제: $q_\phi(\mathbf{z}|\mathbf{x})$에서 직접 sampling하면 $\phi$에 대한 gradient가 흐르지 않는다 (sampling은 미분 불가능한 연산).

The core technical innovation of the paper. Problem: directly sampling from $q_\phi(\mathbf{z}|\mathbf{x})$ blocks gradients to $\phi$ (sampling is a non-differentiable operation).

**해결책**: 확률변수 $\mathbf{z}$를 deterministic 변수 $\tilde{\mathbf{z}} = g_\phi(\boldsymbol{\epsilon}, \mathbf{x})$로 표현하되, 확률성은 보조 noise 변수 $\boldsymbol{\epsilon} \sim p(\boldsymbol{\epsilon})$에 담는다.

**Solution**: express the random variable $\mathbf{z}$ as a deterministic variable $\tilde{\mathbf{z}} = g_\phi(\boldsymbol{\epsilon}, \mathbf{x})$, pushing the randomness into an auxiliary noise variable $\boldsymbol{\epsilon} \sim p(\boldsymbol{\epsilon})$.

**Gaussian 예시**: $\mathbf{z} \sim q_\phi(\mathbf{z}|\mathbf{x}) = \mathcal{N}(\boldsymbol{\mu}, \boldsymbol{\sigma}^2\mathbf{I})$일 때:

$$\mathbf{z} = \boldsymbol{\mu} + \boldsymbol{\sigma} \odot \boldsymbol{\epsilon}, \quad \boldsymbol{\epsilon} \sim \mathcal{N}(\mathbf{0}, \mathbf{I})$$

이 변환이 유효한 이유의 증명: $q_\phi(\mathbf{z}|\mathbf{x})\prod_i dz_i = p(\boldsymbol{\epsilon})\prod_i d\epsilon_i$이므로 $\int q_\phi(\mathbf{z}|\mathbf{x})f(\mathbf{z})\,d\mathbf{z} = \int p(\boldsymbol{\epsilon})f(g_\phi(\boldsymbol{\epsilon}, \mathbf{x}))\,d\boldsymbol{\epsilon}$. 따라서:

$$\mathbb{E}_{q_\phi(\mathbf{z}|\mathbf{x})}[f(\mathbf{z})] \simeq \frac{1}{L}\sum_{l=1}^{L}f(g_\phi(\boldsymbol{\epsilon}^{(l)}, \mathbf{x})), \quad \boldsymbol{\epsilon}^{(l)} \sim p(\boldsymbol{\epsilon})$$

이 추정량은 $g_\phi$와 $f$가 미분 가능하면 $\phi$에 대해 미분 가능하다.

This estimator is differentiable w.r.t. $\phi$ as long as $g_\phi$ and $f$ are differentiable.

```

변환 전 (미분 불가):           변환 후 (미분 가능):
Before (non-differentiable):   After (differentiable):

  x ──→ encoder ──→ μ, σ         x ──→ encoder ──→ μ, σ
                     │                                │  │
                     ▼                                │  │
              z ~ N(μ, σ²)        ε ~ N(0,1) ──→ z = μ + σ⊙ε
              (sampling 차단)                    (deterministic, 미분 가능)
                     │                                │
                     ▼                                ▼
                  decoder                          decoder
```

$g_\phi(\cdot)$로 사용할 수 있는 세 가지 접근법:
1. **Inverse CDF**: $\boldsymbol{\epsilon} \sim \mathcal{U}(\mathbf{0}, \mathbf{I})$, $g_\phi = $ inverse CDF of $q_\phi$
2. **Location-scale family**: $\boldsymbol{\epsilon} \sim$ standard distribution, $g = $ location + scale $\cdot \boldsymbol{\epsilon}$ (Gaussian, Laplace, Logistic 등)
3. **Composition**: 보조 변수의 변환으로 표현 (Log-Normal, Gamma, Dirichlet 등)

Three approaches for choosing $g_\phi(\cdot)$:
1. **Inverse CDF**: $\boldsymbol{\epsilon} \sim \mathcal{U}(\mathbf{0}, \mathbf{I})$, $g_\phi = $ inverse CDF of $q_\phi$
2. **Location-scale family**: $\boldsymbol{\epsilon} \sim$ standard distribution, $g = $ location + scale $\cdot \boldsymbol{\epsilon}$ (Gaussian, Laplace, Logistic, etc.)
3. **Composition**: express as transformations of auxiliary variables (Log-Normal, Gamma, Dirichlet, etc.)

### §3 Example: Variational Auto-Encoder (pp. 5)

구체적인 VAE 구현 예시이다. 세 가지 선택:

A concrete VAE implementation example. Three choices:

1. **Prior**: $p_\theta(\mathbf{z}) = \mathcal{N}(\mathbf{z}; \mathbf{0}, \mathbf{I})$ — centered isotropic multivariate Gaussian
2. **Encoder**: $q_\phi(\mathbf{z}|\mathbf{x})$ — MLP가 $\boldsymbol{\mu}$와 $\log \boldsymbol{\sigma}^2$를 출력
3. **Decoder**: $p_\theta(\mathbf{x}|\mathbf{z})$ — Bernoulli MLP (이진 데이터) 또는 Gaussian MLP (연속 데이터)

**Encoder 구조** (수식 9):

$$\log q_\phi(\mathbf{z}|\mathbf{x}^{(i)}) = \log \mathcal{N}(\mathbf{z}; \boldsymbol{\mu}^{(i)}, \boldsymbol{\sigma}^{2(i)}\mathbf{I})$$

$\boldsymbol{\mu}^{(i)}$와 $\boldsymbol{\sigma}^{(i)}$는 encoding MLP의 출력이다.

$\boldsymbol{\mu}^{(i)}$ and $\boldsymbol{\sigma}^{(i)}$ are outputs of the encoding MLP.

**최종 SGVB Estimator** (수식 10) — Gaussian prior와 posterior일 때의 구체적 형태:

$$\mathcal{L}(\theta, \phi; \mathbf{x}^{(i)}) \simeq \frac{1}{2}\sum_{j=1}^{J}\left(1 + \log((\sigma_j^{(i)})^2) - (\mu_j^{(i)})^2 - (\sigma_j^{(i)})^2\right) + \frac{1}{L}\sum_{l=1}^{L}\log p_\theta(\mathbf{x}^{(i)}|\mathbf{z}^{(i,l)})$$

where $\mathbf{z}^{(i,l)} = \boldsymbol{\mu}^{(i)} + \boldsymbol{\sigma}^{(i)} \odot \boldsymbol{\epsilon}^{(l)}$ and $\boldsymbol{\epsilon}^{(l)} \sim \mathcal{N}(\mathbf{0}, \mathbf{I})$.

이 수식이 VAE의 실제 loss function이다. 첫째 항은 해석적으로 계산되고 (KL regularization), 둘째 항은 sampling으로 추정된다 (reconstruction).

This equation is the actual VAE loss function. The first term is computed analytically (KL regularization), and the second term is estimated by sampling (reconstruction).

### §4 Related Work (pp. 6)

**Wake-sleep algorithm** (Hinton et al., 1995): VAE와 가장 유사한 선행 연구. Recognition model로 posterior를 근사하는 유일한 on-line 학습 방법이었다. 단점: 두 개의 서로 다른 목적함수를 동시 최적화해야 함 (marginal likelihood의 bound를 최적화하는 것이 아님). VAE와 같은 계산 복잡도.

**Wake-sleep algorithm** (Hinton et al., 1995): most similar prior work. The only other on-line learning method applicable to the same general class of continuous latent variable models. Drawback: requires concurrent optimization of two separate objective functions (neither of which corresponds to optimization of a bound on the marginal likelihood). Same computational complexity as VAE.

**Stochastic variational inference** (Hoffman et al., 2013): 최근 연구. Control variate를 통해 naive gradient estimator의 높은 분산 문제를 완화. VAE의 reparameterization trick과 유사한 기법이 Salimans & Knowles (2013)에서 독립적으로 제안됨.

**Linear auto-encoders와의 연결**: regularization이 없는 autoencoder의 학습 criterion은 mutual information의 하한을 최대화하는 것과 동일. 하지만 reconstruction criterion만으로는 유용한 representation을 학습하기에 불충분. VAE의 ELBO는 variational bound에서 유래한 정규화 항을 자동으로 포함.

### §5 Experiments (pp. 6–8)

**데이터셋**: MNIST (28×28 이진 이미지, 60K/10K train/test)와 Frey Face (20×28 연속 이미지)

**실험 설정**:
- Encoder와 decoder: 동일한 수의 hidden units
- MNIST: 500 hidden units, Frey Face: 200 hidden units
- 초기화: $\mathcal{N}(0, 0.01)$에서 random sampling
- 최적화: Adagrad (step sizes from {0.01, 0.02, 0.1})
- MAP 추정: weight decay $p(\theta) = \mathcal{N}(\mathbf{0}, \mathbf{I})$에 해당
- $M = 100$, $L = 1$

**Experiment settings**:
- Encoder and decoder: equal number of hidden units
- MNIST: 500 hidden units, Frey Face: 200 hidden units
- Initialization: random from $\mathcal{N}(0, 0.01)$
- Optimization: Adagrad (step sizes from {0.01, 0.02, 0.1})
- MAP estimation: weight decay corresponding to $p(\theta) = \mathcal{N}(\mathbf{0}, \mathbf{I})$
- $M = 100$, $L = 1$

- AEVB가 wake-sleep보다 모든 실험에서 **빠르게 수렴**하고 **더 나은 해**에 도달
- Latent space 차원 $N_z$가 증가해도 overfitting 없음 — ELBO의 정규화 효과

- 계산 비용: Intel Xeon CPU에서 100만 training sample당 20–40분, ~40 GFLOPS

Results 1 — Variational Lower Bound (Figure 2):
- AEVB converges **faster** and reaches **better solutions** than wake-sleep in all experiments
- No overfitting even as latent space dimensionality $N_z$ increases — due to ELBO's regularization effect
- Tested with $N_z = 3, 5, 10, 20, 200$

- Also compared with Monte Carlo EM (MCEM): MCEM is not on-line, inefficient for large data
- AEVB reaches marginal likelihood comparable to MCEM, but faster
- Superior in both small ($N_{train} = 1000$) and full ($N_{train} = 50000$) dataset settings

**결과 3 — Visualisation (Appendix A, Figures 4 & 5)**:
- 2D latent space: Frey Face와 MNIST의 learned manifold 시각화
- 잠재 공간에서 linearly spaced coordinates를 Gaussian inverse CDF로 변환하여 다양한 이미지 생성
- 높은 차원의 latent space (5D, 10D, 20D)에서 random sampling한 MNIST 이미지도 품질 양호

Results 3 — Visualisation (Appendix A, Figures 4 & 5):
- 2D latent space: visualized learned manifolds for Frey Face and MNIST
- Linearly spaced coordinates transformed through Gaussian inverse CDF generate diverse images
- Random samples from higher-dimensional latent spaces (5D, 10D, 20D) also show good quality

### Appendix B: KL Divergence (Gaussian Case) (pp. 10–11)

Prior $p_\theta(\mathbf{z}) = \mathcal{N}(\mathbf{0}, \mathbf{I})$와 posterior $q_\phi(\mathbf{z}|\mathbf{x}^{(i)}) = \mathcal{N}(\boldsymbol{\mu}, \boldsymbol{\sigma}^2)$가 모두 Gaussian일 때 KL divergence의 해석적 해를 유도한다.

Derives the analytical solution for KL divergence when both prior $p_\theta(\mathbf{z}) = \mathcal{N}(\mathbf{0}, \mathbf{I})$ and posterior $q_\phi(\mathbf{z}|\mathbf{x}^{(i)}) = \mathcal{N}(\boldsymbol{\mu}, \boldsymbol{\sigma}^2)$ are Gaussian.

두 항을 각각 계산:

Computing two terms separately:

$$\int q_\theta(\mathbf{z})\log p(\mathbf{z})\,d\mathbf{z} = -\frac{J}{2}\log(2\pi) - \frac{1}{2}\sum_{j=1}^{J}(\mu_j^2 + \sigma_j^2)$$

$$\int q_\theta(\mathbf{z})\log q_\theta(\mathbf{z})\,d\mathbf{z} = -\frac{J}{2}\log(2\pi) - \frac{1}{2}\sum_{j=1}^{J}(1 + \log \sigma_j^2)$$

따라서 / Therefore:

$$-D_{KL}(q_\phi(\mathbf{z}) \| p_\theta(\mathbf{z})) = \frac{1}{2}\sum_{j=1}^{J}\left(1 + \log(\sigma_j^2) - \mu_j^2 - \sigma_j^2\right)$$

이 결과가 수식 (10)의 첫째 항에 직접 사용된다. 구현 시 이 공식만으로 KL term을 계산할 수 있어 매우 효율적이다.

This result is directly used as the first term of equation (10). In implementation, the KL term can be computed solely with this formula, making it very efficient.

### Appendix C: MLP Encoders and Decoders (pp. 11)

**Bernoulli MLP Decoder** (이진 데이터, 수식 11):

$$\log p(\mathbf{x}|\mathbf{z}) = \sum_{i=1}^{D} x_i \log y_i + (1-x_i) \cdot \log(1-y_i)$$
$$\text{where } \mathbf{y} = f_\sigma(\mathbf{W}_2 \tanh(\mathbf{W}_1\mathbf{z} + \mathbf{b}_1) + \mathbf{b}_2)$$

$f_\sigma(\cdot)$: sigmoid activation. $\theta = \{\mathbf{W}_1, \mathbf{W}_2, \mathbf{b}_1, \mathbf{b}_2\}$.

**Gaussian MLP Encoder/Decoder** (연속 데이터, 수식 12):

$$\log p(\mathbf{x}|\mathbf{z}) = \log \mathcal{N}(\mathbf{x}; \boldsymbol{\mu}, \boldsymbol{\sigma}^2\mathbf{I})$$
$$\boldsymbol{\mu} = \mathbf{W}_4\mathbf{h} + \mathbf{b}_4, \quad \log \boldsymbol{\sigma}^2 = \mathbf{W}_5\mathbf{h} + \mathbf{b}_5, \quad \mathbf{h} = \tanh(\mathbf{W}_3\mathbf{z} + \mathbf{b}_3)$$

Encoder로 사용할 때는 $\mathbf{z}$와 $\mathbf{x}$를 교환하고, 가중치들이 variational parameter $\phi$가 된다.

When used as encoder, $\mathbf{z}$ and $\mathbf{x}$ are swapped, and the weights become variational parameters $\phi$.

### Appendix D: Marginal Likelihood Estimator (pp. 11–12)

Low-dimensional latent space ($< 5$ dimensions)에서 marginal likelihood를 추정하는 방법을 제시한다.

Presents a method to estimate the marginal likelihood in low-dimensional latent spaces ($< 5$ dimensions).

3단계 과정:
1. Gradient-based MCMC (HMC)로 posterior에서 $L$개 sample 추출
2. 이 sample들에 density estimator $q(\mathbf{z})$ 적합
3. 새로운 posterior sample과 $q(\mathbf{z})$를 사용하여 추정:

$$p_\theta(\mathbf{x}^{(i)}) \simeq \left(\frac{1}{L}\sum_{l=1}^{L}\frac{q(\mathbf{z}^{(l)})}{p_\theta(\mathbf{z})p_\theta(\mathbf{x}^{(i)}|\mathbf{z}^{(l)})}\right)^{-1}$$

### Appendix F: Full VB (pp. 12–14)

파라미터 $\theta$와 latent variable $\mathbf{z}$ 모두에 대해 variational inference를 수행하는 확장을 제시한다. $\theta$에도 prior $p_\alpha(\theta)$를 두고 approximate posterior $q_\phi(\theta)$로 근사한다.

Presents the extension of performing variational inference on both parameters $\theta$ and latent variables $\mathbf{z}$. A prior $p_\alpha(\theta)$ is placed on $\theta$, approximated by $q_\phi(\theta)$.

Reparameterization trick을 $q_\phi(\theta)$와 $q_\phi(\mathbf{z}|\mathbf{x})$ 모두에 적용:

$$\tilde{\theta} = h_\phi(\boldsymbol{\zeta}), \quad \boldsymbol{\zeta} \sim p(\boldsymbol{\zeta})$$
$$\tilde{\mathbf{z}} = g_\phi(\boldsymbol{\epsilon}, \mathbf{x}), \quad \boldsymbol{\epsilon} \sim p(\boldsymbol{\epsilon})$$

본문에서는 $\theta$에 대한 point estimate (MAP)를 사용하지만, 이 부록은 fully Bayesian 확장이 가능함을 보여준다.

The main text uses point estimates (MAP) for $\theta$, but this appendix shows that a fully Bayesian extension is possible.

## 3. 핵심 시사점 / Key Takeaways

1. **Reparameterization trick은 확률적 computation graph의 미분을 가능하게 한다**: $\mathbf{z} = \boldsymbol{\mu} + \boldsymbol{\sigma} \odot \boldsymbol{\epsilon}$으로 변환함으로써 sampling이라는 미분 불가능한 연산을 deterministic + 외부 noise로 분해한다. 이 아이디어는 VAE를 넘어 deep learning 전반에 광범위하게 사용된다 (Gumbel-Softmax, normalizing flows, diffusion models 등).

   The reparameterization trick enables differentiation through stochastic computation graphs: by transforming $\mathbf{z} = \boldsymbol{\mu} + \boldsymbol{\sigma} \odot \boldsymbol{\epsilon}$, the non-differentiable sampling operation is decomposed into a deterministic function plus external noise. This idea extends far beyond VAE into deep learning broadly (Gumbel-Softmax, normalizing flows, diffusion models, etc.).

2. **Amortized inference는 추론의 확장성을 혁신적으로 개선한다**: 기존 variational inference는 데이터포인트마다 별도의 variational parameter를 최적화해야 했다 (예: mean-field). VAE는 하나의 encoder network가 모든 입력에 대해 posterior를 즉시 추론하여, $O(N)$의 파라미터 수를 network 크기로 대체한다.

   Amortized inference revolutionizes inference scalability: traditional variational inference optimized separate variational parameters per datapoint (e.g., mean-field). VAE uses a single encoder network that instantly infers the posterior for any input, replacing $O(N)$ parameters with the network size.

3. **ELBO는 생성 모델의 통합된 학습 목적함수를 제공한다**: ELBO = -KL + Reconstruction으로, 하나의 목적함수가 동시에 (a) encoder를 prior에 가깝게 정규화하고, (b) decoder의 재구성 품질을 최대화한다. Wake-sleep과 달리 단일 목적함수를 최적화한다.

   The ELBO provides a unified training objective for generative models: ELBO = -KL + Reconstruction, where a single objective simultaneously (a) regularizes the encoder toward the prior and (b) maximizes reconstruction quality. Unlike wake-sleep, it optimizes a single objective.

4. **KL divergence가 자동 정규화 역할을 한다**: $D_{KL}(q_\phi \| p_\theta)$ 항이 latent space를 구조화한다. 실험에서 latent 차원이 증가해도 overfitting이 발생하지 않은 것은 불필요한 latent 차원이 자동으로 prior에 수렴하기 때문이다 (자동 관련성 결정, automatic relevance determination과 유사).

   KL divergence acts as automatic regularization: the $D_{KL}(q_\phi \| p_\theta)$ term structures the latent space. In experiments, increasing latent dimensions did not cause overfitting because unnecessary latent dimensions automatically collapse to the prior (similar to automatic relevance determination).

5. **Gaussian 가정이 실용적 계산 효율성의 핵심이다**: Prior와 posterior 모두 Gaussian일 때 KL term을 해석적으로 계산할 수 있어 (Appendix B), 전체 loss에서 sampling이 필요한 부분은 reconstruction term 하나뿐이다. 이로써 $L=1$ sample로도 충분한 성능을 달성한다.

   The Gaussian assumption is key to practical computational efficiency: when both prior and posterior are Gaussian, the KL term can be computed analytically (Appendix B), so sampling is only needed for the reconstruction term. This makes $L=1$ sample sufficient for good performance.

6. **VAE는 확률적 프레임워크 안에서 representation learning을 수행한다**: 기존 autoencoder는 deterministic mapping을 학습하지만, VAE는 latent space에서 분포를 학습한다. 이 확률적 구조 덕분에 새로운 데이터를 생성하고, latent space를 부드럽게(smooth) 만들며, 불확실성을 정량화할 수 있다.

   VAE performs representation learning within a probabilistic framework: unlike traditional autoencoders that learn deterministic mappings, VAE learns distributions in latent space. This probabilistic structure enables generating new data, creating smooth latent spaces, and quantifying uncertainty.

7. **논문의 간결함이 핵심 아이디어의 범용성을 반영한다**: 14페이지의 비교적 짧은 논문에 매우 일반적인 프레임워크를 담고 있다. SGVB estimator는 연속 latent variable을 가진 거의 모든 directed probabilistic model에 적용 가능하며, 특정 모델 구조에 의존하지 않는다.

   The paper's conciseness reflects the generality of the core idea: a relatively short 14-page paper contains a very general framework. The SGVB estimator is applicable to almost any directed probabilistic model with continuous latent variables and does not depend on specific model architectures.

## 4. 수학적 요약 / Mathematical Summary

### 수식 1: Marginal Likelihood 분해 / Marginal Likelihood Decomposition

$$\log p_\theta(\mathbf{x}^{(i)}) = D_{KL}(q_\phi(\mathbf{z}|\mathbf{x}^{(i)}) \| p_\theta(\mathbf{z}|\mathbf{x}^{(i)})) + \mathcal{L}(\theta, \phi; \mathbf{x}^{(i)})$$

- $\log p_\theta(\mathbf{x}^{(i)})$: 데이터포인트 $\mathbf{x}^{(i)}$의 log marginal likelihood / log marginal likelihood of datapoint $\mathbf{x}^{(i)}$
- $D_{KL}(\cdot \| \cdot)$: approximate posterior와 true posterior 사이의 KL divergence (≥ 0) / KL divergence between approximate and true posterior (≥ 0)
- $\mathcal{L}$: variational lower bound (ELBO) — 최대화 대상 / variational lower bound (ELBO) — the quantity to maximize

**유도 / Derivation**:

$$\log p_\theta(\mathbf{x}^{(i)}) = \mathbb{E}_{q_\phi(\mathbf{z}|\mathbf{x})}[\log p_\theta(\mathbf{x}^{(i)})]$$
$$= \mathbb{E}_{q_\phi}[\log \frac{p_\theta(\mathbf{x}^{(i)}, \mathbf{z})}{p_\theta(\mathbf{z}|\mathbf{x}^{(i)})}]$$
$$= \mathbb{E}_{q_\phi}[\log \frac{p_\theta(\mathbf{x}^{(i)}, \mathbf{z})}{q_\phi(\mathbf{z}|\mathbf{x}^{(i)})} \cdot \frac{q_\phi(\mathbf{z}|\mathbf{x}^{(i)})}{p_\theta(\mathbf{z}|\mathbf{x}^{(i)})}]$$
$$= \underbrace{\mathbb{E}_{q_\phi}[\log \frac{p_\theta(\mathbf{x}^{(i)}, \mathbf{z})}{q_\phi(\mathbf{z}|\mathbf{x}^{(i)})}]}_{\mathcal{L}} + \underbrace{\mathbb{E}_{q_\phi}[\log \frac{q_\phi(\mathbf{z}|\mathbf{x}^{(i)})}{p_\theta(\mathbf{z}|\mathbf{x}^{(i)})}]}_{D_{KL}}$$

### 수식 2: ELBO (형태 2) / ELBO (Form 2)

$$\mathcal{L}(\theta, \phi; \mathbf{x}^{(i)}) = -D_{KL}(q_\phi(\mathbf{z}|\mathbf{x}^{(i)}) \| p_\theta(\mathbf{z})) + \mathbb{E}_{q_\phi(\mathbf{z}|\mathbf{x}^{(i)})}[\log p_\theta(\mathbf{x}^{(i)}|\mathbf{z})]$$

- $-D_{KL}(q_\phi \| p_\theta)$: 정규화 항 — encoder 출력을 prior에 가깝게 유지 / Regularization — keeps encoder output close to prior
- $\mathbb{E}_{q_\phi}[\log p_\theta(\mathbf{x}|\mathbf{z})]$: 기대 재구성 log-likelihood — decoder 품질 측정 / Expected reconstruction log-likelihood — measures decoder quality

### 수식 3: Reparameterization Trick (Gaussian)

$$\mathbf{z} = \boldsymbol{\mu} + \boldsymbol{\sigma} \odot \boldsymbol{\epsilon}, \quad \boldsymbol{\epsilon} \sim \mathcal{N}(\mathbf{0}, \mathbf{I})$$

- $\boldsymbol{\mu}$: encoder가 출력하는 평균 벡터 / mean vector output by encoder
- $\boldsymbol{\sigma}$: encoder가 출력하는 표준편차 벡터 / standard deviation vector output by encoder
- $\boldsymbol{\epsilon}$: 표준 정규분포에서 샘플링한 noise / noise sampled from standard normal
- $\odot$: element-wise 곱 / element-wise product

### 수식 4: KL Divergence (Gaussian, Appendix B)

$$-D_{KL}(q_\phi(\mathbf{z}|\mathbf{x}^{(i)}) \| p_\theta(\mathbf{z})) = \frac{1}{2}\sum_{j=1}^{J}\left(1 + \log(\sigma_j^{(i)})^2 - (\mu_j^{(i)})^2 - (\sigma_j^{(i)})^2\right)$$

- $J$: latent space의 차원 수 / dimensionality of latent space
- $\mu_j^{(i)}$: 데이터포인트 $i$에 대한 encoder 출력 평균의 $j$번째 원소 / $j$-th element of encoder mean for datapoint $i$
- $\sigma_j^{(i)}$: 데이터포인트 $i$에 대한 encoder 출력 표준편차의 $j$번째 원소 / $j$-th element of encoder std for datapoint $i$

### 수식 5: 최종 VAE Loss (수식 10)

$$\mathcal{L}(\theta, \phi; \mathbf{x}^{(i)}) \simeq \frac{1}{2}\sum_{j=1}^{J}\left(1 + \log((\sigma_j^{(i)})^2) - (\mu_j^{(i)})^2 - (\sigma_j^{(i)})^2\right) + \frac{1}{L}\sum_{l=1}^{L}\log p_\theta(\mathbf{x}^{(i)}|\mathbf{z}^{(i,l)})$$

- 첫째 항: 해석적으로 계산 가능한 KL regularization / Analytically computed KL regularization
- 둘째 항: $L$개 sample로 추정하는 reconstruction term / Reconstruction term estimated with $L$ samples
- 실제로 $L = 1$이면 충분 ($M = 100$ minibatch와 함께) / In practice, $L = 1$ suffices (with $M = 100$ minibatch)

### 수식 6: Bernoulli Decoder Loss (수식 11)

$$\log p(\mathbf{x}|\mathbf{z}) = \sum_{i=1}^{D} x_i \log y_i + (1-x_i)\log(1-y_i)$$

- $D$: 데이터 차원 (예: MNIST에서 784 = 28×28) / data dimensionality (e.g., 784 for MNIST)
- $y_i$: decoder MLP의 sigmoid 출력 — 각 pixel이 1일 확률 / sigmoid output of decoder MLP — probability of each pixel being 1
- 이것은 binary cross-entropy loss와 동일 / This is equivalent to binary cross-entropy loss

### 수식 7: Gaussian Decoder (수식 12)

$$\log p(\mathbf{x}|\mathbf{z}) = \log \mathcal{N}(\mathbf{x}; \boldsymbol{\mu}, \boldsymbol{\sigma}^2\mathbf{I})$$

- 연속 데이터 (예: Frey Face)에 사용 / Used for continuous data (e.g., Frey Face)
- $\boldsymbol{\mu}$와 $\log \boldsymbol{\sigma}^2$ 모두 decoder MLP에서 출력 / Both $\boldsymbol{\mu}$ and $\log \boldsymbol{\sigma}^2$ output by decoder MLP
- $\boldsymbol{\sigma}^2$를 고정하면 MSE loss와 비례 / If $\boldsymbol{\sigma}^2$ is fixed, proportional to MSE loss

## 7. 구체적 수치 예시 / Worked Numerical Example

### VAE Forward Pass (MNIST, 1개 데이터포인트)

**설정**: $J = 2$ (2D latent space), 1개의 28×28 이진 이미지

**Step 1 — Encoder**: 입력 $\mathbf{x} \in \{0, 1\}^{784}$를 MLP에 통과시켜 $\boldsymbol{\mu}$와 $\log \boldsymbol{\sigma}^2$를 출력

$$\boldsymbol{\mu} = [0.5, -0.3], \quad \log \boldsymbol{\sigma}^2 = [-1.0, -0.5]$$
$$\boldsymbol{\sigma} = [\exp(-0.5), \exp(-0.25)] = [0.607, 0.779]$$

**Step 2 — Reparameterization**: noise 샘플링 및 $\mathbf{z}$ 계산

$$\boldsymbol{\epsilon} = [0.2, -0.8] \sim \mathcal{N}(0, 1)$$
$$\mathbf{z} = \boldsymbol{\mu} + \boldsymbol{\sigma} \odot \boldsymbol{\epsilon} = [0.5 + 0.607 \times 0.2, \; -0.3 + 0.779 \times (-0.8)]$$
$$= [0.621, -0.923]$$

**Step 3 — KL Divergence 계산** ($J = 2$):

$$-D_{KL} = \frac{1}{2}\sum_{j=1}^{2}(1 + \log \sigma_j^2 - \mu_j^2 - \sigma_j^2)$$

$j=1$: $\frac{1}{2}(1 + (-1.0) - 0.25 - 0.368) = \frac{1}{2}(-0.618) = -0.309$

$j=2$: $\frac{1}{2}(1 + (-0.5) - 0.09 - 0.607) = \frac{1}{2}(-0.197) = -0.099$

$$-D_{KL} = -0.309 + (-0.099) = -0.408$$

KL = 0.408 → encoder가 prior $\mathcal{N}(0,1)$에서 약간 벗어나 있음을 의미.

KL = 0.408 → the encoder deviates slightly from the prior $\mathcal{N}(0,1)$.

**Step 4 — Reconstruction** ($L = 1$): $\mathbf{z} = [0.621, -0.923]$을 decoder MLP에 통과

$$\text{Reconstruction} = \log p_\theta(\mathbf{x}|\mathbf{z}) = \sum_{i=1}^{784} x_i \log y_i + (1-x_i)\log(1-y_i)$$

예를 들어 decoder 출력 $\mathbf{y}$가 원본 $\mathbf{x}$와 잘 일치하면 이 값은 0에 가깝고 (완벽한 재구성), 잘 일치하지 않으면 큰 음수 (나쁜 재구성).

For example, if decoder output $\mathbf{y}$ matches the original $\mathbf{x}$ well, this value is close to 0 (perfect reconstruction); if not, it becomes a large negative number (poor reconstruction).

**Step 5 — 전체 ELBO**:

$$\mathcal{L} = -0.408 + (-85.3) = -85.708$$

학습 목표: 이 값을 **최대화** (= $-\mathcal{L}$을 최소화)

Training objective: **maximize** this value (= minimize $-\mathcal{L}$)
