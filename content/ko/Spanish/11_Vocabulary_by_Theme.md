#### 축소사 형성 규칙

**1. 모음으로 끝나는 단어:**
- 모음 탈락 + -ito/-ita
- casa → casita
- perro → perrito

**2. 자음으로 끝나는 단어:**
- 그대로 + -ito/-ita (또는 -cito/-cita)
- amor → amorcito
- pan → panecito (e 삽입)

**3. -co/-ca로 끝나는 단어:**
- c → qu + -ito/-ita
- poco → poquito
- chico → chiquito

**4. -go/-ga로 끝나는 단어:**
- g → gu + -ito/-ita
- amigo → amiguito
- largo → larguito

**5. 1음절 단어:**
- -ecito/-ecita
- flor → florecita
- pan → panecito
- pez → pececito

## 학습 팁 (Study Tips)

1. **축소사 연습**: 일상 대화에서 친근함을 표현할 때 축소사 활용
2. **확대사 주의**: 확대사는 종종 부정적 뉘앙스가 있으므로 맥락 파악 필수
3. **지역별 차이**: 축소사 선호도가 지역마다 다름 (멕시코: -ito, 카리브: -ico)
4. **관용어 문맥**: 관용어는 반드시 문맥과 함께 학습
5. **현대 어휘**: SNS와 뉴스를 통해 최신 어휘 습득
