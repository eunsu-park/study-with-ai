# Study with AI

A personal study project for learning diverse subjects from beginner to expert level, with AI (Claude) as a study partner.

The core method: **read landmark papers chronologically, understand the ideas, and implement them in code** — building deep knowledge from first principles.

## How It Works

### Paper-Driven Learning

Each topic is structured around a curated list of landmark papers, read in chronological order. For each paper, the workflow is:

1. **Prepare** — Review prerequisites, historical context, and key vocabulary
2. **Read** — Read the original paper
3. **Discuss** — Clarify difficult passages and answer questions
4. **Summarize** — Write structured notes capturing key insights
5. **Implement** — Reproduce key algorithms or results in code (when applicable)

### Content Types

| Type | Format | Location |
|---|---|---|
| Paper reading notes | Markdown | `papers/<paper_name>/notes.md` |
| Code implementations | Jupyter Notebook | `papers/<paper_name>/implementation.ipynb` |
| Theory notes | Markdown | `notes/` |
| Practice & analysis | Jupyter Notebook | `notebooks/` |
| Utility scripts | Python / IDL | `scripts/` |
