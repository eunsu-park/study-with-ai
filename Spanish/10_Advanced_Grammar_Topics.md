## 1. 관계절

관계절은 명사를 수식하는 절로, 관계대명사나 관계부사로 시작합니다.

### A. 관계대명사 (Relative Pronouns)

#### 기본 관계대명사 표

| 관계대명사 | 용도 | 예시 | 의미 |
|-----------|------|------|------|
| que | 사람/사물 (가장 흔함) | El libro que leo | 내가 읽는 책 |
| quien(es) | 사람만 (전치사와 함께) | La persona con quien hablo | 내가 대화하는 사람 |
| el/la cual, los/las cuales | 격식체, 전치사 후 | El motivo por el cual vine | 내가 온 이유 |
| cuyo/a/os/as | 소유 (whose) | El autor cuyo libro leí | 내가 책을 읽은 작가 |
| lo que | 중성 (문장/개념) | Lo que dijiste es verdad | 네가 말한 것은 사실이다 |
| lo cual | 중성 격식 | Llegó tarde, lo cual me molestó | 늦게 왔는데, 그것이 나를 짜증나게 했다 |

#### 관계부사 (Relative Adverbs)

| 관계부사 | 용도 | 예시 | 의미 |
|----------|------|------|------|
| donde | 장소 | La ciudad donde vivo | 내가 사는 도시 |
| cuando | 시간 | El día cuando llegué | 내가 도착한 날 |
| como | 방식 | La forma como lo hizo | 그가 그것을 한 방식 |

### B. QUE - 가장 흔한 관계대명사

**사람과 사물 모두 사용 가능:**

```
주어 역할:
- El hombre que habla es mi padre
  (말하고 있는 남자는 내 아버지다)

- El libro que está en la mesa es mío
  (탁자 위에 있는 책은 내 것이다)

목적어 역할:
- El libro que compré es interesante
  (내가 산 책은 재미있다)

- La chica que conocí ayer es simpática
  (내가 어제 만난 소녀는 친절하다)
```

**전치사 + que (사물에만):**

| 전치사 | 예시 | 의미 |
|--------|------|------|
| en que | La casa en que vivo | 내가 사는 집 |
| con que | El bolígrafo con que escribo | 내가 쓰는 펜 |
| de que | El tema de que hablamos | 우리가 말한 주제 |
| a que | El lugar a que voy | 내가 가는 곳 |

**참고:** 사람에게는 전치사 + quien(es) 또는 el/la que 사용

### C. QUIEN(ES) - 사람만

**주로 전치사와 함께 사용:**

```
a quien(es) - ~에게
- La persona a quien escribí no respondió
  (내가 편지를 쓴 사람이 답하지 않았다)

con quien(es) - ~와 함께
- El amigo con quien viajé es mexicano
  (내가 함께 여행한 친구는 멕시코 사람이다)

de quien(es) - ~의, ~로부터
- El hombre de quien te hablé llegó
  (내가 너에게 말한 남자가 도착했다)

para quien(es) - ~를 위해
- La persona para quien trabajo es amable
  (내가 일하는 사람은 친절하다)
```

**주어로 사용 (격식체, 비제한적 관계절):**

```
Mi hermano, quien vive en Madrid, me visitó
(마드리드에 사는 내 형이 나를 방문했다)

Los estudiantes, quienes trabajaron duro, aprobaron
(열심히 공부한 학생들이 합격했다)
```

### D. EL/LA CUAL, LOS/LAS CUALES - 격식체

**복잡한 문장이나 격식적 글쓰기에서 사용:**

```
전치사 + el/la cual:
- La razón por la cual no vine fue el tráfico
  (내가 오지 않은 이유는 교통 때문이었다)

- El edificio detrás del cual estacioné es un banco
  (내가 뒤에 주차한 건물은 은행이다)

- Las circunstancias bajo las cuales ocurrió son desconocidas
  (그것이 발생한 상황은 알려지지 않았다)
```

**que와의 차이:**
- el cual은 더 격식적
- 긴 전치사구 후에 선호됨 (a pesar del cual, cerca del cual)
- 선행사가 불명확할 때 성/수로 명확화

### E. CUYO/A/OS/AS - 소유 관계

**소유 관계를 나타내며, 뒤따르는 명사와 성/수 일치:**

| 형태 | 예시 | 의미 |
|------|------|------|
| cuyo | El autor cuyo libro leí | 그의 책을 내가 읽은 작가 |
| cuya | La mujer cuya hija conozco | 그녀의 딸을 내가 아는 여자 |
| cuyos | El hombre cuyos hijos estudian aquí | 그의 아들들이 여기서 공부하는 남자 |
| cuyas | La ciudad cuyas calles son estrechas | 그 도로들이 좁은 도시 |

**주의:** cuyo는 뒤의 명사와 일치 (선행사와 아님)

```
El profesor cuya clase tomé es excelente
(내가 그의 수업을 들은 교수는 훌륭하다)
- cuya는 clase(여성)와 일치

El país cuyos habitantes son amables
(그 주민들이 친절한 나라)
- cuyos는 habitantes(남성 복수)와 일치
```

### F. 관계부사: DONDE, CUANDO, COMO

**DONDE (장소):**
```
La ciudad donde nací es pequeña
(내가 태어난 도시는 작다)

El restaurante donde comimos era caro
(우리가 먹은 식당은 비쌌다)

= en que / en el que
La ciudad en que nací = La ciudad donde nací
```

**CUANDO (시간):**
```
El día cuando llegué llovía
(내가 도착한 날 비가 왔다)

La época cuando vivíamos felices
(우리가 행복하게 살던 시절)

= en que
El día en que llegué = El día cuando llegué
```

**COMO (방식):**
```
La forma como lo hizo me sorprendió
(그가 그것을 한 방식이 나를 놀라게 했다)

No me gusta el modo como habla
(나는 그가 말하는 방식이 싫다)

= en que
La forma en que lo hizo = La forma como lo hizo
```

### G. LO QUE / LO CUAL - 중성 관계대명사

**LO QUE (절이나 개념을 받음):**
```
Lo que dijiste es interesante
(네가 말한 것은 재미있다)

No entiendo lo que quieres
(나는 네가 원하는 것을 이해하지 못한다)

Haz lo que quieras
(네가 원하는 것을 해라)
```

**LO CUAL (앞 문장 전체를 받음):**
```
Llegó tarde, lo cual me molestó
(그는 늦게 왔고, 그것이 나를 짜증나게 했다)

No llamó, lo cual me preocupó
(그는 전화하지 않았고, 그것이 나를 걱정하게 했다)
```

### H. 제한적 vs 비제한적 관계절

**제한적 관계절 (Restrictive):**
- 선행사를 특정하는 필수 정보
- 쉼표 없음

```
Los estudiantes que estudian mucho aprueban
(많이 공부하는 학생들은 합격한다)
→ 모든 학생이 아닌, 많이 공부하는 학생들만
```

**비제한적 관계절 (Non-restrictive):**
- 부가 정보 제공
- 쉼표로 분리

```
Los estudiantes, que estudian mucho, aprueban
(학생들은, 많이 공부하는데, 합격한다)
→ 모든 학생이 많이 공부하고 합격함
```

## 2. 수동태

스페인어에서 수동태는 여러 방식으로 표현됩니다.

### A. SER + 과거분사 (Voz Pasiva con SER)

**형성:**
```
ser (활용) + 과거분사 (주어와 성/수 일치)
```

**능동문 → 수동문 변환:**

| 능동문 | 수동문 |
|--------|--------|
| Juan escribió el libro | El libro fue escrito por Juan |
| (후안이 책을 썼다) | (그 책은 후안에 의해 쓰여졌다) |

**시제별 예시:**

| 시제 | 예시 | 의미 |
|------|------|------|
| 현재 | El libro es leído por muchos | 그 책은 많은 사람들에게 읽힌다 |
| 과거 (불완료) | El libro era leído por muchos | 그 책은 많은 사람들에게 읽히곤 했다 |
| 과거 (완료) | El libro fue leído por muchos | 그 책은 많은 사람들에게 읽혔다 |
| 미래 | El libro será leído por muchos | 그 책은 많은 사람들에게 읽힐 것이다 |
| 현재완료 | El libro ha sido leído por muchos | 그 책은 많은 사람들에게 읽혀왔다 |
| 조건 | El libro sería leído por muchos | 그 책은 많은 사람들에게 읽힐 것이다 |

**행위자 표현: por**
```
La carta fue escrita por María
(그 편지는 마리아에 의해 쓰여졌다)

El edificio fue diseñado por un arquitecto famoso
(그 건물은 유명한 건축가에 의해 설계되었다)
```

**과거분사의 성/수 일치:**
```
El libro fue publicado (남성 단수)
La novela fue publicada (여성 단수)
Los libros fueron publicados (남성 복수)
Las novelas fueron publicadas (여성 복수)
```

### B. ESTAR + 과거분사 (결과 상태)

**SER vs ESTAR 차이:**

| SER + 과거분사 | ESTAR + 과거분사 |
|---------------|-----------------|
| 행위/동작 | 상태/결과 |
| La puerta fue cerrada (por alguien) | La puerta está cerrada |
| 문이 (누군가에 의해) 닫혔다 | 문이 닫혀 있다 |

**예시:**

```
SER (동작):
La tienda fue abierta a las 9
(가게가 9시에 열렸다 - 여는 행위)

ESTAR (상태):
La tienda está abierta ahora
(가게가 지금 열려 있다 - 현재 상태)
```

**일반적인 ESTAR + 과거분사:**

| 표현 | 의미 |
|------|------|
| estar abierto/a | 열려 있다 |
| estar cerrado/a | 닫혀 있다 |
| estar roto/a | 부서져 있다 |
| estar terminado/a | 끝나 있다 |
| estar preparado/a | 준비되어 있다 |
| estar escrito/a | 쓰여 있다 |
| estar hecho/a | 만들어져 있다 |
| estar situado/a | 위치해 있다 |
| estar ocupado/a | 사용 중이다 |
| estar sentado/a | 앉아 있다 |

### C. SE 수동태 (Pasiva Refleja)

**가장 흔히 사용되는 수동 표현!**

**형성:**
```
se + 동사 (3인칭 단수/복수)
```

**예시:**

| SE 수동태 | 의미 |
|----------|------|
| Se habla español | 스페인어가 사용된다 |
| Se vende esta casa | 이 집이 팔린다 |
| Se venden casas | 집들이 팔린다 |
| Se necesitan empleados | 직원이 필요하다 |
| Se construyó un puente | 다리가 건설되었다 |
| Se publicó el libro | 책이 출판되었다 |

**주어와의 일치:**
- 단수 주어 → se + 동사 단수
- 복수 주어 → se + 동사 복수

```
Se vende un coche (단수)
(차 한 대가 팔린다)

Se venden coches (복수)
(차들이 팔린다)
```

**SER 수동태 vs SE 수동태:**

| SER 수동태 | SE 수동태 |
|-----------|----------|
| 행위자 명시 가능 | 행위자 명시 불가 |
| 격식적 | 일상적 |
| 덜 흔함 | 매우 흔함 |
| El libro fue escrito por Juan | Se escribió el libro |

### D. SE 비인칭 (Se Impersonal)

**불특정 주체를 나타냄:**

```
se + 동사 3인칭 단수
```

**SE 수동태 vs SE 비인칭:**

| SE 수동태 | SE 비인칭 |
|----------|----------|
| 사물 주어 | 사람 관련/동사 자체 |
| Se venden casas (집들이 팔린다) | Se vive bien aquí (여기서 잘 산다) |
| 동사가 주어와 일치 | 항상 단수 |

**SE 비인칭 예시:**

| 표현 | 의미 |
|------|------|
| Se dice que... | ~라고 말한다 |
| Se cree que... | ~라고 믿는다 |
| Se puede... | ~할 수 있다 |
| Se necesita... | ~가 필요하다 |
| Se trabaja mucho aquí | 여기서 많이 일한다 |
| Se vive bien en España | 스페인에서 잘 산다 |
| Se come bien aquí | 여기서 음식이 맛있다 |

**사람 직접목적어와 함께 (a 필요):**
```
Se ayuda a los pobres (가난한 사람들을 돕는다)
Se busca a los criminales (범인들을 찾는다)
Se entrevistó a los candidatos (후보자들을 면접했다)
```

### E. 수동태 요약

| 유형 | 형식 | 사용 |
|------|------|------|
| SER 수동태 | ser + 과거분사 + por | 행위자 명시, 격식 |
| ESTAR 결과 | estar + 과거분사 | 상태/결과 |
| SE 수동태 | se + 동사 (일치) | 일반적 수동, 사물 |
| SE 비인칭 | se + 단수 동사 | 불특정 주체 |

## 3. 고급 조건문

### A. 조건문 유형 복습

| 유형 | 조건절 | 주절 | 용도 |
|------|--------|------|------|
| Type 0 | Si + 현재 | 현재 | 일반적 진리 |
| Type 1 | Si + 현재 | 미래/현재/명령 | 실현 가능 |
| Type 2 | Si + 과거 접속법 | 조건 | 가정법 현재 |
| Type 3 | Si + 과거완료 접속법 | 조건완료 | 가정법 과거 |

**예시:**

```
Type 0 (일반적 진리):
Si calientas agua a 100°C, hierve
(물을 100도로 가열하면 끓는다)

Type 1 (실현 가능):
Si estudias, aprobarás
(공부하면 합격할 거야)

Type 2 (현재 반사실):
Si tuviera dinero, viajaría
(돈이 있다면 여행할 텐데)

Type 3 (과거 반사실):
Si hubiera estudiado, habría aprobado
(공부했었다면 합격했을 텐데)
```

### B. 혼합 조건문 (Mixed Conditionals)

**과거 조건 → 현재 결과:**
```
Si + 과거완료 접속법, 조건 단순

Si hubiera estudiado medicina, sería médico ahora
(의학을 공부했었다면 지금 의사일 텐데)

Si no hubiera comido tanto, no estaría enfermo
(그렇게 많이 먹지 않았다면 지금 아프지 않을 텐데)
```

**현재 조건 → 과거 결과:**
```
Si + 과거 접속법, 조건완료

Si fuera más inteligente, habría entendido
(더 똑똑했다면 이해했을 텐데)

Si no fuera tan tímido, le habría hablado
(그렇게 수줍지 않았다면 그녀에게 말했을 텐데)
```

### C. 대안 조건 구조 (Alternative Conditional Structures)

#### DE + 부정사 (격식, 문어체)

**si절을 대체:**

| Si절 | De + 부정사 |
|------|------------|
| Si hubiera sabido | De haberlo sabido |
| Si pudiera | De poder |
| Si tuviera tiempo | De tener tiempo |

```
De haberlo sabido, no habría venido
(알았다면 오지 않았을 텐데)
= Si lo hubiera sabido, no habría venido

De ser posible, lo haría
(가능하다면 할 텐데)
= Si fuera posible, lo haría
```

#### COMO + 접속법 (경고/위협)

```
Como no vengas, me enfadaré
(오지 않으면 화낼 거야)

Como llegues tarde otra vez, no te espero
(또 늦으면 안 기다릴 거야)
```

#### A NO SER QUE + 접속법 (unless)

```
Iré a no ser que llueva
(비가 오지 않는 한 갈 거야)

No lo haré a no ser que me lo pidan
(요청하지 않는 한 안 할 거야)
```

#### A MENOS QUE + 접속법 (unless)

```
Vendrá a menos que esté enfermo
(아프지 않는 한 올 거야)

No aprobarás a menos que estudies
(공부하지 않는 한 합격 못 해)
```

#### CON TAL DE QUE + 접속법 (provided that)

```
Te ayudo con tal de que me pagues
(돈만 주면 도와줄게)

Lo haré con tal de que no me molestes
(방해만 안 하면 할게)
```

#### SIEMPRE QUE + 접속법 (as long as)

```
Puedes salir siempre que vuelvas pronto
(빨리 돌아오기만 하면 나가도 돼)

Te prestaré el libro siempre que lo cuides
(잘 관리하기만 하면 책 빌려줄게)
```

#### SIEMPRE Y CUANDO + 접속법 (as long as)

```
Te ayudaré siempre y cuando me escuches
(내 말을 듣는 한 도와줄게)
```

#### EN CASO DE QUE + 접속법 (in case)

```
Llámame en caso de que necesites ayuda
(도움이 필요하면 전화해)

Lleva paraguas en caso de que llueva
(비가 올 경우를 대비해 우산 가져가)
```

#### SALVO QUE + 접속법 (unless)

```
Iré salvo que haya problema
(문제가 없는 한 갈 거야)
```

### D. 조건 완료형의 다른 용법

**과거의 추측:**
```
Serían las tres cuando llegó
(그가 도착했을 때 3시쯤 됐을 거야)

Tendría unos 30 años
(30살 정도 됐을 거야)
```

**간접화법에서의 미래:**
```
Dijo que vendría (직접: "Vendré")
(올 거라고 말했다)

Prometió que lo haría (직접: "Lo haré")
(할 거라고 약속했다)
```

**정중한 요청:**
```
¿Podría ayudarme?
(도와주시겠어요?)

¿Sería posible cambiar la fecha?
(날짜를 바꿀 수 있을까요?)
```

### E. 조건문 시제 일치 표

| 조건절 | 주절 | 예시 |
|--------|------|------|
| Si + 현재 직설법 | 현재/미래/명령 | Si llueve, no salgo |
| Si + 과거 불완료 직설법 | 과거 불완료 | Si llovía, no salía |
| Si + 과거 접속법 | 조건 단순 | Si lloviera, no saldría |
| Si + 과거완료 접속법 | 조건 완료 | Si hubiera llovido, no habría salido |

### F. 조건문 요약표

| 접속사/구조 | 의미 | 법 |
|------------|------|---|
| si | if | 직설법/접속법 |
| como | if (warning) | 접속법 |
| a no ser que | unless | 접속법 |
| a menos que | unless | 접속법 |
| con tal de que | provided that | 접속법 |
| siempre que | as long as | 접속법 |
| en caso de que | in case | 접속법 |
| salvo que | unless | 접속법 |
| de + 부정사 | if (formal) | - |

### A. 관계절 연습

**빈칸에 적절한 관계대명사를 넣으세요:**

1. El libro _______ compré es interesante. (que)

3. El autor _______ novela leí ganó un premio. (cuya)
4. No entiendo lo _______ dices. (que)

### B. 수동태 연습

**능동문을 수동문으로 바꾸세요:**

1. Juan escribió la carta. → La carta fue escrita por Juan.
2. Millones leen este libro. → Este libro es leído por millones.
3. Construyeron el puente. → Se construyó el puente.

### C. 조건문 연습

**적절한 동사 형태로 빈칸을 채우세요:**

1. Si _______ (tener, yo) dinero, viajaría. → tuviera
2. Si hubiera estudiado, _______ (aprobar). → habría aprobado
3. Te ayudo con tal de que me _______ (pagar, tú). → pagues

## 학습 팁 (Study Tips)

1. **관계대명사**: que가 가장 흔함, 전치사와 함께 쓸 때 주의
2. **SE 수동태**: 실제 대화에서 SER 수동태보다 훨씬 흔함
3. **조건문**: 접속법을 사용하는 조건 접속사들 암기
