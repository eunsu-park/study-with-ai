## 1. 대명사

### A. 주격 대명사 (Subject Pronouns)

| 대명사 | 의미 |
|--------|------|
| yo / I / 나 |
| tú / you (informal) / 너 |
| él / he / 그 |
| ella / she / 그녀 |
| usted / you (formal) / 당신 |
| nosotros/as / we / 우리 |
| vosotros/as / you all (informal) / 너희 |
| ellos / they (masculine) / 그들 |
| ellas / they (feminine) / 그녀들 |
| ustedes / you all (formal) / 당신들 |

### B. 재귀 대명사 (Reflexive Pronouns)

| 주어 | 재귀 대명사 | 의미 |
|------|------------|------|
| yo | me | myself / 나 자신을 |
| tú | te | yourself / 너 자신을 |
| él/ella/usted | se | himself/herself/yourself / 그/그녀/당신 자신을 |
| nosotros/as | nos | ourselves / 우리 자신을 |
| vosotros/as | os | yourselves / 너희 자신을 |
| ellos/ellas/ustedes | se | themselves/yourselves / 그들/당신들 자신을 |

**예문:**
- Me levanto / I get up / 나는 일어난다
- Te lavas / You wash yourself / 너는 씻는다
- Se viste / He/She gets dressed / 그/그녀는 옷을 입는다
- Nos preparamos / We prepare ourselves / 우리는 준비한다

### C. 직접 목적어 대명사 (Direct Object Pronouns)

| 목적어 | 대명사 | 의미 |
|--------|--------|------|
| a mí | me | me / 나를 |
| a ti | te | you / 너를 |
| a él | lo | him, it (masc) / 그를, 그것을 |
| a ella | la | her, it (fem) / 그녀를, 그것을 |
| a usted | lo/la | you (formal) / 당신을 |
| a nosotros/as | nos | us / 우리를 |
| a vosotros/as | os | you all / 너희를 |
| a ellos | los | them (masc) / 그들을 |
| a ellas | las | them (fem) / 그녀들을 |
| a ustedes | los/las | you all / 당신들을 |

**예문:**
- Lo veo / I see it/him / 나는 그것/그를 본다
- Te quiero / I love you / 나는 너를 사랑한다
- La conozco / I know her / 나는 그녀를 안다
- Los compro / I buy them / 나는 그것들을 산다

### D. 간접 목적어 대명사 (Indirect Object Pronouns)

| 목적어 | 대명사 | 의미 |
|--------|--------|------|
| a mí | me | to me / 나에게 |
| a ti | te | to you / 너에게 |
| a él/ella/usted | le | to him/her/you / 그/그녀/당신에게 |
| a nosotros/as | nos | to us / 우리에게 |
| a vosotros/as | os | to you all / 너희에게 |
| a ellos/ellas/ustedes | les | to them/you all / 그들/당신들에게 |

**예문:**
- Me da el libro / He gives the book to me / 그는 나에게 책을 준다
- Te escribo / I write to you / 나는 너에게 쓴다
- Le digo / I tell him/her / 나는 그/그녀에게 말한다
- Nos compran regalos / They buy gifts for us / 그들은 우리에게 선물을 산다

**Le/Les → Se 규칙:**
직접 목적어와 함께 쓸 때 le/les는 se로 변경
- Se lo doy / I give it to him/her / 나는 그/그녀에게 그것을 준다
- Se la muestro / I show it to him/her / 나는 그/그녀에게 그것을 보여준다

### E. 소유 대명사 (Possessive Pronouns)

| 소유자 | 남성 단수 | 여성 단수 | 남성 복수 | 여성 복수 |
|--------|----------|----------|----------|----------|
| yo | el mío | la mía | los míos | las mías |
| tú | el tuyo | la tuya | los tuyos | las tuyas |
| él/ella/usted | el suyo | la suya | los suyos | las suyas |
| nosotros/as | el nuestro | la nuestra | los nuestros | las nuestras |
| vosotros/as | el vuestro | la vuestra | los vuestros | las vuestras |
| ellos/ellas/ustedes | el suyo | la suya | los suyos | las suyas |

**예문:**
- Este libro es el mío / This book is mine / 이 책은 내 것이다
- Esa casa es la tuya / That house is yours / 저 집은 네 것이다

### F. 지시 대명사 (Demonstrative Pronouns)

| 거리 | 남성 단수 | 여성 단수 | 중성 | 남성 복수 | 여성 복수 |
|------|----------|----------|------|----------|----------|
| 가까이 | este / this | esta / this | esto / this | estos / these | estas / these |
| 중간 | ese / that | esa / that | eso / that | esos / those | esas / those |
| 멀리 | aquel / that | aquella / that | aquello / that | aquellos / those | aquellas / those |

**중성 지시대명사:**
- esto / this (thing) / 이것
- eso / that (thing) / 그것
- aquello / that (thing) over there / 저것

**예문:**
- ¿Qué es esto? / What is this? / 이것은 무엇입니까?
- Eso es verdad / That's true / 그것은 사실이다

## 2. 접속사

### A. 등위 접속사 (Coordinating Conjunctions)

1. **y** (e) / and / 그리고
   - Juan y María / Juan and María / 후안과 마리아
   - *참고: i/hi로 시작하는 단어 앞에서는 e*
   - padre e hijo / father and son / 아버지와 아들

2. **o** (u) / or / 또는
   - café o té / coffee or tea / 커피 또는 차
   - *참고: o/ho로 시작하는 단어 앞에서는 u*
   - siete u ocho / seven or eight / 7 또는 8

3. **pero** / but / 하지만
   - Es caro pero bueno / It's expensive but good / 비싸지만 좋다

4. **sino** / but rather / 그게 아니라
   - No es rojo sino azul / It's not red but blue / 빨간게 아니라 파란색이다

5. **ni** / neither, nor / ~도 아니고
   - No tengo tiempo ni dinero / I have neither time nor money / 시간도 돈도 없다

### B. 원인/이유 접속사 (Cause/Reason)

6. **porque** / because / 왜냐하면
   - Estudio porque quiero aprender / I study because I want to learn / 배우고 싶어서 공부한다

7. **como** / since, as / ~이므로
   - Como llueve, me quedo / Since it's raining, I stay / 비가 와서 있는다

8. **pues** / well, since / ~하니까
   - No puedo ir, pues estoy ocupado / I can't go, since I'm busy / 바빠서 못 간다

9. **ya que** / since / ~이므로
   - Ya que estás aquí, ayúdame / Since you're here, help me / 여기 있으니까 도와줘

10. **puesto que** / since / ~이므로
    - Puesto que no hay tiempo, empecemos / Since there's no time, let's start / 시간이 없으니 시작하자

### C. 결과 접속사 (Consequence)

11. **así que** / so, therefore / 그래서
    - Estoy cansado, así que voy a dormir / I'm tired, so I'll sleep / 피곤해서 잘 거야

12. **por lo tanto** / therefore / 따라서
    - No hay evidencia, por lo tanto es inocente / There's no evidence, therefore he's innocent / 증거가 없으므로 무죄다

13. **por eso** / that's why / 그래서
    - Llueve, por eso llevo paraguas / It's raining, that's why I carry an umbrella / 비가 와서 우산을 가져간다

14. **entonces** / then, so / 그러면
    - ¿No vienes? Entonces voy solo / You're not coming? Then I'll go alone / 안 와? 그럼 혼자 갈게

### D. 목적 접속사 (Purpose)

15. **para que** / so that / ~하도록 (+ 접속법)
    - Te lo digo para que lo sepas / I tell you so you know / 네가 알도록 말해준다

16. **a fin de que** / in order that / ~하기 위해 (+ 접속법)
    - Trabajo a fin de que mi familia viva bien / I work so my family lives well / 가족이 잘 살도록 일한다

### E. 조건 접속사 (Conditional)

17. **si** / if / 만약
    - Si llueve, no voy / If it rains, I won't go / 비가 오면 안 간다

18. **a menos que** / unless / ~하지 않는 한 (+ 접속법)
    - Voy a menos que llueva / I'll go unless it rains / 비가 오지 않는 한 갈게

19. **con tal de que** / provided that / ~하기만 하면 (+ 접속법)
    - Te ayudo con tal de que me pagues / I'll help you if you pay me / 돈만 주면 도와줄게

20. **siempre que** / as long as / ~하는 한
    - Siempre que estudies, aprobarás / As long as you study, you'll pass / 공부하기만 하면 합격할 거야

### F. 양보 접속사 (Concessive)

21. **aunque** / although / 비록 ~이지만
    - Aunque llueve, voy / Although it's raining, I'm going / 비가 오지만 간다

22. **a pesar de que** / despite / ~에도 불구하고
    - A pesar de que es difícil, lo intento / Despite it being difficult, I try / 어렵지만 노력한다

23. **sin embargo** / however / 그러나
    - Es difícil, sin embargo lo haré / It's difficult, however I'll do it / 어렵지만 할 거야

24. **no obstante** / nevertheless / 그럼에도 불구하고
    - Estudió mucho, no obstante reprobó / He studied a lot, nevertheless he failed / 많이 공부했지만 낙제했다

### G. 시간 접속사 (Temporal)

25. **cuando** / when / ~할 때
    - Cuando llegues, llámame / When you arrive, call me / 도착하면 전화해

26. **mientras** / while / ~하는 동안
    - Mientras estudias, yo cocino / While you study, I'll cook / 네가 공부하는 동안 내가 요리할게

27. **antes de que** / before / ~하기 전에 (+ 접속법)
    - Llámame antes de que salgas / Call me before you leave / 나가기 전에 전화해

28. **después de que** / after / ~한 후에
    - Llámame después de que llegues / Call me after you arrive / 도착한 후에 전화해

29. **hasta que** / until / ~할 때까지
    - Espero hasta que vengas / I'll wait until you come / 네가 올 때까지 기다릴게

30. **desde que** / since (time) / ~한 이후로
    - Desde que llegué, he estado ocupado / Since I arrived, I've been busy / 도착한 이후로 바빴다

31. **en cuanto** / as soon as / ~하자마자
    - En cuanto llegue, te llamo / As soon as I arrive, I'll call you / 도착하자마자 전화할게

32. **tan pronto como** / as soon as / ~하자마자
    - Tan pronto como termine, salgo / As soon as I finish, I'm leaving / 끝나자마자 나간다

### H. 비교 접속사 (Comparative)

33. **como** / as, like / ~처럼
    - Es como su padre / He's like his father / 그는 아버지 같다

34. **tan... como** / as... as / ~만큼 ~하다
    - Es tan alto como su hermano / He's as tall as his brother / 그는 형만큼 크다

35. **más... que** / more... than / ~보다 더
    - Es más alto que yo / He's taller than me / 그는 나보다 크다

36. **menos... que** / less... than / ~보다 덜
    - Es menos caro que ese / It's less expensive than that one / 저것보다 덜 비싸다

### I. 설명/추가 접속사 (Explanatory/Additive)

37. **es decir** / that is to say / 즉
    - Viene mañana, es decir, el lunes / He's coming tomorrow, that is, Monday / 내일, 즉 월요일에 온다

38. **además** / besides, furthermore / 게다가
    - Es caro. Además, no me gusta / It's expensive. Besides, I don't like it / 비싸다. 게다가 마음에 안 든다

39. **también** / also / 또한
    - Hablo español y también inglés / I speak Spanish and also English / 스페인어와 영어도 말한다

40. **tampoco** / neither, either / ~도 아니다
    - No como carne y tampoco pescado / I don't eat meat and I don't eat fish either / 고기도 생선도 안 먹는다

## 3. 전치사

### A. 기본 전치사

1. **a** / to, at / ~에
   - Voy a la escuela / I go to school / 나는 학교에 간다
   - a las tres / at three / 3시에

2. **de** / of, from / ~의, ~로부터
   - el libro de Juan / Juan's book / 후안의 책
   - Soy de Corea / I'm from Korea / 나는 한국 출신이다

3. **en** / in, on, at / ~에, ~에서
   - Estoy en casa / I'm at home / 나는 집에 있다
   - en la mesa / on the table / 탁자 위에
   - en enero / in January / 1월에

4. **con** / with / ~와 함께
   - Voy con mi amigo / I go with my friend / 나는 친구와 간다

5. **sin** / without / ~ 없이
   - café sin azúcar / coffee without sugar / 설탕 없는 커피

6. **para** / for, in order to / ~를 위해, ~하기 위해
   - Es para ti / It's for you / 너를 위한 거야
   - Estudio para aprender / I study to learn / 배우기 위해 공부한다

7. **por** / for, by, through / ~때문에, ~에 의해, ~를 통해
   - por la mañana / in the morning / 아침에
   - por favor / please / 부탁합니다
   - Gracias por todo / Thanks for everything / 모든 것에 감사합니다

8. **sobre** / on, about / ~위에, ~에 관하여
   - el libro está sobre la mesa / the book is on the table / 책이 탁자 위에 있다
   - hablamos sobre política / we talk about politics / 우리는 정치에 대해 말한다

9. **bajo** / under / ~아래
   - bajo la mesa / under the table / 탁자 아래

10. **entre** / between, among / ~사이에
    - entre tú y yo / between you and me / 너와 나 사이에

11. **hacia** / toward / ~를 향해
    - Voy hacia el norte / I go toward the north / 나는 북쪽으로 간다

12. **hasta** / until, up to / ~까지
    - hasta mañana / until tomorrow / 내일까지
    - hasta las 5 / up to 5 / 5시까지

13. **desde** / from, since / ~로부터, ~이래로
    - desde aquí / from here / 여기서부터
    - desde ayer / since yesterday / 어제부터

14. **durante** / during / ~동안
    - durante el verano / during summer / 여름 동안

15. **tras** / after, behind / ~후에, ~뒤에
    - tras la puerta / behind the door / 문 뒤에

16. **contra** / against / ~에 반대하여
    - contra la pared / against the wall / 벽에 기대어

17. **según** / according to / ~에 따르면
    - según él / according to him / 그에 따르면

### B. 전치사구 (Prepositional Phrases)

18. **delante de** / in front of / ~앞에
    - delante de la casa / in front of the house / 집 앞에

19. **detrás de** / behind / ~뒤에
    - detrás de ti / behind you / 너 뒤에

20. **encima de** / on top of / ~위에
    - encima de la mesa / on top of the table / 탁자 위에

21. **debajo de** / under / ~아래
    - debajo de la cama / under the bed / 침대 아래

22. **al lado de** / next to / ~옆에
    - al lado de la escuela / next to the school / 학교 옆에

23. **cerca de** / near / ~근처에
    - cerca de aquí / near here / 여기 근처에

24. **lejos de** / far from / ~에서 멀리
    - lejos de la ciudad / far from the city / 도시에서 멀리

25. **dentro de** / inside / ~안에
    - dentro de la caja / inside the box / 상자 안에

26. **fuera de** / outside / ~밖에
    - fuera de la casa / outside the house / 집 밖에

27. **a través de** / through / ~를 통해
    - a través de la ventana / through the window / 창문을 통해

28. **alrededor de** / around / ~주위에
    - alrededor del parque / around the park / 공원 주위에

29. **antes de** / before / ~전에
    - antes de comer / before eating / 먹기 전에

30. **después de** / after / ~후에
    - después de la clase / after class / 수업 후에

## 5. 의문사

1. **¿Qué?** / What? / 무엇?
   - ¿Qué es esto? / What is this? / 이것은 무엇입니까?

2. **¿Quién?** / Who? / 누구?
   - ¿Quién eres? / Who are you? / 당신은 누구입니까?

3. **¿Cuándo?** / When? / 언제?
   - ¿Cuándo vienes? / When are you coming? / 언제 오니?

4. **¿Dónde?** / Where? / 어디?
   - ¿Dónde está? / Where is it? / 어디에 있어?

5. **¿Por qué?** / Why? / 왜?

## 학습 팁 (Study Tips)

1. **대명사 순서**: 재귀 → 간접 → 직접 순서 암기
2. **접속사 카테고리별**: 시간, 원인, 조건 등 용도별로 학습
3. **전치사 활용**: 자주 쓰는 표현을 통째로 암기
4. **부사 파생**: 형용사 + mente 규칙 익히기
5. **의문사 연습**: 매일 질문 만들기
6. **수사 실습**: 전화번호, 가격 등을 스페인어로 말하기
7. **문장 속 학습**: 단독보다 문장에서 자연스럽게 익히기
