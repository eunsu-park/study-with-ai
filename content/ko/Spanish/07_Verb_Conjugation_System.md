## 1. 동사 체계 개요

### 1.1 동사 분류

```
┌─────────────────────────────────────────────────────────────────┐
│                    스페인어 동사 분류                            │
│                                                                 │
│   -AR 동사 (1군)      -ER 동사 (2군)      -IR 동사 (3군)        │
│   hablar (말하다)     comer (먹다)        vivir (살다)          │
│   trabajar (일하다)   beber (마시다)      escribir (쓰다)       │
│   estudiar (공부하다) leer (읽다)         partir (떠나다)       │
│                                                                 │
│   가장 많음           두 번째              세 번째               │
│   (약 75%)           (약 15%)             (약 10%)              │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 시제 체계 요약

```
직설법 (Indicativo):
┌────────────────────┬──────────────────────────────────────────────┐
│ 시제               │ 예시 (hablar)                                 │
├────────────────────┼──────────────────────────────────────────────┤
│ 현재               │ hablo, hablas, habla, hablamos, habláis, hablan │
│ 현재 진행          │ estoy hablando                                │
│ 단순 과거          │ hablé, hablaste, habló, hablamos, hablasteis, hablaron │
│ 불완료 과거        │ hablaba, hablabas, hablaba, hablábamos...     │
│ 미래               │ hablaré, hablarás, hablará, hablaremos...     │
│ 조건법             │ hablaría, hablarías, hablaría, hablaríamos... │
│ 현재 완료          │ he hablado, has hablado, ha hablado...        │
│ 과거 완료          │ había hablado, habías hablado...              │
└────────────────────┴──────────────────────────────────────────────┘

접속법 (Subjuntivo):
┌────────────────────┬──────────────────────────────────────────────┐
│ 시제               │ 예시 (hablar)                                 │
├────────────────────┼──────────────────────────────────────────────┤
│ 현재               │ hable, hables, hable, hablemos, habléis, hablen │
│ 과거               │ hablara/hablase, hablaras, hablara...         │
│ 현재 완료          │ haya hablado, hayas hablado...                │
│ 과거 완료          │ hubiera/hubiese hablado...                    │
└────────────────────┴──────────────────────────────────────────────┘

명령법 (Imperativo):
┌────────────────────┬──────────────────────────────────────────────┐
│ 형태               │ 예시 (hablar)                                 │
├────────────────────┼──────────────────────────────────────────────┤
│ 긍정               │ habla (tú), hable (usted), hablad (vosotros) │
│ 부정               │ no hables, no hable, no habléis              │
└────────────────────┴──────────────────────────────────────────────┘
```

### 1.3 규칙 동사 활용표

```
현재 시제 규칙 활용:
┌────────┬──────────┬──────────┬──────────┐
│ 인칭   │ -AR      │ -ER      │ -IR      │
│        │ (hablar) │ (comer)  │ (vivir)  │
├────────┼──────────┼──────────┼──────────┤
│ yo     │ habl-o   │ com-o    │ viv-o    │
│ tú     │ habl-as  │ com-es   │ viv-es   │
│ él     │ habl-a   │ com-e    │ viv-e    │
│ nosotros│ habl-amos│ com-emos │ viv-imos │
│ vosotros│ habl-áis│ com-éis  │ viv-ís   │
│ ellos  │ habl-an  │ com-en   │ viv-en   │
└────────┴──────────┴──────────┴──────────┘

단순 과거 규칙 활용:
┌────────┬──────────┬──────────┬──────────┐
│ 인칭   │ -AR      │ -ER      │ -IR      │
├────────┼──────────┼──────────┼──────────┤
│ yo     │ habl-é   │ com-í    │ viv-í    │
│ tú     │ habl-aste│ com-iste │ viv-iste │
│ él     │ habl-ó   │ com-ió   │ viv-ió   │
│ nosotros│ habl-amos│ com-imos │ viv-imos │
│ vosotros│ habl-asteis│com-isteis│viv-isteis│
│ ellos  │ habl-aron│ com-ieron│ viv-ieron│
└────────┴──────────┴──────────┴──────────┘
```

## 2. 어간 변화 동사

### 2.1 e → ie 변화

```
┌─────────────────────────────────────────────────────────────────┐
│                    e → ie 어간 변화                              │
│                                                                 │
│   pensar (생각하다)  querer (원하다)   sentir (느끼다)          │
│   pienso             quiero            siento                   │
│   piensas            quieres           sientes                  │
│   piensa             quiere            siente                   │
│   pensamos           queremos          sentimos   ← 변화 없음   │
│   pensáis            queréis           sentís     ← 변화 없음   │
│   piensan            quieren           sienten                  │
│                                                                 │
│   ※ nosotros, vosotros는 변화하지 않음 (boot verb)             │
│                                                                 │
│   기타 동사:                                                    │
│   cerrar (닫다), empezar (시작하다), entender (이해하다),      │
│   perder (잃다), preferir (선호하다), mentir (거짓말하다)      │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 o → ue 변화

```
┌─────────────────────────────────────────────────────────────────┐
│                    o → ue 어간 변화                              │
│                                                                 │
│   poder (할 수 있다)  volver (돌아오다)  dormir (자다)          │
│   puedo              vuelvo            duermo                   │
│   puedes             vuelves           duermes                  │
│   puede              vuelve            duerme                   │
│   podemos            volvemos          dormimos  ← 변화 없음    │
│   podéis             volvéis           dormís    ← 변화 없음    │
│   pueden             vuelven           duermen                  │
│                                                                 │
│   기타 동사:                                                    │
│   encontrar (찾다), contar (세다), costar (비용이 들다),       │
│   mostrar (보여주다), recordar (기억하다), morir (죽다)        │
│                                                                 │
│   특수: jugar (놀다) → juego, juegas... (u → ue)               │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 e → i 변화

```
┌─────────────────────────────────────────────────────────────────┐
│                    e → i 어간 변화 (-IR 동사만)                  │
│                                                                 │
│   pedir (요청하다)   servir (제공하다)  repetir (반복하다)      │
│   pido               sirvo             repito                   │
│   pides              sirves            repites                  │
│   pide               sirve             repite                   │
│   pedimos            servimos          repetimos ← 변화 없음    │
│   pedís              servís            repetís   ← 변화 없음    │
│   piden              sirven            repiten                  │
│                                                                 │
│   ※ 단순 과거에서 3인칭도 변화함:                               │
│   pedir → pidió, pidieron                                       │
│   servir → sirvió, sirvieron                                   │
│                                                                 │
│   기타 동사:                                                    │
│   seguir (따르다), conseguir (얻다), vestir (입다)             │
└─────────────────────────────────────────────────────────────────┘
```

### 2.4 어간 변화 요약 도표

```
┌──────────┬──────────┬──────────┬──────────┬──────────────────────┐
│ 변화     │ yo       │ tú       │ nos.     │ 대표 동사            │
├──────────┼──────────┼──────────┼──────────┼──────────────────────┤
│ e → ie   │ pienso   │ piensas  │ pensamos │ pensar, querer,     │
│          │          │          │          │ entender, sentir    │
├──────────┼──────────┼──────────┼──────────┼──────────────────────┤
│ o → ue   │ puedo    │ puedes   │ podemos  │ poder, volver,      │
│          │          │          │          │ dormir, encontrar   │
├──────────┼──────────┼──────────┼──────────┼──────────────────────┤
│ e → i    │ pido     │ pides    │ pedimos  │ pedir, servir,      │
│ (-IR만)  │          │          │          │ seguir, repetir     │
├──────────┼──────────┼──────────┼──────────┼──────────────────────┤
│ u → ue   │ juego    │ juegas   │ jugamos  │ jugar               │
└──────────┴──────────┴──────────┴──────────┴──────────────────────┘
```

## 3. 불규칙 동사 패턴

### 3.1 1인칭 단수 불규칙 (-go 형)

```
┌─────────────────────────────────────────────────────────────────┐
│                    yo 형태 -go 불규칙                            │
│                                                                 │
│   동사        yo 형태    나머지는 규칙적                        │
├─────────────────────────────────────────────────────────────────┤
│   tener       tengo      tienes, tiene, tenemos, tenéis, tienen│
│   venir       vengo      vienes, viene, venimos, venís, vienen │
│   poner       pongo      pones, pone, ponemos, ponéis, ponen   │
│   salir       salgo      sales, sale, salimos, salís, salen    │
│   hacer       hago       haces, hace, hacemos, hacéis, hacen   │
│   decir       digo       dices, dice, decimos, decís, dicen    │
│   traer       traigo     traes, trae, traemos, traéis, traen   │
│   caer        caigo      caes, cae, caemos, caéis, caen        │
│   oír         oigo       oyes, oye, oímos, oís, oyen           │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 완전 불규칙 동사

```
ser (이다/~이다):
┌────────┬────────┬────────┬────────┬────────┬────────┐
│ 시제   │ yo     │ tú     │ él     │ nos.   │ ellos  │
├────────┼────────┼────────┼────────┼────────┼────────┤
│ 현재   │ soy    │ eres   │ es     │ somos  │ son    │
│ 단순과거│ fui    │ fuiste │ fue    │ fuimos │ fueron │
│ 불완료 │ era    │ eras   │ era    │ éramos │ eran   │
└────────┴────────┴────────┴────────┴────────┴────────┘

ir (가다):
┌────────┬────────┬────────┬────────┬────────┬────────┐
│ 시제   │ yo     │ tú     │ él     │ nos.   │ ellos  │
├────────┼────────┼────────┼────────┼────────┼────────┤
│ 현재   │ voy    │ vas    │ va     │ vamos  │ van    │
│ 단순과거│ fui    │ fuiste │ fue    │ fuimos │ fueron │
│ 불완료 │ iba    │ ibas   │ iba    │ íbamos │ iban   │
└────────┴────────┴────────┴────────┴────────┴────────┘

※ ser와 ir의 단순 과거가 동일함!

estar (있다/~한 상태이다):
┌────────┬────────┬────────┬────────┬────────┬────────┐
│ 시제   │ yo     │ tú     │ él     │ nos.   │ ellos  │
├────────┼────────┼────────┼────────┼────────┼────────┤
│ 현재   │ estoy  │ estás  │ está   │ estamos│ están  │
│ 단순과거│ estuve │estuviste│estuvo │estuvimos│estuvieron│
└────────┴────────┴────────┴────────┴────────┴────────┘
```

│ estar    │ estuv-     │ estuve, estuviste, estuvo...           │
│ poder    │ pud-       │ pude, pudiste, pudo...                 │
│ poner    │ pus-       │ puse, pusiste, puso...                 │

## 4. 재귀 동사

### 4.1 재귀 대명사

```
┌─────────────────────────────────────────────────────────────────┐
│                    재귀 대명사                                   │
│                                                                 │
│   주어         재귀 대명사       예시 (lavarse - 씻다)          │
├─────────────────────────────────────────────────────────────────┤
│   yo           me               me lavo (나는 씻는다)           │
│   tú           te               te lavas (너는 씻는다)          │
│   él/ella/Ud.  se               se lava (그는 씻는다)          │
│   nosotros     nos              nos lavamos (우리는 씻는다)     │
│   vosotros     os               os laváis (너희는 씻는다)       │
│   ellos/Uds.   se               se lavan (그들은 씻는다)       │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 재귀 동사 유형

```
1. 일상 활동 (Daily routines):
   levantarse (일어나다)    → Me levanto a las 7.
   acostarse (눕다/자다)    → Se acuesta tarde.
   ducharse (샤워하다)      → Nos duchamos por la mañana.
   vestirse (옷 입다)       → Te vistes rápido.
   peinarse (머리 빗다)     → Me peino antes de salir.
   afeitarse (면도하다)     → Se afeita todos los días.

2. 감정/상태 변화:
   alegrarse (기뻐하다)     → Me alegro de verte.
   enojarse (화나다)        → Se enoja fácilmente.
   aburrirse (지루해하다)   → Nos aburrimos en clase.
   preocuparse (걱정하다)   → No te preocupes.
   sentirse (느끼다)        → Me siento bien.

3. 상호 동작 (서로):
   conocerse (서로 알다)    → Nos conocemos desde niños.
   ayudarse (서로 돕다)     → Se ayudan mucho.
   quererse (서로 사랑하다) → Se quieren mucho.
```

### 4.3 재귀 vs 비재귀

```
비재귀 (타인에게)          재귀 (자신에게)
─────────────────────────────────────────────────────
lavar (씻기다)             lavarse (씻다)
Lavo el coche.             Me lavo las manos.
(차를 씻는다)              (손을 씻는다)

llamar (부르다)            llamarse (이름이 ~이다)
Llamo a María.             Me llamo Juan.
(마리아를 부른다)          (내 이름은 후안이다)

despertar (깨우다)         despertarse (깨다)
Despierto a mi hijo.       Me despierto temprano.
(아들을 깨운다)            (일찍 일어난다)

ir (가다)                  irse (떠나다)
Voy al trabajo.            Me voy ahora.
(직장에 간다)              (지금 떠난다)
```

## 5. 복합 시제

### 5.1 haber 활용

```
┌────────┬──────────────────────────────────────────────────────────┐
│ 시제   │ haber 활용                                               │
├────────┼──────────────────────────────────────────────────────────┤
│ 현재   │ he, has, ha, hemos, habéis, han                         │
│ 불완료 │ había, habías, había, habíamos, habíais, habían         │
│ 접속법 │ haya, hayas, haya, hayamos, hayáis, hayan               │
│ 과거접속│ hubiera, hubieras, hubiera, hubiéramos, hubierais...   │
└────────┴──────────────────────────────────────────────────────────┘

과거분사 형성:
-AR → -ado: hablar → hablado
-ER → -ido: comer → comido
-IR → -ido: vivir → vivido

불규칙 과거분사:
hacer → hecho      poner → puesto      ver → visto
escribir → escrito  abrir → abierto     decir → dicho
volver → vuelto     morir → muerto      romper → roto
```

### 5.2 복합 시제 형성

```
현재 완료 (Pretérito Perfecto):
haber (현재) + 과거분사
─────────────────────────────────────────
He hablado con él. (그와 이야기했다.)
¿Has comido? (먹었어?)
Han llegado tarde. (그들은 늦게 도착했다.)

과거 완료 (Pluscuamperfecto):
haber (불완료) + 과거분사
─────────────────────────────────────────
Había terminado cuando llegaste.
(네가 도착했을 때 이미 끝냈었다.)

미래 완료 (Futuro Perfecto):
haber (미래) + 과거분사
─────────────────────────────────────────
Habrá llegado para las cinco.
(5시까지는 도착했을 것이다.)

조건 완료 (Condicional Perfecto):
haber (조건) + 과거분사
─────────────────────────────────────────
Habría ido si hubiera sabido.
(알았더라면 갔을 텐데.)

접속법 현재 완료:
haber (접속법 현재) + 과거분사
─────────────────────────────────────────
Espero que haya llegado bien.
(무사히 도착했길 바란다.)

접속법 과거 완료:
haber (접속법 과거) + 과거분사
─────────────────────────────────────────
Si hubiera sabido, habría venido.
(알았더라면, 왔을 텐데.)
```

## 6. 연습 문제

### 연습 1: 어간 변화 동사 활용
빈칸을 채우세요.

```
1. pensar (yo, 현재) → _______
   → pienso

2. poder (ellos, 현재) → _______
   → pueden

3. pedir (tú, 현재) → _______
   → pides

4. dormir (nosotros, 현재) → _______
   → dormimos (변화 없음)

5. querer (ella, 현재) → _______
   → quiere
```
