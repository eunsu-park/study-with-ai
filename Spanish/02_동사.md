## 3. 자주 쓰이는 재귀동사

재귀동사는 -se로 끝나며, 행동이 주어 자신에게 돌아옵니다.

### 일상 루틴 (Daily Routine)

1. **levantarse** / to get up / 일어나다
   - Me levanto a las 7 / I get up at 7 / 나는 7시에 일어난다

2. **despertarse** / to wake up / 깨다
   - Me despierto temprano / I wake up early / 나는 일찍 깬다

3. **ducharse** / to shower / 샤워하다
   - Me ducho por la mañana / I shower in the morning / 나는 아침에 샤워한다

4. **bañarse** / to bathe / 목욕하다
   - Me baño cada día / I bathe every day / 나는 매일 목욕한다

5. **lavarse** / to wash oneself / 씻다
   - Me lavo las manos / I wash my hands / 나는 손을 씻는다

6. **cepillarse** / to brush / 닦다
   - Me cepillo los dientes / I brush my teeth / 나는 이를 닦는다

7. **peinarse** / to comb one's hair / 머리를 빗다
   - Me peino el pelo / I comb my hair / 나는 머리를 빗는다

8. **afeitarse** / to shave / 면도하다
   - Me afeito cada día / I shave every day / 나는 매일 면도한다

9. **maquillarse** / to put on makeup / 화장하다
   - Me maquillo por la mañana / I put on makeup in the morning / 나는 아침에 화장한다

10. **vestirse** / to get dressed / 옷을 입다
    - Me visto rápido / I get dressed quickly / 나는 빨리 옷을 입는다

11. **acostarse** / to go to bed / 자러 가다
    - Me acuesto a las 11 / I go to bed at 11 / 나는 11시에 잔다

12. **dormirse** / to fall asleep / 잠들다
    - Me duermo rápido / I fall asleep quickly / 나는 빨리 잠든다

### 위치와 자세 (Position)

13. **sentarse** / to sit down / 앉다
    - Me siento aquí / I sit here / 나는 여기 앉는다

14. **pararse** / to stand up / 서다
    - Me paro cuando entra el profesor / I stand up when the teacher enters / 선생님이 들어오면 나는 선다

15. **quedarse** / to stay / 머무르다
    - Me quedo en casa / I stay home / 나는 집에 머문다

### 이동 (Movement)

16. **irse** / to leave, to go away / 떠나다
    - Me voy ahora / I'm leaving now / 나는 지금 간다

17. **mudarse** / to move (residence) / 이사하다
    - Me mudo a Madrid / I'm moving to Madrid / 나는 마드리드로 이사한다

18. **acercarse** / to approach, get closer / 다가가다
    - Me acerco a la ventana / I approach the window / 나는 창문으로 다가간다

19. **alejarse** / to move away / 멀어지다
    - Me alejo del peligro / I move away from danger / 나는 위험에서 멀어진다

### 감정과 반응 (Emotions and Reactions)

20. **alegrarse** / to be happy / 기뻐하다
    - Me alegro de verte / I'm happy to see you / 너를 만나서 기쁘다

21. **preocuparse** / to worry / 걱정하다
    - Me preocupo por ti / I worry about you / 나는 너를 걱정한다

22. **enojarse** / to get angry / 화나다
    - Me enojo fácilmente / I get angry easily / 나는 쉽게 화난다

23. **asustarse** / to get scared / 놀라다
    - Me asusto con las películas de terror / I get scared with horror movies / 나는 공포 영화에 놀란다

24. **aburrirse** / to get bored / 지루해하다
    - Me aburro en clase / I get bored in class / 나는 수업에서 지루해한다

25. **divertirse** / to have fun / 즐기다
    - Me divierto con mis amigos / I have fun with my friends / 나는 친구들과 즐긴다

26. **enamorarse** / to fall in love / 사랑에 빠지다
    - Me enamoro fácilmente / I fall in love easily / 나는 쉽게 사랑에 빠진다

27. **sorprenderse** / to be surprised / 놀라다
    - Me sorprendo con la noticia / I'm surprised by the news / 나는 그 소식에 놀란다

28. **sentirse** / to feel / 느끼다
    - Me siento bien / I feel good / 나는 기분이 좋다

### 인지와 정체성 (Cognition and Identity)

29. **llamarse** / to be called / ~라고 불리다
    - Me llamo Juan / My name is Juan / 내 이름은 후안이다

30. **acordarse** / to remember / 기억하다
    - Me acuerdo de ti / I remember you / 나는 너를 기억한다

31. **olvidarse** / to forget / 잊다
    - Me olvido de las llaves / I forget the keys / 나는 열쇠를 잊는다

32. **darse cuenta** / to realize / 깨닫다
    - Me doy cuenta del error / I realize the mistake / 나는 실수를 깨닫는다

33. **equivocarse** / to make a mistake / 실수하다
    - Me equivoco a veces / I make mistakes sometimes / 나는 때때로 실수한다

### 상호 작용 (Reciprocal Actions)

34. **conocerse** / to know each other / 서로 알다
    - Nos conocemos bien / We know each other well / 우리는 서로 잘 안다

35. **verse** / to see each other / 서로 보다
    - Nos vemos mañana / We'll see each other tomorrow / 우리는 내일 볼 거야

36. **hablarse** / to talk to each other / 서로 말하다
    - No nos hablamos / We don't talk to each other / 우리는 서로 말하지 않는다

37. **escribirse** / to write to each other / 서로 쓰다
    - Nos escribimos mensajes / We write messages to each other / 우리는 서로 메시지를 쓴다

38. **ayudarse** / to help each other / 서로 돕다
    - Nos ayudamos mucho / We help each other a lot / 우리는 서로 많이 돕는다

39. **quererse** / to love each other / 서로 사랑하다
    - Nos queremos mucho / We love each other a lot / 우리는 서로 많이 사랑한다

40. **pelearse** / to fight with each other / 서로 싸우다
    - Nos peleamos a veces / We fight sometimes / 우리는 때때로 싸운다

## 4. 동사 활용 패턴

### A. 규칙 동사 현재형

#### -AR 동사 (hablar)
| 주어 | 활용 | 의미 |
|------|------|------|
| yo | hablo | I speak / 나는 말한다 |
| tú | hablas | you speak / 너는 말한다 |
| él/ella/usted | habla | he/she speaks / 그/그녀는 말한다 |
| nosotros/as | hablamos | we speak / 우리는 말한다 |
| vosotros/as | habláis | you all speak / 너희는 말한다 |
| ellos/ellas/ustedes | hablan | they speak / 그들은 말한다 |

#### -ER 동사 (comer)
| 주어 | 활용 | 의미 |
|------|------|------|
| yo | como | I eat / 나는 먹는다 |
| tú | comes | you eat / 너는 먹는다 |
| él/ella/usted | come | he/she eats / 그/그녀는 먹는다 |
| nosotros/as | comemos | we eat / 우리는 먹는다 |
| vosotros/as | coméis | you all eat / 너희는 먹는다 |
| ellos/ellas/ustedes | comen | they eat / 그들은 먹는다 |

#### -IR 동사 (vivir)
| 주어 | 활용 | 의미 |
|------|------|------|
| yo | vivo | I live / 나는 산다 |
| tú | vives | you live / 너는 산다 |
| él/ella/usted | vive | he/she lives / 그/그녀는 산다 |
| nosotros/as | vivimos | we live / 우리는 산다 |
| vosotros/as | vivís | you all live / 너희는 산다 |
| ellos/ellas/ustedes | viven | they live / 그들은 산다 |

### B. 불규칙 동사 현재형

#### SER (to be)
yo soy, tú eres, él/ella/usted es, nosotros somos, vosotros sois, ellos son

#### ESTAR (to be)
yo estoy, tú estás, él/ella/usted está, nosotros estamos, vosotros estáis, ellos están

#### TENER (to have)
yo tengo, tú tienes, él/ella/usted tiene, nosotros tenemos, vosotros tenéis, ellos tienen

#### IR (to go)
yo voy, tú vas, él/ella/usted va, nosotros vamos, vosotros vais, ellos van

#### HACER (to do/make)
yo hago, tú haces, él/ella/usted hace, nosotros hacemos, vosotros hacéis, ellos hacen

#### DECIR (to say)
yo digo, tú dices, él/ella/usted dice, nosotros decimos, vosotros decís, ellos dicen

#### PODER (can)
yo puedo, tú puedes, él/ella/usted puede, nosotros podemos, vosotros podéis, ellos pueden

#### QUERER (to want)
yo quiero, tú quieres, él/ella/usted quiere, nosotros queremos, vosotros queréis, ellos quieren

#### SABER (to know)
yo sé, tú sabes, él/ella/usted sabe, nosotros sabemos, vosotros sabéis, ellos saben

#### VENIR (to come)
yo vengo, tú vienes, él/ella/usted viene, nosotros venimos, vosotros venís, ellos vienen

### C. 어간 변화 동사 (Stem-Changing Verbs)

#### E → IE
**pensar** (to think): pienso, piensas, piensa, pensamos, pensáis, piensan
**querer** (to want): quiero, quieres, quiere, queremos, queréis, quieren
**preferir** (to prefer): prefiero, prefieres, prefiere, preferimos, preferís, prefieren

#### O → UE
**poder** (can): puedo, puedes, puede, podemos, podéis, pueden
**dormir** (to sleep): duermo, duermes, duerme, dormimos, dormís, duermen
**volver** (to return): vuelvo, vuelves, vuelve, volvemos, volvéis, vuelven

#### E → I
**pedir** (to ask for): pido, pides, pide, pedimos, pedís, piden
**servir** (to serve): sirvo, sirves, sirve, servimos, servís, sirven
**repetir** (to repeat): repito, repites, repite, repetimos, repetís, repiten

### D. 재귀동사 활용 (Reflexive Verb Conjugation)

**levantarse** (to get up):
- yo **me** levanto
- tú **te** levantas
- él/ella/usted **se** levanta
- nosotros/as **nos** levantamos
- vosotros/as **os** levantáis
- ellos/ellas/ustedes **se** levantan

## 학습 팁 (Study Tips)

1. **매일 동사 연습**: 하루에 동사 5개씩 활용 연습
2. **문장으로 외우기**: 단독 동사보다 문장 속에서 암기
3. **재귀동사 일상화**: 하루 루틴을 스페인어로 말하기
4. **불규칙 동사 집중**: 자주 쓰는 불규칙 동사부터 마스터
5. **활용표 작성**: 자주 쓰는 동사의 활용표 만들어 반복 학습
6. **듣기와 병행**: 원어민 발음으로 동사 활용 익히기
7. **상황별 분류**: 테마별로 동사 그룹화하여 학습
