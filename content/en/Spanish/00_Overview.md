# Spanish Learning Guide

## Introduction

This folder contains reference materials for learning Spanish grammar and vocabulary. You can systematically learn Spanish through materials organized by parts of speech.

**Target Audience**: Spanish beginners to intermediate learners

## Learning Sequence

1. **Grammar Basics**: 01_Grammar → 03_Nouns → 04_Adjectives
2. **Verb Study**: 02_Verbs → 07_Verb_Conjugation_System
3. **Other Parts of Speech**: 05_Other_Parts_of_Speech (including adverbs, interjections, discourse markers)
4. **Advanced Grammar**: 06_Subjunctive_Advanced → 10_Advanced_Grammar_Topics
5. **Practical Expressions**: 08_Regional_Variations_Dialects → 09_Daily_Conversation_Idioms → 11_Vocabulary_by_Theme
