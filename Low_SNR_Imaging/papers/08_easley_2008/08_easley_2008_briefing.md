# Discrete Shearlet (Easley+ 2008): Pre-Reading Briefing / 사전 읽기 브리핑

**Paper**: Easley, G., Labate, D., & Lim, W.-Q., "Sparse directional image representations using the discrete shearlet transform", *Applied and Computational Harmonic Analysis* 25(1), 25–46 (2008). [DOI: 10.1016/j.acha.2007.09.003]
**Author(s)**: Glenn Easley, Demetrio Labate, Wang-Q Lim
**Year**: 2008

---

## 1. 핵심 기여 / Core Contribution

### 한국어
이 논문은 **shearlet transform의 첫 이산 구현**을 제시한다. Shearlet은 *composite wavelet* 이론의 특수 사례로, anisotropic dilation $A_0=\mathrm{diag}(4,2)$와 *unit shear* $B_0=\bigl(\begin{smallmatrix}1&1\\0&1\end{smallmatrix}\bigr)$의 두 행렬군이 단일 mother $\psi^{(0)}$에 작용하여 시스템을 생성한다. 핵심 우위는 (i) *shear*가 정수 격자를 보존해 디지털 격자 친화적, (ii) curvelet과 동등한 최적 NLA rate $O(N^{-2}(\log N)^3)$, (iii) Laplacian Pyramid + pseudo-polar grid의 깔끔한 $O(N^2 \log N)$ 알고리즘, (iv) curvelet은 *every other* scale에서만 방향 수가 doubling하는 반면 shearlet은 *every* scale에서 doubling하여 같은 spatial scale에서 더 fine한 angular resolution을 제공한다는 점이다.

### English
This paper delivers the **first discrete implementation of the shearlet transform** — a special case of *composite wavelets* generated by anisotropic dilation $A_0=\mathrm{diag}(4,2)$ and *unit shear* $B_0$. Four advantages over curvelets/contourlets: (i) shears preserve the integer lattice, making the construction Cartesian-natural; (ii) optimal nonlinear-approximation rate $O(N^{-2}(\log N)^3)$ for piecewise-$C^2$ images with $C^2$ singularity curves — equivalent to curvelets; (iii) clean discrete algorithm via Laplacian Pyramid + pseudo-polar grid in $O(N^2 \log N)$; (iv) directional resolution doubles at *every* finer scale (vs every-other for curvelets).

## 2. 역사적 맥락 / Historical Context

### 시대 배경 / The Setting

#### 한국어
2000년대 초반, 표준 wavelet은 *isotropic*이라 곡선 edge 표현에 비효율적($\varepsilon_M \sim M^{-1}$)임이 알려져, *방향성 multiscale 변환*들이 폭발적으로 등장했다 — Candès–Donoho의 curvelet(rotation 기반, polar geometry), Do–Vetterli의 contourlet(filter-bank 기반, 논문 #5), 그리고 Guo-Kutyniok-Labate의 composite wavelet 프레임워크(2005). Curvelet은 이론적으로 우아했지만 polar 좌표라 격자 이산화가 어려웠고, contourlet은 격자 친화적이었지만 MRA가 없어 이론과 실제 사이의 불일치가 있었다. Shearlet은 이 둘을 동시에 해결하기 위해 등장한다.

#### English
In the early 2000s standard wavelets were known to be *isotropic*, hence inefficient for curved edges ($\varepsilon_M \sim M^{-1}$). A flurry of *directional multiscale transforms* followed: Candès–Donoho's curvelets (rotation, polar geometry), Do–Vetterli's contourlets (filter-bank, paper #5), and Guo-Kutyniok-Labate's composite-wavelet framework (2005). Curvelets were elegant but polar (hard to discretise on grids); contourlets were Cartesian but lacked MRA. Shearlets resolve both tensions simultaneously.

### 타임라인 / Timeline

```
1989 ─── Mallat — wavelet MRA
2000-04 ─ Candès-Donoho — curvelets (1st & 2nd generation)
2005 ─── Do-Vetterli — Contourlet (paper #5)
2005 ─── Guo-Kutyniok-Labate — Composite wavelet framework
2006 ─── Candès-Demanet-Donoho-Ying — Fast Discrete Curvelet (paper #6)
2007 ─── Labate-Lim-Kutyniok-Weiss — first (continuous) shearlet paper
2008 ★★ EASLEY-LABATE-LIM — Discrete Shearlet Transform (THIS PAPER)
2010 ─── Lim — anisotropic-subsampled shearlet (low-redundancy)
2014+ ── ShearLab software, pyshearlab, GPU implementations
```

## 4. 핵심 용어 / Key Vocabulary

| 용어 / Term | 설명 / Explanation |
|---|---|
| Composite wavelet / 복합 웨이블릿 | 두 행렬 군 $A^j$ (scale)과 $B^\ell$ (shear/rotation)으로 정의되는 affine system. / Affine system from two matrix groups: $A^j$ (scale), $B^\ell$ (shear/rotation). |
| Anisotropic dilation / 비등방 확장 | $A_0 = \mathrm{diag}(4, 2)$ — $\xi_1$ 4×, $\xi_2$ 2× scale. Parabolic scaling 구현. / Diagonal scaling with different rates per axis, implementing parabolic scaling. |
| Shear matrix / 전단 행렬 | $B_0 = \bigl(\begin{smallmatrix}1&1\\0&1\end{smallmatrix}\bigr)$ — area-preserving, lattice-preserving. / Area- and lattice-preserving 2×2 matrix. |
| Parabolic scaling / 포물선 스케일링 | spatial support $2^{-j} \times 2^{-2j}$, *width ∝ length²*. Curvelet과 공유. / Spatial scaling where width scales as length squared. |
| Parseval frame / 파스발 프레임 | $\sum |\langle f, \psi_n\rangle|^2 = \|f\|^2$를 만족하는 (보통 redundant) 시스템. / Possibly redundant system satisfying $\sum|\langle f,\psi\rangle|^2 = \|f\|^2$. |
| Cone $\mathcal D_0, \mathcal D_1$ / 콘 | 주파수 영역을 수평/수직 cone으로 분할 → 두 셋의 shearlet system. / Frequency-domain split into horizontal/vertical cones for two shearlet systems. |
| Mother shearlet / 모-쉬어릿 | $\hat\psi^{(0)}(\xi) = \hat\psi_1(\xi_1)\hat\psi_2(\xi_2/\xi_1)$ — radial × angular 분리. / Defined as a radial × angular product in frequency. |
| Pseudo-polar grid / 의사극격자 | $(\xi_1, \xi_2) \to (\xi_1, \xi_2/\xi_1)$ — Cartesian DFT의 *변광 기울기 sample lines*. / Cartesian DFT samples along lines of varying slope through origin. |
| Laplacian Pyramid / 라플라시안 피라미드 | Multi-scale 분해: bandpass + coarse residual. / Multi-scale decomposition: bandpass + low-pass residual. |
| NLA rate / 비선형 근사 속도 | top-$N$ coefficient 재구성 오차 $\|f - f^S_N\|^2$의 $N$ 의존성. / Decay rate of best $N$-term reconstruction error. |
| Fast slant stack / 패스트 슬랜트 스택 | Pseudo-polar grid 위의 효율적 FFT 추출 알고리즘 ($O(N^2 \log N)$). / Efficient FFT extraction on pseudo-polar grid. |
| Anisotropic subsampling / 비등방 서브샘플링 | 본 논문에서는 *미적용* — 이후 Lim 2010에서 도입. / Not used in this paper; introduced later for low-redundancy variants. |

## 5. 수식 미리보기 / Equations Preview

**Composite wavelet system / 복합 웨이블릿 시스템**:
$$
\mathcal A_{AB}(\psi) = \{|\det A|^{j/2}\,\psi(B^\ell A^j x - k): j, \ell \in \mathbb Z, k \in \mathbb Z^2\}
$$

**Shearlet 행렬과 mother / Shearlet matrices and mother**:
$$
A_0 = \begin{pmatrix}4 & 0\\ 0 & 2\end{pmatrix}, \quad B_0 = \begin{pmatrix}1 & 1\\ 0 & 1\end{pmatrix}, \quad \hat\psi^{(0)}(\xi_1, \xi_2) = \hat\psi_1(\xi_1)\,\hat\psi_2(\xi_2/\xi_1)
$$

**Admissibility / 인정 조건 (Parseval frame for cone $\mathcal D_0$)**:
$$
\sum_j |\hat\psi_1(2^{-2j}\omega)|^2 = 1, \quad \sum_\ell |\hat\psi_2(2^j \omega - \ell)|^2 = 1
$$

**최적 NLA rate / Optimal nonlinear-approximation rate**:
$$
\|f - f^S_N\|^2_2 \le C\,N^{-2}(\log N)^3 \quad (f \in C^2 \text{ piecewise}, C^2 \text{ singularity curves})
$$

**Discrete shearlet 계수 / Discrete shearlet coefficient**:
$$
\langle f, \psi^{(0)}_{j,\ell,k}\rangle \approx \mathrm{IFFT}\bigl(\hat f_d^j(u, v)\,\overline{W(2^j v - \ell)}\bigr)
$$

## 6. 읽기 가이드 / Reading Guide

### 한국어
- **§1 (동기)**: wavelet의 $M^{-1}$ 한계, curvelet의 polar 어려움, contourlet의 MRA 부재 — 세 동기를 정리한다.
- **§2 (정의)**: Eq. 2.2의 composite wavelet 정의와 Eq. 2.3의 mother shearlet $\hat\psi^{(0)} = \hat\psi_1(\xi_1)\hat\psi_2(\xi_2/\xi_1)$를 *천천히* 음미. Fig. 1b의 trapezoidal frequency support와 spatial support의 Fourier-spatial duality를 이해해야 이후 모든 설명이 명료해진다.
- **§2 두 cone 트릭**: 단일 cone만으로는 모든 방향을 cover할 수 없음 → $\mathcal D_0, \mathcal D_1$ 두 개로 분할 → Theorem 2.1의 Parseval frame.
- **§3 (이산 알고리즘)**: Fig. 2의 5-step pipeline (LP → DFT → pseudo-polar → 1-D bandpass → IFFT)을 코드 흐름처럼 따라가기. Pseudo-polar grid 정의(§3.1 Definition 3.1)가 핵심.
- **§4 (실험)**: 짧음 — 기본 PSNR 향상만 확인.
- **흔한 오해**: shearlet의 "shear"는 *영상 자체를 회전*하는 것이 아니라 *주파수 박스를 기울이는* 것. 격자에서 자연스러운 이유.
- **수학적 함정**: $\psi^{(0)}$은 cone $\mathcal D_0$에 대해서만 Parseval frame을 만든다. 전체 평면을 다루려면 $\psi^{(1)}$과 결합해야 함 (Theorem 2.1).

### English
- **§1 (motivation)**: organise the three motivations — wavelets' $M^{-1}$ limit, curvelets' polar awkwardness, contourlets' missing MRA.
- **§2 (definitions)**: meditate on Eq. 2.2 (composite wavelets) and Eq. 2.3 (mother shearlet $\hat\psi^{(0)} = \hat\psi_1(\xi_1)\hat\psi_2(\xi_2/\xi_1)$). Internalise Fig. 1b's trapezoidal frequency support and its spatial dual.
- **§2 two-cone trick**: a single cone cannot cover all directions, hence the split $\mathcal D_0, \mathcal D_1$ and the Parseval-frame Theorem 2.1.
- **§3 (discrete algorithm)**: trace Fig. 2's 5-step pipeline (LP → DFT → pseudo-polar → 1-D bandpass → IFFT) as if writing code. Definition 3.1 of the pseudo-polar grid is the key.
- **§4 (experiments)**: short — confirm baseline PSNR gains.
- **Pitfall**: the "shear" does not rotate the image; it tilts the frequency boxes — that is why it is grid-friendly.
- **Math trap**: $\psi^{(0)}$ is a Parseval frame *only* for cone $\mathcal D_0$; full-plane coverage requires combining with $\psi^{(1)}$ (Theorem 2.1).

## 7. 현대적 의의 / Modern Significance

### 한국어
이산 shearlet 변환은 *수학적 보증*과 *디지털 격자 친화성*을 동시에 제공하는 가장 깔끔한 방향성 변환으로, 의료영상(CT denoising, MRI artifact 억제), 천체물리(코로나 loop 검출), 형광현미경 등 *훈련 데이터가 부족한* 과학영상 도메인에서 여전히 사용된다. ShearLab(2014+), pyshearlab, GPU 구현 등 풍부한 소프트웨어 생태계가 존재한다. 또한 2017+ deep learning 시대에도 shearlet은 여전히 *prior*로서 가치 있는데, (i) hard-thresholded shearlet은 deep CNN baseline보다 작은 모델로 비슷한 성능, (ii) plug-and-play 프레임워크의 regulariser로 사용, (iii) 학습 가능 shearlet ("ShearletNet" 2018+)이 학습된 dictionary와 결합. 본 논문이 정립한 LP + pseudo-polar 알고리즘은 이후 모든 shearlet 변형의 기본 골격이다.

### English
The discrete shearlet transform is the cleanest directional multiscale transform combining *mathematical guarantees* with *Cartesian-grid friendliness*. It remains in use in medical imaging (CT denoising, MRI artifact suppression), astrophysics (coronal loop detection), and fluorescence microscopy — domains where training data is scarce. A rich software ecosystem (ShearLab, pyshearlab, GPU implementations) backs continued adoption. Even in the deep-learning era, shearlets remain valuable as priors: (i) hard-thresholded shearlets match small CNN baselines; (ii) they serve as regularisers in plug-and-play frameworks; (iii) learnable shearlets ("ShearletNet", 2018+) merge them with learned dictionaries. The LP + pseudo-polar algorithm formalised here is the skeleton of every later shearlet variant.
