## 1. 접속법 개요

### 1.1 접속법이란?

```
┌─────────────────────────────────────────────────────────────────┐
│                    서법(Mood) 비교                              │
│                                                                 │
│   직설법 (Indicativo)     접속법 (Subjuntivo)                   │
│   ├── 사실, 확실성        ├── 희망, 의심, 감정                  │
│   ├── 객관적 진술         ├── 주관적 태도                       │
│   └── 현실                └── 가상, 불확실                      │
│                                                                 │
│   예시:                                                         │
│   Sé que viene. (온다는 것을 안다) - 직설법                     │
│   Espero que venga. (오길 바란다) - 접속법                      │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 접속법 시제 체계

```
접속법 시제:
┌────────────────────────────────────────────────────────────────┐
│ 현재 접속법 (Presente de Subjuntivo)                           │
│   → 현재/미래의 불확실한 상황                                   │
│   Espero que llegue pronto. (곧 도착하길 바란다)               │
│                                                                 │
│ 과거 접속법 (Imperfecto de Subjuntivo)                         │
│   → 과거/비현실적 상황, 공손한 요청                             │
│   Si tuviera tiempo, viajaría. (시간이 있다면, 여행할 텐데)    │
│                                                                 │
│ 현재 완료 접속법 (Pretérito Perfecto de Subjuntivo)            │
│   → 과거 완료된 불확실한 상황                                   │
│   Espero que haya llegado. (도착했길 바란다)                   │
│                                                                 │
│ 과거 완료 접속법 (Pluscuamperfecto de Subjuntivo)              │
│   → 과거의 비현실적 완료 상황                                   │
│   Si hubiera sabido... (알았더라면...)                         │
└────────────────────────────────────────────────────────────────┘
```

## 2. 현재 접속법

### 2.1 규칙 활용

```
┌─────────────────────────────────────────────────────────────────┐
│                 현재 접속법 규칙 활용                            │
│                                                                 │
│   -AR 동사        -ER 동사        -IR 동사                      │
│   (hablar)       (comer)         (vivir)                       │
├─────────────────────────────────────────────────────────────────┤
│   hable          coma            viva                          │
│   hables         comas           vivas                         │
│   hable          coma            viva                          │
│   hablemos       comamos         vivamos                       │
│   habléis        comáis          viváis                        │
│   hablen         coman           vivan                         │
│                                                                 │
│   규칙: -AR 동사는 -e, -ER/-IR 동사는 -a 모음 사용             │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 불규칙 활용

```
주요 불규칙 동사:
┌──────────┬───────────────────────────────────────────────────────┐
│ 동사     │ 현재 접속법                                           │
├──────────┼───────────────────────────────────────────────────────┤
│ ser      │ sea, seas, sea, seamos, seáis, sean                  │
│ estar    │ esté, estés, esté, estemos, estéis, estén            │
│ ir       │ vaya, vayas, vaya, vayamos, vayáis, vayan            │
│ haber    │ haya, hayas, haya, hayamos, hayáis, hayan            │
│ saber    │ sepa, sepas, sepa, sepamos, sepáis, sepan            │
│ dar      │ dé, des, dé, demos, deis, den                        │
│ ver      │ vea, veas, vea, veamos, veáis, vean                  │
├──────────┼───────────────────────────────────────────────────────┤
│ 어간 변화 동사 (e→ie, o→ue, e→i 패턴 유지)                     │
│ pensar   │ piense, pienses, piense, pensemos, penséis, piensen  │
│ poder    │ pueda, puedas, pueda, podamos, podáis, puedan        │
│ pedir    │ pida, pidas, pida, pidamos, pidáis, pidan            │
└──────────┴───────────────────────────────────────────────────────┘
```

### 2.3 사용 예문

```
희망/바람:
- Espero que tengas un buen día.
  (좋은 하루 보내길 바란다.)
- Ojalá llegue a tiempo.
  (제시간에 도착하길 바란다.)

감정:
- Me alegra que estés aquí.
  (네가 여기 있어서 기쁘다.)
- Siento que no puedas venir.
  (네가 올 수 없어서 유감이다.)

의심/불확실:
- No creo que sea verdad.
  (그게 사실이라고 생각하지 않는다.)
- Dudo que lo sepa.
  (그가 알고 있는지 의심스럽다.)

명령/요청:
- Quiero que vengas conmigo.
  (나와 함께 오길 원한다.)
- Te pido que me ayudes.
  (도와달라고 부탁한다.)
```

## 3. 과거 접속법

### 3.1 활용 형태

```
┌─────────────────────────────────────────────────────────────────┐
│            과거 접속법 (Imperfecto de Subjuntivo)               │
│                                                                 │
│   두 가지 형태 (-ra / -se)                                      │
│   (의미는 동일, -ra 형태가 더 일반적)                           │
│                                                                 │
│   hablar                                                        │
│   -ra 형태           -se 형태                                   │
│   hablara            hablase                                    │
│   hablaras           hablases                                   │
│   hablara            hablase                                    │
│   habláramos         hablásemos                                 │
│   hablarais          hablaseis                                  │
│   hablaran           hablasen                                   │
│                                                                 │
│   comer: comiera/comiese, comieras/comieses...                 │
│   vivir: viviera/viviese, vivieras/vivieses...                 │
│                                                                 │
│   ※ 어간은 직설법 단순과거 3인칭 복수에서 파생                   │
│   hablaron → habla- → hablara                                  │
│   comieron → comie- → comiera                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 불규칙 동사

```
과거 접속법 불규칙:
┌──────────┬───────────────────────────────────────────────────────┐
│ 동사     │ 과거 접속법 (-ra 형태)                                │
├──────────┼───────────────────────────────────────────────────────┤
│ ser/ir   │ fuera, fueras, fuera, fuéramos, fuerais, fueran     │
│ estar    │ estuviera, estuvieras, estuviera...                  │
│ tener    │ tuviera, tuvieras, tuviera...                        │
│ poder    │ pudiera, pudieras, pudiera...                        │
│ poner    │ pusiera, pusieras, pusiera...                        │
│ saber    │ supiera, supieras, supiera...                        │
│ hacer    │ hiciera, hicieras, hiciera...                        │
│ decir    │ dijera, dijeras, dijera...                           │
│ querer   │ quisiera, quisieras, quisiera...                     │
│ venir    │ viniera, vinieras, viniera...                        │
└──────────┴───────────────────────────────────────────────────────┘
```

### 3.3 사용 상황

```
1. 과거 시점의 불확실한 상황:
   - Esperaba que viniera. (그가 오길 바랐다.)
   - No creía que fuera posible. (가능하다고 생각하지 않았다.)

2. 공손한 요청 (quisiera):
   - Quisiera un café, por favor. (커피 한 잔 부탁드립니다.)
   - Quisiera hablar con el gerente. (매니저와 이야기하고 싶습니다.)

3. 가정/조건문:
   - Si pudiera, te ayudaría. (할 수 있다면, 도와줄 텐데.)
   - Si fuera rico, viajaría mucho. (부자라면, 많이 여행할 텐데.)

4. "como si" (마치 ~인 것처럼):
   - Habla como si supiera todo. (모든 것을 아는 것처럼 말한다.)
   - Actúa como si no pasara nada. (아무 일도 없는 것처럼 행동한다.)
```

## 4. 접속법 트리거

### 4.1 WEIRDO 규칙

```
┌─────────────────────────────────────────────────────────────────┐
│                    WEIRDO 기억법                                 │
│                                                                 │
│   W - Wishes (희망/바람)                                        │
│       querer que, desear que, esperar que, ojalá (que)         │
│                                                                 │
│   E - Emotions (감정)                                           │
│       alegrarse de que, sentir que, temer que, molestar que    │
│                                                                 │
│   I - Impersonal expressions (비인칭 표현)                      │
│       es importante que, es necesario que, es posible que      │
│                                                                 │
│   R - Recommendations (조언/명령)                               │
│       recomendar que, sugerir que, aconsejar que, pedir que    │
│                                                                 │
│   D - Doubt/Denial (의심/부정)                                  │
│       dudar que, no creer que, no pensar que, negar que        │
│                                                                 │
│   O - Ojalá (바람) + 접속사                                     │
│       para que, antes de que, sin que, a menos que             │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 상세 예문

```
희망 (Wishes):
- Quiero que estudies más. (더 공부하길 원한다.)
- Deseo que seas feliz. (행복하길 바란다.)
- Espero que llueva mañana. (내일 비가 오길 바란다.)
- Ojalá venga pronto. (빨리 오길 바란다.)

감정 (Emotions):
- Me alegro de que estés bien. (네가 잘 있어서 기쁘다.)
- Siento que no puedas venir. (네가 올 수 없어서 유감이다.)
- Temo que sea demasiado tarde. (너무 늦을까 두렵다.)
- Me molesta que hables así. (네가 그렇게 말해서 짜증난다.)

비인칭 (Impersonal):
- Es importante que llegues a tiempo.
  (제시간에 도착하는 것이 중요하다.)
- Es necesario que estudies. (공부할 필요가 있다.)
- Es posible que llueva. (비가 올 수 있다.)
- Es mejor que te vayas. (네가 가는 것이 낫다.)

조언/명령 (Recommendations):
- Te recomiendo que descanses. (쉬라고 권한다.)
- Sugiero que lo pienses bien. (잘 생각해 보라고 제안한다.)
- Te pido que me ayudes. (도와달라고 부탁한다.)
- Ordeno que salgas. (나가라고 명령한다.)

의심/부정 (Doubt/Denial):
- Dudo que lo sepa. (그가 알고 있는지 의심스럽다.)
- No creo que sea verdad. (그게 사실이라고 생각하지 않는다.)
- Niego que haya dicho eso. (그것을 말했다는 것을 부인한다.)
```

### 4.3 접속사 + 접속법

```
항상 접속법을 요구하는 접속사:
┌─────────────────┬───────────────────────────────────────────────┐
│ 접속사          │ 예문                                           │
├─────────────────┼───────────────────────────────────────────────┤
│ para que        │ Te lo digo para que lo sepas.                 │
│ (~ 하도록)      │ (네가 알도록 말해 주는 거야.)                  │
├─────────────────┼───────────────────────────────────────────────┤
│ antes de que    │ Llámame antes de que salgas.                  │
│ (~ 하기 전에)   │ (나가기 전에 전화해.)                          │
├─────────────────┼───────────────────────────────────────────────┤
│ sin que         │ Salió sin que nadie lo viera.                 │
│ (~ 없이)        │ (아무도 보지 않고 나갔다.)                     │
├─────────────────┼───────────────────────────────────────────────┤
│ a menos que     │ Iré a menos que llueva.                       │
│ (~ 하지 않으면) │ (비가 오지 않으면 갈 거야.)                    │
├─────────────────┼───────────────────────────────────────────────┤
│ con tal de que  │ Lo haré con tal de que me pagues.             │
│ (~만 한다면)    │ (돈을 준다면 할게.)                            │
├─────────────────┼───────────────────────────────────────────────┤
│ en caso de que  │ En caso de que no venga, llámame.             │
│ (~한 경우에)    │ (그가 오지 않는 경우, 전화해.)                 │
└─────────────────┴───────────────────────────────────────────────┘
```

## 5. 조건문과 접속법

### 5.1 조건문 유형

```
┌─────────────────────────────────────────────────────────────────┐
│                    조건문 유형                                   │
│                                                                 │
│   Type 1: 실현 가능한 조건 (직설법)                             │
│   Si + 직설법 현재, 직설법 미래/현재                            │
│   Si llueve, no salgo. (비가 오면, 안 나간다.)                  │
│   Si estudias, aprobarás. (공부하면, 합격할 거야.)              │
│                                                                 │
│   Type 2: 비현실적/가상의 조건 (접속법)                         │
│   Si + 과거 접속법, 조건법 단순                                  │
│   Si tuviera dinero, compraría un coche.                       │
│   (돈이 있다면, 차를 살 텐데.)                                  │
│                                                                 │
│   Type 3: 과거의 비현실적 조건 (접속법)                         │
│   Si + 과거 완료 접속법, 조건법 완료                            │
│   Si hubiera estudiado, habría aprobado.                       │
│   (공부했더라면, 합격했을 텐데.)                                │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 Type 2 조건문 예시

```
현재/미래의 가상 상황:

Si fuera rico, viajaría por todo el mundo.
(부자라면, 전 세계를 여행할 텐데.)

Si pudiera volar, iría a verte ahora mismo.
(날 수 있다면, 지금 바로 만나러 갈 텐데.)

Si tuviera más tiempo, aprendería japonés.
(시간이 더 있다면, 일본어를 배울 텐데.)

Si no lloviera, iríamos a la playa.
(비가 오지 않는다면, 해변에 갈 텐데.)
```

### 5.3 Type 3 조건문 예시

```
과거의 비현실적 상황:

Si hubiera sabido, te habría ayudado.
(알았더라면, 도와줬을 텐데.)

Si hubieras venido, habrías visto a María.
(왔더라면, 마리아를 봤을 텐데.)

Si no hubiera llovido, habríamos ido al parque.
(비가 오지 않았더라면, 공원에 갔을 텐데.)

혼합 조건문 (과거 조건 → 현재 결과):
Si hubiera estudiado medicina, ahora sería médico.
(의학을 공부했더라면, 지금 의사일 텐데.)
```

## 6. 연습 문제

### 연습 1: 직설법 vs 접속법 선택
올바른 형태를 선택하세요.

```
1. Creo que (viene / venga) mañana.
   → viene (creer + 긍정 = 직설법)

2. No creo que (viene / venga) mañana.
   → venga (no creer = 접속법)

3. Sé que (está / esté) aquí.
   → está (saber = 확실 = 직설법)

4. Espero que (tienes / tengas) suerte.
   → tengas (esperar = 희망 = 접속법)

5. Es verdad que (llueve / llueva).
   → llueve (es verdad = 사실 = 직설법)

6. Es posible que (llueve / llueva).
   → llueva (es posible = 불확실 = 접속법)
```

### 연습 2: 접속법 활용 완성
빈칸을 채우세요.

```
1. Quiero que tú _______ (venir) a la fiesta.
   → vengas

2. Esperamos que ellos _______ (llegar) a tiempo.
   → lleguen

3. Si yo _______ (tener) dinero, viajaría.
   → tuviera

4. Me alegro de que tú _______ (estar) bien.
   → estés

5. Es importante que nosotros _______ (estudiar).
   → estudiemos

6. Si ella _______ (saber), te lo habría dicho.
   → hubiera sabido
```

### 연습 3: 문장 작문
주어진 상황에 맞게 접속법을 사용하여 문장을 작성하세요.

```
1. 친구에게 건강하길 바란다고 말하기
   → Espero que estés bien. / Deseo que te mejores.

2. 내일 비가 오지 않기를 바라기
   → Ojalá no llueva mañana.

3. 부자라면 큰 집을 살 것이라고 말하기
   → Si fuera rico, compraría una casa grande.

4. 일찍 도착하도록 부탁하기
   → Te pido que llegues temprano.

5. 그녀가 올지 의심하기
   → Dudo que ella venga.
```
