## 1. Verb System Overview

### 1.1 Verb Classification

```
┌─────────────────────────────────────────────────────────────────┐
│                    Spanish Verb Classification                  │
│                                                                 │
│   -AR verbs (Group 1)  -ER verbs (Group 2)  -IR verbs (Group 3)│
│   hablar (to speak)    comer (to eat)       vivir (to live)    │
│   trabajar (to work)   beber (to drink)     escribir (to write)│
│   estudiar (to study)  leer (to read)       partir (to leave)  │
│                                                                 │
│   Most common          Second                Third             │
│   (about 75%)          (about 15%)           (about 10%)       │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 Tense System Summary

```
Indicative (Indicativo):
┌────────────────────┬──────────────────────────────────────────────┐
│ Tense              │ Example (hablar)                             │
├────────────────────┼──────────────────────────────────────────────┤
│ Present            │ hablo, hablas, habla, hablamos, habláis, hablan │
│ Present Progressive│ estoy hablando                               │
│ Preterite          │ hablé, hablaste, habló, hablamos, hablasteis, hablaron │
│ Imperfect          │ hablaba, hablabas, hablaba, hablábamos...    │
│ Future             │ hablaré, hablarás, hablará, hablaremos...    │
│ Conditional        │ hablaría, hablarías, hablaría, hablaríamos...│
│ Present Perfect    │ he hablado, has hablado, ha hablado...       │
│ Past Perfect       │ había hablado, habías hablado...             │
└────────────────────┴──────────────────────────────────────────────┘

Subjunctive (Subjuntivo):
┌────────────────────┬──────────────────────────────────────────────┐
│ Tense              │ Example (hablar)                             │
├────────────────────┼──────────────────────────────────────────────┤
│ Present            │ hable, hables, hable, hablemos, habléis, hablen │
│ Past               │ hablara/hablase, hablaras, hablara...        │
│ Present Perfect    │ haya hablado, hayas hablado...               │
│ Past Perfect       │ hubiera/hubiese hablado...                   │
└────────────────────┴──────────────────────────────────────────────┘

Imperative (Imperativo):
┌────────────────────┬──────────────────────────────────────────────┐
│ Form               │ Example (hablar)                             │
├────────────────────┼──────────────────────────────────────────────┤
│ Affirmative        │ habla (tú), hable (usted), hablad (vosotros)│
│ Negative           │ no hables, no hable, no habléis             │
└────────────────────┴──────────────────────────────────────────────┘
```

## 2. Stem-Changing Verbs

### 2.1 e → ie Change

```
┌─────────────────────────────────────────────────────────────────┐
│                    e → ie Stem Change                           │
│                                                                 │
│   pensar (to think)  querer (to want)   sentir (to feel)       │
│   pienso             quiero             siento                  │
│   piensas            quieres            sientes                 │
│   piensa             quiere             siente                  │
│   pensamos           queremos           sentimos   ← No change  │
│   pensáis            queréis            sentís     ← No change  │
│   piensan            quieren            sienten                 │
│                                                                 │
│   ※ nosotros, vosotros do not change (boot verb)              │
│                                                                 │
│   Other verbs:                                                  │
│   cerrar (to close), empezar (to begin), entender (to understand),│
│   perder (to lose), preferir (to prefer), mentir (to lie)      │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 o → ue Change

```
┌─────────────────────────────────────────────────────────────────┐
│                    o → ue Stem Change                           │
│                                                                 │
│   poder (to be able) volver (to return)  dormir (to sleep)     │
│   puedo              vuelvo              duermo                 │
│   puedes             vuelves             duermes                │
│   puede              vuelve              duerme                 │
│   podemos            volvemos            dormimos  ← No change  │
│   podéis             volvéis             dormís    ← No change  │
│   pueden             vuelven             duermen                │
│                                                                 │
│   Other verbs:                                                  │
│   encontrar (to find), contar (to count), costar (to cost),    │
│   mostrar (to show), recordar (to remember), morir (to die)    │
│                                                                 │
│   Special: jugar (to play) → juego, juegas... (u → ue)         │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 e → i Change

```
┌─────────────────────────────────────────────────────────────────┐
│                    e → i Stem Change (-IR verbs only)           │
│                                                                 │
│   pedir (to request) servir (to serve)  repetir (to repeat)    │
│   pido               sirvo              repito                  │
│   pides              sirves             repites                 │
│   pide               sirve              repite                  │
│   pedimos            servimos           repetimos ← No change   │
│   pedís              servís             repetís   ← No change   │
│   piden              sirven             repiten                 │
│                                                                 │
│   ※ In preterite, 3rd person also changes:                     │
│   pedir → pidió, pidieron                                       │
│   servir → sirvió, sirvieron                                    │
│                                                                 │
│   Other verbs:                                                  │
│   seguir (to follow), conseguir (to get), vestir (to dress)    │
└─────────────────────────────────────────────────────────────────┘
```

```
┌──────────┬──────────┬──────────┬──────────┬──────────────────────┐
│ Change   │ yo       │ tú       │ nos.     │ Representative Verbs │
├──────────┼──────────┼──────────┼──────────┼──────────────────────┤
│ e → ie   │ pienso   │ piensas  │ pensamos │ pensar, querer,      │
│          │          │          │          │ entender, sentir     │
├──────────┼──────────┼──────────┼──────────┼──────────────────────┤

├──────────┼──────────┼──────────┼──────────┼──────────────────────┤
│ e → i    │ pido     │ pides    │ pedimos  │ pedir, servir,       │
│ (-IR only)│         │          │          │ seguir, repetir      │
├──────────┼──────────┼──────────┼──────────┼──────────────────────┤
```

## 3. Irregular Verb Patterns

### 3.1 First Person Singular Irregular (-go form)

```
┌─────────────────────────────────────────────────────────────────┐
│                    yo form -go Irregular                        │
│                                                                 │
│   Verb        yo form     Rest is regular                       │
├─────────────────────────────────────────────────────────────────┤
│   tener       tengo       tienes, tiene, tenemos, tenéis, tienen│
│   venir       vengo       vienes, viene, venimos, venís, vienen│
│   poner       pongo       pones, pone, ponemos, ponéis, ponen  │
│   salir       salgo       sales, sale, salimos, salís, salen   │
│   hacer       hago        haces, hace, hacemos, hacéis, hacen  │
│   decir       digo        dices, dice, decimos, decís, dicen   │
│   traer       traigo      traes, trae, traemos, traéis, traen  │
│   caer        caigo       caes, cae, caemos, caéis, caen       │
│   oír         oigo        oyes, oye, oímos, oís, oyen          │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 Completely Irregular Verbs

```
ser (to be):
┌────────┬────────┬────────┬────────┬────────┬────────┐
│ Tense  │ yo     │ tú     │ él     │ nos.   │ ellos  │
├────────┼────────┼────────┼────────┼────────┼────────┤
│ Present│ soy    │ eres   │ es     │ somos  │ son    │
│ Preterite│ fui  │ fuiste │ fue    │ fuimos │ fueron │
│ Imperfect│ era  │ eras   │ era    │ éramos │ eran   │
└────────┴────────┴────────┴────────┴────────┴────────┘

ir (to go):
┌────────┬────────┬────────┬────────┬────────┬────────┐
│ Tense  │ yo     │ tú     │ él     │ nos.   │ ellos  │
├────────┼────────┼────────┼────────┼────────┼────────┤
│ Present│ voy    │ vas    │ va     │ vamos  │ van    │
│ Preterite│ fui  │ fuiste │ fue    │ fuimos │ fueron │
│ Imperfect│ iba  │ ibas   │ iba    │ íbamos │ iban   │
└────────┴────────┴────────┴────────┴────────┴────────┘

※ ser and ir have identical preterite forms!

estar (to be):
┌────────┬────────┬────────┬────────┬────────┬────────┐
│ Tense  │ yo     │ tú     │ él     │ nos.   │ ellos  │
├────────┼────────┼────────┼────────┼────────┼────────┤
│ Present│ estoy  │ estás  │ está   │ estamos│ están  │
│ Preterite│estuve│estuviste│estuvo │estuvimos│estuvieron│
└────────┴────────┴────────┴────────┴────────┴────────┘
```

## 4. Reflexive Verbs

### 4.1 Reflexive Pronouns

```
┌─────────────────────────────────────────────────────────────────┐
│                    Reflexive Pronouns                           │
│                                                                 │
│   Subject      Reflexive       Example (lavarse - to wash)     │
│                Pronoun                                          │
├─────────────────────────────────────────────────────────────────┤
│   yo           me              me lavo (I wash myself)          │
│   tú           te              te lavas (you wash yourself)     │
│   él/ella/Ud.  se              se lava (he washes himself)      │
│   nosotros     nos             nos lavamos (we wash ourselves)  │
│   vosotros     os              os laváis (you wash yourselves)  │
│   ellos/Uds.   se              se lavan (they wash themselves)  │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 Types of Reflexive Verbs

```
1. Daily routines:
   levantarse (to get up)       → Me levanto a las 7.
   acostarse (to go to bed)     → Se acuesta tarde.
   ducharse (to shower)         → Nos duchamos por la mañana.
   vestirse (to get dressed)    → Te vistes rápido.
   peinarse (to comb hair)      → Me peino antes de salir.
   afeitarse (to shave)         → Se afeita todos los días.

2. Emotion/state changes:
   alegrarse (to be happy)      → Me alegro de verte.
   enojarse (to get angry)      → Se enoja fácilmente.
   aburrirse (to get bored)     → Nos aburrimos en clase.
   preocuparse (to worry)       → No te preocupes.
   sentirse (to feel)           → Me siento bien.

3. Reciprocal actions (each other):
   conocerse (to know each other) → Nos conocemos desde niños.
   ayudarse (to help each other)  → Se ayudan mucho.
   quererse (to love each other)  → Se quieren mucho.
```

### 4.3 Reflexive vs Non-Reflexive

```
Non-reflexive (to others)      Reflexive (to oneself)
─────────────────────────────────────────────────────
lavar (to wash)                lavarse (to wash oneself)
Lavo el coche.                 Me lavo las manos.
(I wash the car)               (I wash my hands)

llamar (to call)               llamarse (to be named)
Llamo a María.                 Me llamo Juan.
(I call María)                 (My name is Juan)

despertar (to wake someone)    despertarse (to wake up)
Despierto a mi hijo.           Me despierto temprano.
(I wake my son)                (I wake up early)

ir (to go)                     irse (to leave)
Voy al trabajo.                Me voy ahora.
(I go to work)                 (I'm leaving now)
```

## 5. Compound Tenses

### 5.1 haber Conjugation

```
┌────────┬──────────────────────────────────────────────────────────┐
│ Tense  │ haber conjugation                                        │
├────────┼──────────────────────────────────────────────────────────┤
│ Present│ he, has, ha, hemos, habéis, han                          │
│ Imperfect│ había, habías, había, habíamos, habíais, habían       │
│ Subjunctive│ haya, hayas, haya, hayamos, hayáis, hayan           │
│ Past Subj│ hubiera, hubieras, hubiera, hubiéramos, hubierais...  │
└────────┴──────────────────────────────────────────────────────────┘

Past Participle Formation:
-AR → -ado: hablar → hablado
-ER → -ido: comer → comido
-IR → -ido: vivir → vivido

Irregular Past Participles:
hacer → hecho      poner → puesto     ver → visto
escribir → escrito  abrir → abierto   decir → dicho
volver → vuelto    morir → muerto    romper → roto
```

### 5.2 Compound Tense Formation

```
Present Perfect (Pretérito Perfecto):
haber (present) + past participle
─────────────────────────────────────────
He hablado con él. (I have spoken with him.)
¿Has comido? (Have you eaten?)
Han llegado tarde. (They have arrived late.)

Past Perfect (Pluscuamperfecto):
haber (imperfect) + past participle
─────────────────────────────────────────
Había terminado cuando llegaste.
(I had finished when you arrived.)

Future Perfect (Futuro Perfecto):
haber (future) + past participle
─────────────────────────────────────────
Habrá llegado para las cinco.
(He will have arrived by five o'clock.)

Conditional Perfect (Condicional Perfecto):
haber (conditional) + past participle
─────────────────────────────────────────
Habría ido si hubiera sabido.
(I would have gone if I had known.)

Present Perfect Subjunctive:
haber (present subjunctive) + past participle
─────────────────────────────────────────
Espero que haya llegado bien.
(I hope he has arrived safely.)

Past Perfect Subjunctive:
haber (past subjunctive) + past participle
─────────────────────────────────────────
Si hubiera sabido, habría venido.
(If I had known, I would have come.)
```

## 6. Practice Exercises

### Exercise 1: Stem-Changing Verb Conjugation
Fill in the blanks.

```
1. pensar (yo, present) → _______
   → pienso

2. poder (ellos, present) → _______
   → pueden

3. pedir (tú, present) → _______
   → pides

4. dormir (nosotros, present) → _______
   → dormimos (no change)

5. querer (ella, present) → _______
   → quiere
```

### Exercise 2: Irregular Preterite Conjugation
Convert to preterite tense.

```
1. tener (yo) → _______
   → tuve

2. hacer (él) → _______
   → hizo

3. ir (nosotros) → _______
   → fuimos

4. decir (ellos) → _______
   → dijeron

5. poder (tú) → _______
   → pudiste
```

### Exercise 3: Reflexive Verb Sentence Completion
Fill in the appropriate reflexive pronoun.

```
1. Yo _______ levanto a las siete.
   → me

2. Ella _______ ducha por la mañana.
   → se

3. Nosotros _______ acostamos tarde.
   → nos

4. ¿Tú _______ sientes bien?
   → te

5. Ellos _______ conocieron en Madrid.
   → se
```

### Exercise 4: Compound Tense Formation
Convert to the given tense.

```
1. hablar (yo, present perfect) → _______
   → he hablado

2. escribir (ella, past perfect) → _______
   → había escrito

3. ver (nosotros, present perfect) → _______
   → hemos visto

4. hacer (tú, present perfect subjunctive) → _______
   → hayas hecho
```
